# 面试评分维度说明

> 从面试官的实际评分表出发，帮你理解每个维度在看什么。

---

## 通用评分维度（大多数公司的面试评价表都包含以下维度）

### 1. 专业能力 (Technical/Professional Competence)

| 等级 | 标准 |
|------|------|
| 优秀 | 专业知识扎实且有深度，能解决复杂问题，方案有创新性 |
| 良好 | 专业知识满足岗位要求，能独立解决常见问题 |
| 合格 | 基本面够，但深度不足，需要培养 |
| 不通过 | 基础薄弱，无法胜任岗位核心工作 |

### 2. 问题解决能力 (Problem Solving)

| 等级 | 标准 |
|------|------|
| 优秀 | 能快速定位问题本质，结构化分析，提出多个可行方案并做trade-off |
| 良好 | 能拆解问题，给出合理方案 |
| 合格 | 需要引导才能找到方向 |
| 不通过 | 面对问题无从下手，缺乏分析能力 |

### 3. 沟通表达 (Communication)

| 等级 | 标准 |
|------|------|
| 优秀 | 结构清晰、简洁有力、能根据听众调整表达方式 |
| 良好 | 表达清楚，逻辑基本通顺 |
| 合格 | 能说明白但啰嗦或结构差 |
| 不通过 | 表达混乱，面试官无法理解你在说什么 |

### 4. 团队合作 (Collaboration)

| 等级 | 标准 |
|------|------|
| 优秀 | 有清晰的协作案例，能处理冲突、推动共识、影响他人 |
| 良好 | 有团队意识，能与人正常合作 |
| 合格 | 以个人为主，但不排斥合作 |
| 不通过 | 缺乏团队意识，或有明显的协作障碍 |

### 5. 文化匹配 (Culture Fit)

| 等级 | 标准 |
|------|------|
| 优秀 | 价值观高度契合，工作风格和团队互补 |
| 良好 | 基本匹配，没有明显冲突 |
| 合格 | 需要适应，但可以融入 |
| 不通过 | 价值观或工作风格与团队严重不匹配 |

### 6. 成长潜力 (Growth Potential)

| 等级 | 标准 |
|------|------|
| 优秀 | 学习能力强，有自驱力，能看到1-2年后的成长空间 |
| 良好 | 有学习意愿和能力 |
| 合格 | 能胜任当前工作，但成长速度不确定 |
| 不通过 | 安于现状，缺乏成长动力 |

## 面试官的隐性评分

这些不会写在评价表上，但会影响面试官的最终判断：

```
加分项（面试官内心的 +1）：
  + 准时到达，状态良好
  + 提问质量高，说明做了功课
  + 对自己的不足坦诚但有改进计划
  + 能用数据说话
  + 谈吐自信但不傲慢
  + 面试结束后发了得体的感谢邮件

减分项（面试官内心的 -1）：
  - 迟到且不提前通知
  - 说前公司/前同事坏话
  - 回答明显在背模板
  - 对自己的成果含糊其辞（可能有水分）
  - 薪资预期完全不合理
  - 全程没有任何问题要问
  - 态度傲慢或过度自卑
```

## 如何利用这些维度自我评估

面试结束后，用以下方式给自己打分：

```
维度         | 我的表现 | 面试官可能的判断 | 改进方向
────────────────────────────────────────────────────
专业能力      | [1-5]   | [推测]          | [具体]
问题解决      | [1-5]   | [推测]          | [具体]
沟通表达      | [1-5]   | [推测]          | [具体]
团队合作      | [1-5]   | [推测]          | [具体]
文化匹配      | [1-5]   | [推测]          | [具体]
成长潜力      | [1-5]   | [推测]          | [具体]
```

一般来说，6个维度中有4个达到"良好"以上就有通过的可能。
但如果任何一个维度是"不通过"，整体通过的概率很低。
