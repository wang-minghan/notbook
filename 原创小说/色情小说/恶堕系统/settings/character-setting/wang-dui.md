# 王队 — Rush嗨操派对主理人

## 基础信息

- **角色定位**：攻方④——药物群交+车轮战的开启者
- **年龄**：45岁
- **身份**：退役运动员，搏击俱乐部老板，地下派对组织者
- **外形**：退役拳击手身板——肩膀极宽、脖子粗、手臂比常人大腿粗。脸上有旧拳痕，鼻梁有点歪，笑起来有股大哥式的豪爽
- **鸡巴**：22cm / 极粗 / 龟头圆钝 / 像一截棒球棍——不算最长但最粗，撑开感最恐怖
- **核心癖好**：Rush嗨操派对 + 药物群交 + 双龙 + 车轮战

## 人物特点

王队的场子从来不缺两样东西——人，和rush。每次3-5人，进门先递rush。张野第一次接触到药物的世界。

### 派对流程
1. 开局：王队按张野在沙发上，rush怼到鼻孔——"深吸，憋住……再吸一口"
2. 轮操：rush的作用下意识变得模糊，鸡巴轮着上，每个人的脸都是模糊的
3. 双龙压轴：王队和另一个最粗的兄弟同时插进来，前后穴同时被塞满
4. 收场：张野瘫在沙发上，浑身精液和汗，瞳孔还有点散

### 功能
从王队这里，张野学会了用药物辅助做爱。以后每次接客前都会自己先吸一口——没有rush他甚至觉得"不够爽"。这是身体改造之外另一种形式的依赖。

## 精神内核

"好逼当然要大家一起操——兄弟们排好队，一人一轮，我最后收尾。"
