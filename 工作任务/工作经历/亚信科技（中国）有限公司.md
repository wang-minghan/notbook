# 亚信科技（中国）有限公司

亚信科技是你第二段核心算法工程经历的承载公司，重点方向是**大模型推理增强、RAG、本体约束、Agent 编排与领域微调**。

## 你在这里做了什么

- 主导多 Agent 大模型推理增强系统设计
- 构建混合检索：BM25 + BGE + RRF + Cross Encoder
- 构建 Neo4j 本体知识建模与关系约束推理
- 做 Qwen + LoRA 领域微调与 vLLM 推理部署
- 设计语义指令映射，把业务规则转为可组合指令单元

## 关键成果

- 实现本体约束推理链建模与可视化分析
- 系统性抑制幻觉，提升多轮推理稳定性
- 在 RAG 与纯向量检索对比中获得明显优势
- 推动客户交付落地
