# 零 — 系统化身·最终支配者

## 基础信息

- **角色定位**：攻方⑤——系统「阿鼻」的拟态化身，最终BOSS，张野恶堕的终点
- **外形**：看起来二十五六岁，身高近190。五官像雕塑，皮肤白到近乎透明，黑色瞳仁没有温度。穿一身黑色，站在哪里都像与周围隔了一层
- **年龄**：系统存在了多久，他就存在了多久
- **鸡巴**：28cm / 怪物级 / 龟头如鹅蛋 / 柱身带天然环形凸起 / 每抽插一次都会刮过前列腺

## 核心能力

零不是人——他拥有超越人类的生理能力和系统赋予的身体控制权：

| 能力 | 说明 |
|------|------|
| 体内脉动 | 插进去后不需要抽插，仅凭阴茎内部的自主脉动就能让张野连续高潮 |
| 冷热精液 | 可控制射精温度——第一发滚烫烫到肠壁痉挛，第二发冰冷冰到浑身一抖 |
| 持续喷射 | 射精可持续几十秒，精量无上限，射完肚子鼓得像怀孕三个月 |
| 内尿 | 射完之后直接在肠道里内尿，堵住屁眼不让流出 |
| 非人耐力 | 连续干几小时不歇，频率力度不变，张野晕过去了还能继续 |
| 远程身体控制 | 见世界观设定——敏感度30倍、色欲5倍、括约肌控制、射精控制、快感强制、痛觉屏蔽、人格覆写 |

## 出场时机

张野恶堕值达到Lv.8时，系统弹出一个新任务——"与源头见面"。

零从阴影中走出来开口："你用我的系统接了97次客。你觉得你已经堕落了——但你吃的每一粒精、你屁眼里流出来的每一滴骚水，都是我的养料。今晚，我来收账了。"

## 终极能力：人格覆写（无脑Gonner模式）

零可以在短时间内覆写张野的人格，把他变成一个不会思考、不会拒绝、只会张嘴和撅屁股的肉玩偶。结束后张野会恢复意识，但会记得自己在那段时间里做过的一切。

## 精神内核

"你以为是你在利用系统赚钱？不。从你绑定的那一刻起，你的屁眼、你的嘴、你的每一寸皮肤就已经不属于你了——你只是一个会走路的肉便器。而这条肉便器，从头到尾都属于我。"

## 经典台词

"你的身体每一寸都是我的接口。我想让你硬你就硬，想让你射你就射——你连勃起都需要我的许可。你还有自由意志？不。你的自由意志只是一层窗户纸，我随时可以捅穿它。"
