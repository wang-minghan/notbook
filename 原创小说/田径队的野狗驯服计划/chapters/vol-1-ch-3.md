# 第三章·打赌

- **memo:** 省赛当天威胁落地 → 赵叔在赛场看台出现 → 赛后威胁"不来说不定哪天你对象就知道了" → 器材室打赌"射了就放过你" → 沈骁输 → 被操 → 剃毛仪式
- **stage:** planning
- **字数目标:** 3000-4000字
- **本章驱动力:** 威胁驱动（赵叔掌握了全部主动）
- **本章在全卷中的角色:** 第一次被迫的性交——从"摸了但忍住了"到"被操到射了"

## 场景

### 场景1｜省赛赛场｜白天｜权重：高

- **入口情绪**：紧张、兴奋——省赛，沈骁的主场
- **核心事件**：沈骁跑四百米决赛 → 以最好成绩夺冠 → 冲线后看台欢呼 → 他在看台上看到了赵叔 → 赵叔在人群中对他笑了一下，竖起一根手指（"一"——不是庆祝的手势，是催债的倒计时）
- **情绪拐点**：从夺冠的狂喜 → 看到赵叔后的恐惧坠落
- **信息差**：队友教练不知道沈骁的恐惧 ↦ 沈骁知道赵叔来看他比赛意味着什么
- **出口情绪**：恐惧、不安——威胁正式落地
- **权重理由**：夺冠是沈骁的高光时刻，也是赵叔宣示"你的一切都在我掌控中"的时刻

### 场景2｜颁奖后更衣室｜黄昏｜权重：中

- **入口情绪**：志得意满（试图掩饰恐惧）
- **核心事件**：沈骁接到赵叔发来的短信——就一句："今晚器材室，打完球过来。不来说不定哪天你对象就知道了。"→ 沈骁握着手机在更衣室坐了很久，队友来叫庆祝他假装身体不舒服
- **情绪拐点**：从"今天是我赢了"到"我根本没有赢"
- **出口情绪**：绝望的接受

### 场景3｜器材室｜夜晚｜权重：高

- **入口情绪**：恐惧、崩溃边缘
- **核心事件**：沈骁走进器材室 → 赵叔已经等在那里 → 赵叔说"先聊聊"→ 打赌：沈骁如果能忍住不射，赵叔就放过他 → 沈骁接受 → 赵叔操他 → 沈骁没忍住射了 → 赵叔："你输了。"
- **情绪拐点**：从"我还有机会赢" → 被操到射 → 彻底输掉
- **信息差**：沈骁以为打赌公平 ↦ 赵叔从一开始就知道沈骁的身体已经被他开发到"碰几下就射"的程度
- **出口情绪**：耻辱、崩溃、意识到自己真的完了

### 场景3b｜器材室（接续）｜剃毛仪式｜权重：中

- **核心事件**：赵叔拿出剃须刀和泡沫 → 把沈骁按在垫子上剃掉阴毛 → 沈骁挣扎但被压住 → 剃完赵叔说"这才像样，跟个小孩一样干干净净的"→ 沈骁看着自己被剃光的耻部哭了
- **情绪拐点**：剃毛是身体上的最后一个标志——成了"被调教过"的人
- **出口情绪**：羞耻、失重、仪式性的屈服

### 场景4｜宿舍｜深夜｜权重：低

- **入口情绪**：麻木
- **核心事件**：沈骁回宿舍的路上碰到了林越（林越在等他）→ 林越问"你今天怎么了"→ 沈骁说"没事，比赛累了"→ 林越走近想抱他，沈骁猛地躲开了 → 林越愣住了
- **信息差**：林越察觉到裂痕但不知道原因 ↦ 沈骁不能让他知道
- **出口情绪**：裂痕初现、酸楚——NTR线的第一次张力

## 人物状态

- **沈骁起点→终点**：拿到省赛冠军（人生顶点）→ 被操 + 被剃毛 + 对林越撒谎（人生最低点）
- **赵叔策略**：用威胁强制执行 → 用"打赌"给沈骁虚假的控制感 → 赢后剃毛 —— 仪式性征服
- **林越：** 关心、担心、被推开——开始感觉到不对劲

## 红线（不可删减）

1. 赵叔出现在赛场看台上对沈骁笑并竖手指
2. 打赌的具体规则和赌注（被操射为止）
3. 剃毛仪式的全过程描写
4. 林越被推开时的错愕表情

## 禁忌（不写）

1. 不写沈骁主动享受（这时候还是被迫的）
2. 不写赵叔对沈骁有"感情"（是纯猎手心态）
3. 不写林越的直接发现（还在铺垫期）

## 毒量检测

- ✅ 威胁正式落地：赛场出现+短信威胁
- ✅ 强制突破：第一次被操（从抚摸→插入）
- ✅ 打赌陷阱：虚假自主权→心理征服
- ✅ 仪式性屈服：剃毛
- ✅ NTR线启动：推开林越，裂缝出现
- ✅ "啊这"台词："你输了。"（赵叔说射了就放过但射了之后当然不放）或"跟个小孩一样干干净净"
- ✅ 身体背叛：被操射（不可否认的快感）
- ✅ 不可逆改变：被剃毛——肉体标志
- ❌ 公开扩散：下一章
