# 多轮面试策略矩阵

---

## 典型面试流程（按公司类型）

### 互联网大厂（4-5轮）

```
第一轮：AI面试/HR电话初筛（15-30分钟）
  ↓ 通过率约40-50%
第二轮：业务面试/技术面试（45-60分钟）
  ↓ 通过率约30-40%
第三轮：交叉面试/二面（45-60分钟）
  ↓ 通过率约50-60%
第四轮：部门leader/总监面（30-45分钟）
  ↓ 通过率约60-70%
第五轮：HR终面+谈薪（30分钟）
  ↓ 通过率约80-90%
```

### 外企/跨国公司（3-4轮）

```
第一轮：HR电话筛选（英文为主）
  ↓
第二轮：Hiring Manager面试（业务+行为，可能全英文）
  ↓
第三轮：Panel面试或交叉面试（多人同时面）
  ↓
第四轮：VP/Director面（文化匹配+高层判断）
```

### 创业公司/中小企业（2-3轮）

```
第一轮：创始人/部门负责人直接面（综合评估）
  ↓
第二轮：团队成员面试或实操考核
  ↓
第三轮：CEO/合伙人终面（有些公司会跳过）
```

### 校招管培（4-6轮）

```
网申 → 在线测评/笔试 → AI面试 → 群面/AC → 单面（1-2轮）→ 终面
```

## 各轮次策略详解

### 第一轮：初筛（HR/AI）

| 维度 | 策略 |
|------|------|
| 核心目标 | 证明基本匹配度，不被筛掉 |
| 时间投入 | 20%精力，不需要过度准备 |
| 关键动作 | 自我介绍流畅、动机清晰、薪资预期合理 |
| 常见淘汰原因 | 表达不清、动机模糊、薪资预期偏差太大、稳定性存疑 |

**必备准备：**
- 2分钟自我介绍（练到自然）
- "为什么想来我们公司"的具体回答
- 期望薪资范围
- 最早可以入职的时间

### 第二轮：专业面试（直属leader/技术官）

| 维度 | 策略 |
|------|------|
| 核心目标 | 证明"你能干活"，展示专业深度 |
| 时间投入 | 40%精力，这是最关键的一轮 |
| 关键动作 | 用案例和数据说话，展示解决问题的过程 |
| 常见淘汰原因 | 专业能力不足、经验深度不够、无法应对追问 |

**必备准备：**
- 简历上每个项目的"深挖包"
- 岗位相关的实操能力（见 case_study_guide.md）
- 3-5个STAR案例（见 behavioral_question_bank.md）

### 第三轮：交叉面试

| 维度 | 策略 |
|------|------|
| 核心目标 | 证明"你能合作"，展示跨团队视角 |
| 时间投入 | 15%精力 |
| 关键动作 | 展示你理解其他角色的关切，讲协作案例 |
| 常见淘汰原因 | 视野狭窄只懂自己领域、沟通表达不佳 |

**必备准备：**
- 至少一个跨部门协作的案例
- 了解面试你的那个部门在做什么
- 展示你理解"他们为什么关心这个"

### 第四轮：终面（VP/高管）

| 维度 | 策略 |
|------|------|
| 核心目标 | 证明"你能走远"，展示格局和潜力 |
| 时间投入 | 15%精力 |
| 关键动作 | 少讲细节多讲判断，展示对行业的理解 |
| 常见淘汰原因 | 格局太小、缺乏独立思考、文化不匹配 |

**必备准备：**
- 对公司战略和行业趋势的理解
- 你的3-5年职业愿景
- 2-3个展示你价值观的故事

### 第五轮：HR谈薪

详见 `salary_negotiation.md`

## 跨轮次一致性管理

### 故事一致性

```
核心原则：每轮面试讲的事实必须一致。

做法：
  1. 准备5个核心案例，写下关键数字和时间线
  2. 每个案例准备"短版"(1分钟)和"长版"(3分钟)
  3. 不同轮次可以从不同角度讲同一个案例，但事实不能变
  4. 面试后记录：这轮面试我讲了哪些案例、提了哪些数字
```

### 信息传递的升级

```
第一轮（HR）：我做了什么 → 我为什么来
第二轮（业务）：我怎么做的 → 做出了什么成果
第三轮（交叉）：我怎么和人合作 → 我怎么推动事情
第四轮（终面）：我怎么思考 → 我能带来什么长期价值

每一轮讲的故事可以相同，但重心要升级。
```

## 面试间隔期的行动指南

```
面试后当天/次日：
  □ 写面试复盘笔记（问了什么、我怎么答的、感觉如何）
  □ 发感谢邮件（外企文化尤其重要）

等待期（3-7天）：
  □ 不要停止其他面试流程
  □ 如果超过承诺时间没消息，主动跟进一次
  □ 利用等待时间准备下一轮的内容

收到下一轮通知：
  □ 了解下一轮面试官的角色和背景
  □ 回顾上一轮面试的表现，调整策略
  □ 准备针对性的反向提问
```
