# 陆辞 — 初恋·背叛者

## 基础信息

- **角色定位**：攻方①——初恋、破处者、温柔开发、终极背叛者
- **年龄**：23岁
- **身份**：体育学院同校同学（不同系），篮球队控球后卫
- **外形**：183cm/75kg，清秀阳光型，戴细框眼镜，看着斯文干净。体育生里少有的书卷气
- **鸡巴**：17cm / 匀称直挺 / 粉褐色 / 不算惊人但够用
- **核心癖好**：恋爱调教 + 精神开发

## 人物经历

### 相遇
在篮球场上认识。陆辞是控卫，张野是前锋，打配合打出了默契。陆辞主动给张野递水、拍他肩膀说"打得好"。张野每次被拍肩都会耳朵红。

### 恋爱
两人在一起了。陆辞是张野的初恋，非常温柔。接吻轻轻的、试探性的。第一次摸张野下面的时候张野整个人绷紧了不敢动，陆辞就停手抱着他说"没事，你不愿意我们就慢慢来"——张野就是被这句话彻底攻陷的。第一次做爱做了很久前戏，插进去的时候张野疼得眼眶红了，陆辞就停下来抱着他亲他额头。

### 裂缝
系统绑定后，张野被逼着卖身，瞒着陆辞。陆辞发现张野回消息变慢、身上偶尔有陌生味道、说情话时会走神。偷偷翻了他手机。

### 背叛
陆辞没有当面揭穿——他跟踪张野到工地，拍下了张野跟赵铮做的视频。然后发过去："我以为你只是忙。原来你在忙这个。我们到此为止。"

张野看到消息的时候刚被赵铮灌了一肚子精。蹲在工地角落，眼泪砸在屏幕上。不恨陆辞——他知道是自己先脏了。

### 功能
陆辞的功能不是"最刺激的炮"而是"最痛的那一刀"。他代表张野对人世间的最后一点留恋——当这点留恋也被砍断之后，张野才真正开始堕落。
