# 个人健身与饮食计划

## 📊 基础数据
- 性别/年龄：男 / 27岁
- 身高/体重：170cm / 71kg
- BMI：24.6（微偏高，适合身体重组）
- 目标：增肌 + 减脂同步进行
- 训练：健身房，每周4-5练（力量 + 有氧）
- 补剂：乳清蛋白粉

## 🎯 热量目标
- 推荐摄入：**2300-2400 kcal/天**
- 蛋白质：**140-150g/天**
- 碳水：**250-280g/天**
- 脂肪：**55-65g/天**
- 饮水：**2.5-3L/天**

## 🥗 每日饮食方案（约 2300-2400 kcal）

### 原则
1. 必须吃早餐！不吃早餐肌肉会分解，代谢降低
2. 每餐都要有优质蛋白（肉/蛋/奶/蛋白粉）
3. 碳水集中在训练前后吃
4. 夜宵换成蛋白+蔬菜组合
5. 蛋白粉在训练后30分钟内喝效果最佳

### 食谱方案 A（基础增肌版）— 推荐训练日使用
| 餐次 | 内容 | 热量≈ |
|------|------|-------|
| 🥣 早餐 7:30 | 全麦面包2片 + 鸡蛋2个 + 牛奶200ml + 香蕉1根 | 420 |
| 🥗 午餐 12:00 | 米饭200g + 鸡胸肉200g + 西兰花/青菜不限量 + 橄榄油 | 650 |
| 💪 练后 | 蛋白粉1勺 + 香蕉1根（训练日） | 180 |
| 🥘 晚餐 18:30 | 红薯/糙米150g + 鱼肉/牛肉200g + 混合蔬菜 + 少许坚果 | 600 |
| 🌙 夜宵（偶尔饿时） | 希腊酸奶100g + 少量水果 / 或 水煮蛋2个 | 150-200 |

### 食谱方案 B（控制热量版）
| 餐次 | 内容 | 热量≈ |
|------|------|-------|
| 🥣 早餐 7:30 | 燕麦片50g + 鸡蛋2个 + 黑咖啡 | 350 |
| 🥗 午餐 12:00 | 糙米150g + 瘦牛肉150g + 菠菜/生菜沙拉 | 580 |
| 💪 练后 | 蛋白粉1勺 + 水 | 120 |
| 🥘 晚餐 18:30 | 南瓜/玉米100g + 鸡腿肉（去皮）200g + 清炒时蔬 | 550 |
| 🌙 夜宵（饿时） | 黄瓜/番茄 + 水煮蛋白3个 | 100 |

### 食谱方案 C（碳水循环 — 训练日高碳/休息日低碳）
**训练日高碳版：**
| 餐次 | 内容 |
|------|------|
| 🥣 早餐 | 全麦馒头/贝果1个 + 鸡蛋2个 + 牛奶200ml |
| 🥗 午餐 | 米饭250g + 鸡胸肉200g + 各类蔬菜 |
| 💪 练后 | 蛋白粉1勺 + 香蕉1根 |
| 🥘 晚餐 | 红薯200g + 鱼肉200g + 蔬菜沙拉 |

**休息日低碳版：**
| 餐次 | 内容 |
|------|------|
| 🥣 早餐 | 鸡蛋3个 + 牛油果半个 + 黑咖啡 |
| 🥗 午餐 | 鸡腿肉200g + 大量蔬菜 + 橄榄油 |
| 🥘 晚餐 | 三文鱼/牛肉200g + 西兰花/芦笋 |
