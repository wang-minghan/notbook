# 赵铮 — 弄脏者·彻底沉沦触发者

## 基础信息

- **角色定位**：攻方③——绑架迷奸，彻底摧毁张野人格底线的转折点
- **年龄**：38岁
- **身份**：工地包工头
- **外形**：黑壮粗犷，满手老茧，一身汗味烟味混着廉价白酒味。寸头，脖子上一条粗金链子，手指关节粗大，穿着迷彩裤和脏工靴
- **鸡巴**：24cm / 极粗 / 龟头像小孩手腕 / 青筋盘虬 / 柱身微弯 / 黑红发亮 / 像一根工地钢管
- **核心癖好**：绑架迷奸 + 公共场所露出 + 颜射口爆 + 内尿

## 人物经历

### 绑架
张野在工地附近路边摊吃炒粉，去公厕时被赵铮从背后捂住嘴，乙醚布捂到鼻子上。醒来时被呈大字型绑在工地板房的床上。

### 迷奸
无润滑直接插入。24cm的粗黑鸡巴没有任何前戏地捅进屁眼。疼到弓起身体、被胶带封住的嘴里发出呜呜的惨叫。赵铮的抽插是最原始的、发泄式的操干。

### 三次射精
- 第一次：全部射在脸上，糊住眼睛和鼻孔，逼他舔干净
- 第二次：内射后直接内尿——滚烫的尿液混着精液灌满肚子，拿破布塞住屁眼
- 第三次：天蒙蒙亮时拉到塔吊平台下，在露天跪着撅屁股被路过工人看见

### 释放
第二天傍晚把张野丢在工地门口，只给了一件脏工服裹着下身。张野蹲在马路牙子上，低头看自己身上那些掐痕、掌印、干涸的精液白痕——笑了。

从那天起，他心里那根叫"底线"的弦彻底断了。

## 功能

赵铮把张野最后一点做人的尊严碾碎了。被绑架、被迷奸、被内尿、被公开露出——这些事之后，系统任务里的"卖身"反而显得温和了。张野开始觉得：反正我已经是这样的人了。
