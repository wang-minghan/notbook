# Prompt — 第二卷·第3章「中出」

## 角色

你是一个都市题材（urban）小说作家。现代都市背景，所有场景发生在真实城市空间——健身房、休息床、傍晚的公交站、昏暗的出租屋。超自然元素仅限「阿鼻系统」本身，其余世界观完全写实。不涉及修仙、穿越、异能战斗等非都市元素。

## 任务指示

创作第二卷·第3章正文（在全卷中的角色：第2层·升级——本章以肉体的升级推进为主要冲突。这是张威第二次约张野。不再是第一次的粗暴撕裂式插入——张威直接内射，第一次把精液灌进张野体内。张野愣住了：被灌满的陌生感、温热液体涌进肠道深处的冲击。张威射完不拔出来，堵着精液继续操，让白沫从穴口溢出。结束后张野夹紧屁眼一路含着那泡精液回出租屋。羞耻——但身体深处有一种诡异的满足感。）

### 字数与压缩策略

- **目标字数**：建议 3500-4000 字（±10% 可接受，叙事完整性优先于字数精确）
- **压缩策略**：超字数时优先压缩低权重场景（关键点1过渡/关键点7章末收束），推荐压缩至150-200字以内——保留1句环境切换 + 1句人物动作 + 1句情绪/氛围留白；不得因压缩删改红线内容或破坏情绪主线

### 本章驱动力

张威第二次约张野，短信就三个字：「来一趟」。张野去健身房，这次插入比第一次顺利——张威发现了，笑了一声「操，学得挺快」。张威一边操一边说粗话，操到临近射精时直接内射——第一次把精液灌进张野体内。射完不拔出来，堵着精液继续抽插，白沫从穴口溢出。结束后张威说「含着，别浪费」。张野夹紧屁眼一路含着那泡精液回出租屋，蹲下排精时——羞耻是确定的，但身体深处有一种说不清的、诡异的满足感。

### 信息释放节奏

肉体推进型——从紧张到放松到震惊到混乱到羞耻到诡异满足。到健身房门口犹豫（过渡）→被摸发现身体已经适应（推进·对话）→顺利插入（推进·动作）→抽插和脏话（推进·对话）→中出的冲击（核心高潮）→堵精白沫溢出的视觉冲击（持续高潮）→「含着，别浪费」指令（过渡）→夹精回程的羞耻（情绪峰值）→出租屋排精后的诡异满足（章末落点）。

### 叙事目标

1. 核心悬念：张野的身体还会像第一次那样疼吗？他会接受被内射吗？
   → 本章结束时：**解答了**——身体已经适应了插入（不再撕裂疼），但被内射的冲击让他愣住了。他没有拒绝，也没有接受，是被动的、措手不及的。
2. 读者情绪：从紧张到放松到震惊到混乱到羞耻到诡异满足——终点落在「身体深处开始记住被灌满的感觉」
3. 爽点设计：顺利插入 + 中出的冲击 + 堵精搅出白沫的视觉 + 夹精回家的全程羞耻 + 章末诡异满足的余韵
   - 铺垫于关键点1-2（过渡+检查身体的适应）
   - 释放于关键点3-4（插入逐渐顺利+脏话对话的粗粝感）
   - 高潮释放于关键点5-6（中出+堵精——本章核心场景）
   - 余韵于关键点7-9（收尾+夹精回家+诡异满足）

### 写作方法论

1. **焦点锁定**：对每一场景，识别其唯一的「核心冲突」或「人物情绪」或「信息差」（三选一，不叠加）。将70%以上笔墨集中于此焦点对象，次要元素简笔带过。高权重场景细化到动作与感知层级；低权重场景允许快速转场，但不得违反焦点锁定规则。

2. **感官分层取舍**：以 2 种感官为主进行细化，氛围允许时增至 3 种（如触觉+视觉+听觉），其余一笔略写。禁止全维度均匀堆砌。
   - 本章感官优先顺序：**触觉 > 视觉 > 听觉**
   - 触觉：张威手指揉肛门的感觉、龟头滑入不再撕裂但依然酸胀的撑开感、精液灌进肠道时的滚烫和冲击感、堵着精液继续操时的滑腻感、白沫流到大腿根儿的粘腻感、夹精走路时肠道里精液轻微晃动的体感
   - 视觉：白沫从穴口溢出来顺着大腿流下的画面、床单上那一圈白色浊液、张野低头看到自己身下精液的震惊、回程路上夹紧肛门走路的体态
   - 听觉：张威的脏话（「操，学得挺快」「屁股夹这么紧，是不是等了一天？」「操，出水了，够骚的」「射给你」「含着，别浪费」）、咕叽咕叽的水声（肠道被操出肠液）、肉体撞击的啪啪声、拔出来时「啵」的一声

3. **叙事节奏交替**：一段密集动作/触感描写 + 一段短留白 + 一句轻环境过渡循环。禁止连续长描写、连续对话、通篇等长段落。性爱场景要有层次推进——试探→插入→加速→中出→堵精→余韵的完整节奏。

4. **情绪外化，不直述**：张野的紧张不写「他感到紧张」，写「他在健身房门口站了几秒才推门」；震惊不写「他很震惊」，写「大脑一片空白、全身僵住、手指攥紧床单」；羞耻不写「他很羞耻」，写「步子迈得小、腿夹得紧、脸颊烫得发红」；诡异满足不写「他很满足」，写「他蹲着低头看那滩精液喘粗气，脑子里还在回放被灌满的那几秒——说不清是什么感觉，但身体记住了」。

## 背景信息

### 前情上下文

第2章结尾：张威第一次约张野，粗暴地操了他——没有前戏、直接插入、20cm的鸡巴撑得张野裂开一样疼。那是张野第一次被男人操，疼得全身发抖，但咬着牙没跑。张威射在张野背上，没有清理，没有安抚。张野一瘸一拐地回了出租屋。

读者情绪残留：疼痛 + 紧张 + 好奇——第一次的粗暴插入让读者看到张威的野蛮和张野的承受力，读者想知道第二次会怎样——会更疼吗？张野还会去吗？

上章留下的缺口：张威说「明天还来」——不是邀请，是通知。读者知道张野会去，但不知道第二次会发生什么。

### 角色初始状态

**张野：**
  起点：收到张威短信「来一趟」。站在健身房门口，犹豫了几秒，推门进去。
  → 经历：走进健身房→被张威摸屁股检查→被按在床上插入→被脏话羞辱→被第一次内射→被堵精→被命令「含着」→夹精走回出租屋→蹲下排精
  → 终点：蹲在地上看那滩从自己体内流出的精液，喘着粗气——羞耻是确定的，但身体深处有一种陌生的、诡异的满足感。
  本章说话方式：几乎不说话。被操时闷哼、粗喘、被闷住的呜咽为主，没有成句的对话。被命令「含着」时没有回答——沉默地收缩括约肌。
  本章微习惯/小动作：到健身房门口时站住了几秒才推门；被摸屁股时身体绷了一下但没有躲；被操时脸埋进手臂里；被内射时手指攥紧床单；夹精回程时步子迈得小、腿夹得紧；蹲下排精后用手背擦了擦流到大腿上的精液

**张威：**
  起点：坐休息椅上玩手机，见张野进来抬下巴指了指休息床的方向——不多话，指令式的。
  → 经历：检查张野的身体→发现后穴不再那么紧→直接插入→一边操一边说粗话→内射（第一次射进张野体内）→堵着精液继续操→拔出来→命令「含着，别浪费」→让张野回去
  → 终点：擦了擦鸡巴丢给张野一张纸巾：「擦擦，回去吧。」——纯粹的工具验收态度。
  本章必须完成的一件事：第一次把精液射进张野体内，并用「含好，别浪费」的指令让他夹精回家——这是张威中出癖好的首次展示，也是张野身体适应程度的第一次检验。
  本章张威的形象：粗暴、直接、掌控者、不多话但句句指令。脏话是张威的标签——「操」「骚逼」「够骚的」是他的语言风格。不温柔，不好奇张野的感受，只验收器物的使用情况。

**重要信息差：**
- 张威知道中出是他的常规玩法，知道张野的身体已经开始适应（不再像第一次那么紧）↦ 张野不知道中出是张威每次必做的事，不知道这只是开始
- 读者看到张野「诡异的满足感」↦ 张野自己说不清这是什么感觉，也不确定这是「想要」还是「不想要」
- 张野第一次知道「被灌满」是什么感觉——温热、粘稠、沉甸甸的陌生体验 ↦ 他不知道身体已经记住了这种感觉，第4章会开始期待

## 案例

**【中出写作示例 · 感官分层取舍（触觉+视觉）】**

> 张威操到临近射精，动作突然加深——一手按住张野的腰，龟头死死顶在最深处，闷吼了一声：「射给你。」张野愣住，还没来得及反应，一股滚烫的精液就灌进肠道深处。那一刻张野大脑一片空白——陌生的温热感从体内最深处涌开，量很大，一股接一股喷在肠道内壁上。张野全身僵住，手指攥紧床单，嘴里发出一声被闷住的呜咽。张威射了七八股才停，鸡巴还在体内一跳一跳地抽搐。

（演示：感官分层取舍·触觉+视觉——以精液灌入肠道的温热冲击（烫/量大/一股接一股）和张野的身体反应（愣住/僵住/攥床单/呜咽）为主，省略听觉描写，符合场景焦点在「第一次被内射的触觉冲击」）

**【堵精写作示例 · 焦点锁定（核心冲突）】**

> 张威射完没拔出来——堵着。他停了几秒，等精液在肠道里浸润开，然后又开始慢慢抽插。鸡巴在灌满精液的肠道里滑进滑出，白沫从穴口溢出来，顺着张野的大腿根往下流。张野低头看到自己身下的床单上沾了一圈白色的浊液——自己被内射的精液和抽插搅出来的泡沫。张威越操越快，每一次抽插都带出更多白沫。张野被这种视觉和触觉的双重冲击弄懵了。

（演示：焦点锁定·核心冲突——「堵精时白沫溢出的视觉冲击」是本场景的核心冲突，笔墨集中在鸡巴在灌满精液的肠道里抽插、白沫溢出、张野低头看到精液和泡沫的震惊，省略其他因素）

**【对话标签替代示例 · 张威的说话方式】**
> 张威抬下巴指了指休息床的方向。张野走过去。张威的手直接摸上他的屁股，拇指按在肛门口揉了揉，发现不再那么紧了，笑了一声：「操，学得挺快。」

（演示：对话标签替代——删掉「张威说」，用动作+直接引语承接，用张野的反应替代对话标签。张威的说话简练、带脏字、指令式）

## 输入

### 场景原材料

**关键点 1｜过渡·环境｜第二次约｜紧张基调｜权重：低**
  核心事件：张威第二次约张野，短信上就三个字：「来一趟」。张野站在健身房门口犹豫了几秒，推门进去。张威正坐休息椅上玩手机，见张野进来，抬下巴指了指休息床的方向。张野走过去的时候能感觉到心跳快了——不是因为期待，是因为知道接下来要发生什么。健身房环境：下午，光线一般，器材区空荡荡的，休息室的帘子半拉着。
  情绪意义：建立第二次约的状态——张野知道要发生什么但依然紧张，但对比第一次，他至少没有逃跑。
  信息差：张野知道接下来会被操但不知道张威今天会中出 ↦ 读者从章标题知道本章有中出
  情绪拐点：收到短信的紧张 → 到健身房门口犹豫 → 最终推门进去的「知道自己逃不掉」的认命感
  → 下一场景的衔接：张威没有废话，直接上手检查
  本场景焦点：人物情绪（张野的紧张和认命——通过站住几秒、心跳快、沉默地走过去外化）
  本场景权重：低（快速过渡，约150-200字）
  感官建议：触觉（手指碰到门把手时微凉的触感）+ 听觉（健身房器材安静的背景、自己心跳的感觉）
  本场景出口情绪：认命的紧张——知道要发生什么，不逃了

**关键点 2｜推进剧情·动作·对话｜检查身体｜适应基调｜权重：中**
  核心事件：张威让张野趴床上。这次张威没有直接干——他用手摸了张野的屁股，拇指按在肛门口揉了揉。张野身体绷了一下，但没有躲。张威发现昨晚操过的地方不再那么紧了，笑了一声说「操，学得挺快」。语气是满意的——像验收一件工具。没有多余的话，没有夸赞，就三个字——「学得挺快」。
  情绪意义：张野的身体适应进度第一次被张威确认——不再像第一次那样紧了。张威的「学得挺快」是一种工具验收式的满意，不是温柔。
  信息差：张威通过手指确认「后穴松了」↦ 张野自己不知道自己的括约肌已经被操开了
  情绪拐点：认命的紧张 → 被揉肛门时身体绷了一下但没有躲的被动接受 → 被评价「学得挺快」的复杂感（被当作工具验收，但身体确实适应了）
  → 下一场景的衔接：张威扶着鸡巴顶进去
  本场景焦点：核心冲突（张威对张野身体适应度的检查——用动作+简短评价完成「验收」，张野从绷紧到不躲的被动接受）
  本场景权重：中——篇幅不长但意义重要，展示身体适应进度的第一步
  感官建议：触觉（张威粗糙的拇指按在肛门口揉了揉的触感、括约肌被按压的压迫感）+ 听觉（张威简短的「操，学得挺快」——带脏字的评价）
  本场景出口情绪：被验收的感觉——身体已经不属于自己了

**关键点 3｜推进剧情·插入 | 顺利进入基调｜权重：中高**
  核心事件：张威扶着鸡巴顶进去。比起第一次的撕裂感，这次明显顺利——龟头滑过括约肌时只有一阵酸胀。张野趴在床上，脸埋进手臂里，闷哼了一声。张威没等他适应就开始抽插，节奏比第一次平稳，但依然有力——每一下都顶到底，龟头撞在张野的前列腺上。张野的呼吸从闷哼变成粗喘。
  情绪意义：身体适应的第一层展示——不再疼到撕裂，但依然酸胀和压迫。张野的闷哼比第一次轻了，从「忍痛」变成「承受」。
  信息差：张威知道这次能顺利进入（检查过了）↦ 张野感受到「不撕裂了但依然被撑开的酸胀」
  情绪拐点：被验收的感觉 → 龟头滑入时的酸胀（不是疼）→ 被直接抽插的承受（没有过渡和适应期——张威不等他）
  → 下一场景的衔接：张威开始说粗话
  本场景焦点：核心冲突（第二次插入的顺利——对比第一次的撕裂，这次只有酸胀感；张威不等他适应的粗暴节奏延续）
  本场景权重：中高——需要和第一次的撕裂痛形成对比，写酸胀感而不是撕裂感
  感官建议：触觉（龟头滑过括约肌时的酸胀撑开感，不再有第一次的热辣撕裂感；龟头每一下撞在前列腺上的压迫感）+ 听觉（张野从闷哼变成粗喘、肉体撞击的啪啪声）
  本场景出口情绪：承受——不疼了但依然被粗暴对待，身体在适应

**关键点 4｜推进剧情·抽插·对话｜粗话升级基调｜权重：中**
  核心事件：张威一边操一边说粗话：「屁股夹这么紧，是不是等了一天？」张野没回答，但呼吸从闷哼变成粗喘——20cm的鸡巴在肠道里撑开每一道褶皱，操开之后身体开始分泌肠液，进出发出咕叽的水声。张威感觉到了，「操，出水了，够骚的」。
  情绪意义：张威用脏话确认张野身体的兴奋反应——出了肠液，身体已经不再是第一次那种干涩的承受了。但张野自己不承认也不回应，只是沉默地喘着。
  信息差：张威通过水声和湿润度知道张野的身体开始有反应了 ↦ 张野自己不知道那是肠液，只觉得被操开了之后没那么干涩了
  情绪拐点：承受 → 身体出现不自主的生理反应（分泌肠液、发出水声）→ 被张威用「够骚的」定性时的羞耻但无法反驳
  → 下一场景的衔接：张威临近射精
  本场景焦点：人物情绪（张野身体出现不自主的兴奋反应——肠液分泌、水声——但心理上不承认也不回应，沉默的羞耻）
  本场景权重：中——对话和身体反应的对照，张野的心理vs身体的分离开始显现
  感官建议：触觉（20cm鸡巴撑开每一道褶皱的饱满感、肠道被操开后分泌肠液的滑腻感）+ 听觉（咕叽咕叽的水声、张威的脏话「出水了」「够骚的」、肉体撞击声）
  本场景出口情绪：身体背叛——不出声但身体出了水

**关键点 5｜推进剧情·中出｜核心场景①｜权重：高（本章情绪强度峰值之一）**
  核心事件：张威操到临近射精，动作突然加深——一手按住张野的腰，龟头死死顶在最深处，闷吼了一声：「射给你。」张野愣住，还没来得及反应，一股滚烫的精液就灌进肠道深处。那一刻张野大脑一片空白——陌生的温热感从体内最深处涌开，量很大，一股接一股喷在肠道内壁上。张野全身僵住，手指攥紧床单，嘴里发出一声被闷住的呜咽。张威射了七八股才停，鸡巴还在体内一跳一跳地抽搐。
  情绪意义：本章第一个核心高潮——第一次被内射的冲击。张野完全措手不及，大脑空白，身体僵住。精液的温度、量、冲击力都是第一次体验。这个场景的强度在于「陌生」——张野不知道这个感觉会是什么样，知道了之后愣住了。
  信息差：张野不知道张威会内射（措手不及）↦ 张威知道这是他的常规玩法（中出上瘾）
  情绪拐点：被操到出水的状态 → 张威突然加深动作按住腰 → 被内射的瞬间——大脑一片空白、全身僵住 → 感受精液一股接一股灌入的快感和冲击并存

  本场景焦点：核心冲突（第一次被内射的瞬间——张野的措手不及、精液灌入肠道的触觉冲击、大脑空白身体僵住的反应。这是本章最重要的感官场景。）

  本场景出口情绪：被灌满的震惊——大脑一片空白，身体记住了这一刻

**关键点 6｜推进剧情·堵精｜核心场景②｜权重：高（本章情绪强度峰值之二）**
  核心事件：张威射完没拔出来——堵着。他停了几秒，等精液在肠道里浸润开，然后又开始慢慢抽插。鸡巴在灌满精液的肠道里滑进滑出，白沫从穴口溢出来，顺着张野的大腿根往下流。张野低头看到自己身下的床单上沾了一圈白色的浊液——自己被内射的精液和抽插搅出来的泡沫。张威越操越快，每一次抽插都带出更多白沫。张野被这种视觉和触觉的双重冲击弄懵了。
  情绪意义：堵精场景的视觉冲击比中出更强烈——张野低头看到自己身下的精液和泡沫，意识到「这些东西是从我体内流出来的」。视觉刺激加上鸡巴在灌满精液的肠道里继续抽插的触觉，双重冲击。
  信息差：张野不知道射完还可以堵着继续操 ↦ 张威的中出癖好包含「不拔出来，堵着继续操，看白沫溢出」

  → 下一场景的衔接：张威终于拔出来
  本场景焦点：核心冲突（堵精的视觉和触觉双重冲击——鸡巴在灌满精液的肠道里继续抽插的滑腻触感、白沫溢出的视觉画面、张野低头看到床单上精液和泡沫的震惊）
  本场景权重：高——必须写白沫溢出的画面（穴口、顺着大腿根流、床单上的浊液圈），以及鸡巴在精液润滑下继续抽插的滑腻触感。视觉冲击是本章最强烈的画面之一。
  感官建议：触觉（鸡巴在灌满精液的肠道里滑进滑出的滑腻特殊触感——比正常操更滑、更湿、有泡沫感；白沫流到大腿根儿的粘腻感）+ 视觉（张野低头看到的画面——自己身下床单上一圈白色浊液、白沫从穴口一股一股溢出、精液顺着大腿往下流）
  本场景出口情绪：被弄懵的双重冲击——视觉和触觉都在告诉张野：你被灌满了

**关键点 7｜过渡·对话·收尾｜指令含精｜基调｜权重：中**
  核心事件：张威终于拔出来，龟头离开穴口时发出「啵」的一声轻响——堵在里面的精液没了阻挡，开始往外流。张威伸手在张野的肛门口拍了一下，说：「含着，别浪费。」张野脸红到脖子根，下意识收缩了括约肌——精液被夹住了，温热的、粘稠的、满满一肚子。张威擦了擦鸡巴丢给他一张纸巾：「擦擦，回去吧。」
  情绪意义：指令式的收尾——「含着，别浪费」不是商量，是指令。张野下意识服从。这个场景的关键在于「下意识」三个字——他没有想就照做了，身体比脑子先反应。
  信息差：张野不知道为什么要「含着」也不知道含到什么时候 ↦ 张威知道这是他的惯例——中出后让受方夹精离开，保持占有的仪式感
  情绪拐点：被弄懵的状态 → 「啵」的一声拔出来 →「含着，别浪费」的指令 → 下意识收缩括约肌夹住精液的服从
  → 下一场景的衔接：张野穿内裤，发现精液被内裤压着贴在肛门口
  本场景焦点：人物情绪（指令式收尾——「含着，别浪费」作为张威的占有仪式；张野的服从展示了身体对指令的自动反应）
  本场景权重：中——对话简洁有力，是张威性格的集中展示
  感官建议：触觉（龟头拔出来时括约肌的突然一空的感觉、肛门口被拍了一下的轻微痛感、夹住精液时括约肌收缩的体感）+ 听觉（「啵」的一声、张威简短的指令「含着，别浪费」「擦擦，回去吧」）
  本场景出口情绪：被命令的服从——身体已经学会听话了

**关键点 8｜推进剧情·堵精的回程｜核心情绪场景｜权重：高**
  核心事件：张野穿上内裤的那一刻，精液被内裤压着贴在肛门口。他站起来，每走一步都能感觉到那泡精液在肠道里轻微晃动。夹着肛门的姿势让走路变得不自然——步子迈得小、腿夹得紧、生怕一个动作没控制好精液就漏出来。从健身房到公交站那段路走得特别漫长。上了公交不敢坐，怕一屁股坐下去精液涌出来。一路夹着屁眼，脸颊烫得发红——不是因为羞耻的表情，是生理上的紧张和慌乱。
  情绪意义：夹精回程的羞耻感是本章的第二个情绪峰值——精液在肠道里的晃动感、夹着肛门走路的不自然、不敢坐下怕涌出来的紧张。这个场景让羞耻感达到最高点，也是最容易让读者共情的部分。
  信息差：张野不知道其他人能不能看出来他在夹着屁眼走路 ↦ 实际上没人注意他，但他自己觉得每一步都像被标记了
  情绪拐点：被命令的服从 → 穿上内裤时精液被贴住的触感 → 站起来走路时精液晃动的陌生体感 → 夹着肛门走路的羞耻 → 上公交不敢坐的紧张
  → 下一场景的衔接：终于回到出租屋
  本场景焦点：人物情绪（夹精回程的全程羞耻——每一步都能感觉到精液的存在、不敢坐下怕涌出来、脸颊发烫是生理紧张不是表情）
  本场景权重：高——这是本章情绪长度的展现，不是几秒的羞耻，是一条路的长度（健身房→公交站→公交车上→走回出租屋），羞耻感的累积效应
  感官建议：触觉（精液在肠道里随着步伐轻微晃动的体感——温热、粘稠、有重量；内裤压在肛门口精液贴着皮肤的触感；夹紧肛门走路时括约肌的紧张感）+ 视觉（夹着腿走路的体态、不敢坐公交站着的姿势、脸颊烫得发红但没有表情的样子）
  本场景出口情绪：羞耻的高峰——全身都在提醒他体内含着另一个男人的精液

**关键点 9｜过渡·情绪·章末落点｜诡异的满足感｜权重：中**
  核心事件：张野终于回到出租屋，锁上门脱了裤子蹲下。括约肌一松，那泡已经被体温焐热的精液从穴口涌出来，滴在地板上汇成一滩白浊。他蹲着低头看那滩精液，喘着粗气——脑子里还在回放被灌满的那几秒。羞耻是确定的。但身体深处有一种说不清的感觉——不是疼，不是难受，是一种陌生的、填满的、沉甸甸的……诡异满足感。他用手背擦了擦流到大腿上的精液，站起来去拿纸巾。没有镜子，但他的脸烫得自己能感觉到红色从脖子根蔓延到耳根。
  情绪意义：章末落点——羞耻是确定的，但「诡异的满足感」出现了。这不是爽，不是喜欢，是一种说不清的、身体层面的满足——被灌满的充实感，被填满的从未体验过的感觉。这粒种子会在第4章「习惯」继续生长。
  信息差：张野说不清「诡异满足感」是什么 ↦ 读者知道这是身体开始记住被灌满的感觉，为第4章的「习惯」和第5章的「操尿」做铺垫
  情绪拐点：羞耻的高峰 → 回到安全空间锁上门 → 蹲下排精时精液涌出的视觉 → 低头看那滩精液时的复杂情绪 → 诡异的满足感（模糊的、说不清的、身体层面的）
  → 下一场景的衔接：第4章「习惯」——张野发现身体开始期待
  本场景焦点：人物情绪（章末落点——耻感与诡异满足的并存。重点在「说不清」和「身体记住了」，不是「喜欢」）
  本场景权重：中——章末收束，约250-350字。核心在蹲下排精的画面和章末的诡异满足感落点
  感官建议：触觉（括约肌松开时精液涌出的体感——温热的、已经不烫了、被体温焐热的温度；精液滴在大腿上的粘腻触感）+ 视觉（地板上汇成一滩的白浊精液、张野蹲着低头看的姿态、手背擦大腿上精液的动作）
  本场景出口情绪：羞耻但身体记住了——诡异的满足感，说不清但真实存在

## 不可违反规则

### 优先级（从高到低）

1. **约束红线** — 情节红线、角色禁区、边界禁止
2. **字数** — 3500-4000字目标（超限启用压缩策略）
3. **T1词 / 认知动词 / 感官** — 修饰阈值、认知动词节制、感官分层
4. **叙事规则**（高于同层·写作规范其他规则）
5. **写作规范同级其他规则**

### 红线约束

**冲突阶梯**：第2层·升级——本章以肉体的升级为主要冲突（张野的身体开始适应vs心理的尚未接受），已脱离第一阶段的疼痛，进入「能够承受」期。

**情节红线（不可删减）：**
1. **中出场景必须写实写细**——精液滚烫的温度、量大、一股接一股喷在内壁上、张野大脑空白全身僵住的反应。不可跳过或概括为「他射在里面了」
2. **堵精场景必须写**——张威射完没拔出来、堵着精液继续操、白沫从穴口溢出、张野低头看到床单上的精液和泡沫。不可省略堵精环节
3. **「含着，别浪费」必须写**——这是张威指令式收尾的标志性对话，也是张野服从的瞬间
4. **夹精回程必须写**——从穿内裤到走路到公交站到不敢坐车，每一步的羞耻感和精液晃动的体感。不可跳过或简化为「他一路夹着精液回去了」
5. **章末的「诡异的满足感」必须写**——蹲下排精后的复杂情绪——羞耻是确定的，但身体深处有一种说不清的满足感。终点落在这粒种子上

**边界禁止（不涉及）：**
1. 不要让张野在章末产生「喜欢」被中出的心理——只是「诡异的满足感」和「羞耻」，距离完全接受的「爽」还差好几章
2. 不要让张威流露出任何温柔或夸奖——保持纯粹征服者的粗暴。「学得挺快」是工具验收式的评价，不是夸奖
3. 不要写张野被内射后「想要更多」或主动要求——他被内射时的反应是愣住、大脑空白、全身僵住，不是主动迎合
4. 不要让张野在回程路上遇到熟人或被认出——保持私密的羞耻过程
5. 精液量和温度的描写要写实——不要夸张成喷泉（七八股，量很大，但写实），也不要省略成一句带过
6. 不要写成分体位的大段机械描述——肉感优先，体位服务于感官描写

**角色禁区：**
张野不会在被内射后说「还要」（他愣住了，大脑空白，不可能说话）
张野不会在被命令「含着」时回答（他沉默地照做了）
张野不会主动找张威聊天或套近乎（每次都是被动+沉默的接受）
张威不会关心张野的感受（一句都不问——「疼不疼」「舒服吗」「还好吗」一概没有）
张威不会叫张野的名字（直接用动作和指令交流）
张威不会说超过10个字的句子（短句、指令式、带脏字——「操，学得挺快」「屁股夹这么紧」「射给你」「含着，别浪费」「擦擦，回去吧」）

### 网文风格基线（任何章节适用）

1. 认知动词节制使用（≤2次/章——「感到」「觉得」「意识到」等）
2. 因果关系自然呈现，不强行解释
3. 叙事自然有温度——禁止极端白描（纯动作描写无情感流动）
4. 全程线性叙事
5. 拒绝完美叙事——允许留白和思绪跳跃
6. 禁止刻意堆砌技法
7. 禁止诗词腔

### 风格建议（可偏离）

- **视角**：第三人称有限，紧贴张野。不写他人内心。禁止全知解释。
- **描写**：用动作感知推动叙事，感官服务于情绪。本章以**触觉（精液灌入肠道的滚烫/被撑开填满的饱满感/白沫流过大腿的粘腻/夹精走路时肠道里的晃动感）和视觉（白沫溢出/床单上的精液/地上那滩精液）** 为主要感官。听觉（脏话/水声/撞击声）辅助。
- **节奏**：段落自然起伏，句式长短交替。性爱场景要有递进：插入→粗话→中出→堵精→余韵。中出时节奏收紧（短句+急促动作），堵精时视觉描写放慢（让读者看到画面），夹精回程节奏稍微放慢（羞耻感的累积需要时间感）。
- **张野的说话方式**：几乎不说话。被操时闷哼、粗喘、呜咽为主。被命令时沉默服从。没有成句的对话。
- **张威的说话方式**：粗犷、直接、带脏字、短句为多（「屁股夹这么紧」「操，学得挺快」「出水了，够骚的」「射给你」「含着，别浪费」「擦擦，回去吧」——不多话，句句指令）。
- **量化写实**：张威的鸡巴20cm/粗直/像涂了油的健身哑铃杆，精液量七八股/量很大，每次抽插都顶到底，龟头撞在前列腺上。数字和对比（第一次vs第二次）增加真实感。
- **身体适应进度**：从第一次的「撕裂痛」到第二次的「酸胀感」，这是本章身体层面的进展——不写爽，写「从疼到能承受」的过渡。
- **脏话是张威的标签**——必须贯穿本章。但张野不说话，所以脏话只来自张威一侧。
- **精液描写**：温度（滚烫→被体温焐热），状态（灌入时液体→堵精后泡沫化→排出时变温），量（一大泡，在地上汇成滩）。写实不夸张。

### 场景写作指引

本章涉及3种场景类型（rough-sex / dialogue / transition）：

**粗暴性交场景（rough-sex）—— 本章主体（关键点3-7）：**
1. **感官分层取触觉+视觉优先**——张野第二次被操，触感（撑开/填满/精液灌入/白沫溢出）和视觉（白沫从自己身下流出/床单上的精液）是核心描写通道。听觉辅助（脏话/水声/撞击声）。
2. **节奏控制**——插入（关键点3）→粗话抽插（关键点4）→中出（关键点5）→堵精（关键点6）→收尾指令（关键点7），逐级递进，中出和堵精是本体重头。插入的节奏是顺畅的、有力的；中出的节奏是突然的、冲击的（短句+急促感）；堵精的节奏是「放慢让读者看到画面再加速」。
3. **情绪推进**——比身体推进更重要。每一阶段的情绪变化：紧张→被验收→承受→身体背叛（出水）→震惊被内射→被弄懵（堵精白沫）→服从指令→羞耻。身体动作服务于情绪线，不是反过来。

**对话场景（dialogue）：**
1. 本章对话集中在张威一侧，张野几乎没有对话——只有闷哼和粗喘。
2. 对话标签全部用动作替代（抬下巴、笑了一声、一手按住腰、伸手拍了一下肛门口），不写「张威说」。
3. 张威的对话简短（3-8个字），带脏字，指令式，不解释不询问。

**过渡场景（transition）：**
1. 关键点1（到健身房）压缩至150-200字：1句短信「来一趟」+ 1句站在门口犹豫 + 1句推门进去 + 1句看到张威 + 1句走过去
2. 关键点7（收尾指令）压缩至200字：「啵」一声拔出来 + 指令「含着，别浪费」+ 张野下意识服从 + 「擦擦，回去吧」
3. 关键点9（章末落点）250-350字：回出租屋锁门 + 脱裤子蹲下 + 精液涌出 + 低头看那滩精液 + 诡异的满足感

### 质感要求

**无用细节（不服务主线的细碎生活化细节）：**
1. 健身房地面的橡胶垫味道——有点旧的、汗味混着消毒水的气味
2. 休息床的塑料皮面——被体温捂热了但是滑的，翻身时有轻微的摩擦声
3. 张威手机里放的音乐——随便放点什么，背景音
4. 窗外透进来的光线——下午的、不算亮的、斜射进健身房
5. 公交车上的味道——傍晚公交的闷、尾气混着人体味
6. 出租屋门锁的声音——锁上的咔哒声，回到安全空间的信号

**对话节奏：张威的对话必须短、脏、指令式**
- 「来一趟」（短信，三个字）
- 「操，学得挺快」（带脏字的验收）
- 「屁股夹这么紧，是不是等了一天？」（羞辱式）
- 「操，出水了，够骚的」（确认身体反应）
- 「射给你」（中出宣告，就两个字）
- 「含着，别浪费」（指令，不是商量）
- 「擦擦，回去吧」（逐客令）

**段落精度分层（按权重分配笔墨）：**
- **高权重场景（关键点5、6、8）**：细节饱满，至少3处动作/感知层级细节用于核心冲突/情绪展开。关键点5细化中出的全过程（精液温度/量/节奏/张野大脑空白）；关键点6细化堵精的视觉冲击（白沫如何溢出/床单上的精液圈/张野低头的震惊）；关键点8细化夹精回程的漫长羞耻（每一步都感觉到精液的晃动/不敢坐的紧张/脸颊发烫的生理反应）
- **中高权重场景（关键点3）**：正常叙事节奏，和第一次插入形成对比（不疼了但酸胀）
- **中权重场景（关键点2、4、7）**：推进剧情和对话，展示张威的控制和张野的被动
- **低权重场景（关键点1、9）**：快速转场和章末收束

**额外约束（本章特有）：**
1. **中出场景必须写细**——不可省略为「他射进去了」。必须写精液灌入时的温度感、量感、节奏感（一股接一股），以及张野的生理反应（大脑空白、全身僵住、攥床单、被闷住的呜咽）。这是读者期待的核心场景。
2. **堵精场景必须有视觉冲击**——白沫从穴口溢出、顺着大腿往下流、床单上被精液和泡沫浸湿的痕迹。让读者「看到」而不是「知道」。
3. **「含着，别浪费」之前的「啵」一声必须写**——龟头拔出来时穴口被堵住的精液解除压力的瞬间，这个声音和画面的细节让堵精的意象更完整。
4. **夹精回程必须有时间感**——「健身房→公交站→公交车上→走回出租屋」是一条有距离的路，不是一步到位。羞耻感需要距离来累积。每一步都能感觉到精液的晃动，是核心。
5. **章末的「诡异的满足感」要模糊化**——不写「他喜欢被灌满」，写「说不清」「不是疼不是难受」「陌生的、填满的、沉甸甸的」「身体深处有一种感觉但又说不清是什么」。
6. **张威的脏话水平**——「操」「骚逼」「够骚的」是他的语言体系，但不用太过分的脏话（不要「贱狗」「母狗」等——那些属于后续更堕落的阶段）。
7. **张野的身体反应要符合当前堕落阶段**——闷哼、粗喘、攥床单、呜咽、脸烫。不写翻白眼、流口水、痉挛（那些属于完全沉沦后的反应）。

### 附录：禁止配替代速查

| ❌ 禁止 | ✅ 替代 |
|---------|--------|
| 直述情绪结论（「张野感到羞耻」） | 用动作/神态外化（「步子迈得小、腿夹得紧、脸颊烫得发红」） |
| 章末升华总结（「这一章改变了他」） | 停在画面/动作上（「他蹲着低头看那滩精液，站起来去拿纸巾」） |
| 副词修饰对话（「张威粗犷地说」） | 去掉副词，用动作暗示（「张威笑了一声：'操，学得挺快'」） |
| 上帝视角（「他不知道这只是开始」） | 留在张野视角内（只写他当下的感受和动作） |
| 「张野心想/意识到/感觉到」 | 直接写事实/动作 |
| 「仿佛/犹如/如同」 | 删除或白描 |
| 「一丝/一抹/些许」 | 具体量词或删除 |
| 「瞳孔一缩/身体一僵」 | 用具体反应替代（「他攥床单攥得指节发白」「他整个人僵住了」） |
| 「突然/忽然」 | ≤4次/章，保留节奏功能的 |
| 「渐渐/逐渐」 | ≤2次/章，用变化细节替代 |
| 精液量夸张（喷泉/水枪） | 写实（七八股，量很大，但不夸张） |

| 张威温柔/关心（「疼吗」「还好吗」） | 一句不问——指令式、粗暴、验收工具的态度 |
| 「舒服」「爽」「快感」（描述张野） | 禁用——张野目前的状态是「承受」和「陌生」，不是爽 |

## 附录：情绪锚点与字数参考

**第三章·中出 情绪锚点（关键定格画面）：**

1. **「操，学得挺快」**（关键点2）——张威检查后发现不再那么紧，笑了一声说出这句话。张野趴在床上，身体绷了一下但没有躲。这是身体适应进度的第一次被确认。

2. **「射给你。」精液灌入的那一刻**（关键点5）——张野愣住，大脑一片空白。滚烫的精液一股接一股喷在肠道内壁上。全身僵住，手指攥紧床单，嘴里发出一声被闷住的呜咽。这是本章最强烈的触觉画面——第一次被内射。

3. **张野低头看到床单上白沫溢出**（关键点6）——从穴口溢出来的白沫顺着大腿根往下流，床单上沾了一圈白色的浊液。视觉冲击——「这些东西是从我体内出来的」。

4. **张野夹着精液走回出租屋**（关键点8）——步子迈得小、腿夹得紧、不敢坐公交。每一步都能感觉到精液在肠道里晃动。羞耻的最高点。

5. **蹲下排精，低头看那滩精液**（关键点9）——括约肌一松，被体温焐热的精液涌出来滴在地板上汇成一滩。他蹲着看，喘着粗气——羞耻是确定的，但身体深处有一种说不清的、诡异的满足感。

**字数分配参考：**
- 关键点1（过渡·第二次约）：150-200字
- 关键点2（检查·验收）：250字
- 关键点3（插入·顺利进入）：350字
- 关键点4（抽插·粗话）：350字
- 关键点5（中出·核心场景①）：700字（本章字数最多的场景——细化被内射的全程感官）
- 关键点6（堵精·核心场景②）：550字（视觉冲击为主，白沫溢出画面）
- 关键点7（收尾·指令）：200字
- 关键点8（夹精回程）：600字（羞耻感的累积，需要写距离感）
- 关键点9（章末落点·诡异满足）：300字
- 预留弹性：150字
- **合计：3500-4000字**

### × 不需要 ★ GV风格追加指令（本章虽是粗暴性爱，但张野尚未完全堕落，不需要GV硬核追加）

**本章不适用 GV 追加指令，原因：**
- 张野还在「承受」阶段——被内射的反应是愣住、大脑空白、全身僵住，不是淫荡地迎合
- 张野不说话——没有成句的对话，只有闷哼和呜咽
- 章末情绪终点是「诡异的满足感」（模糊的、说不清的），不是「被操爽了」
- 张野的身体反应是含蓄的——攥床单、呜咽、脸烫、夹紧肛门走路，不是流口水翻白眼
- 不写群体、不写道具、不写公开——这是张威的私下一对一征服
- 中出和堵精的描写以写实和冲击为主，不以GV的夸张淫秽语风为主
- **本章做的是让读者感受到「身体开始记住被灌满的陌生感」，不是GV式的纯肉刺激**
