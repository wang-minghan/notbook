# Prompt — 第一卷·第7章「背叛」

## 角色

你是一个都市题材（urban）小说作家。现代都市背景，所有场景发生在真实城市空间——大学校园、公交车、城市边缘烂尾楼工地。不涉及修仙、穿越、异能战斗等非都市元素。

本章是**第一卷（初恋与破处）第7章**——爱情线的断裂点：从怀疑到确认到心死。陆辞跟踪张野到工地，拍下张野跟赵铮做的视频，发消息分手。全书唯一一章以陆辞视角（第三人称有限）为主的章节。基调是压抑、沉默、心碎——没有激烈争吵，只有让人窒息的安静决绝。

**风格定位：** 压抑·心碎·无肉戏。前半段跟踪营造紧迫感（动作推进为主），中段目睹给出确认（通过手机屏幕），后段分手+张野崩溃（沉默的结局）。笔触克制，不写眼泪不写嘶吼——力气全在动作和安静里。

---

## 任务指示

创作第一卷·第7章正文（在全卷中的角色：爱情线断裂点——第6章「裂缝」结束时空出了陆辞的反应（翻到手机记录后没有揭穿），第7章给出完整闭环：陆辞从怀疑到跟踪到确认到分手。读者等了这一章——从第6章结尾的压抑开始，他们知道裂缝不可逆转，在等「爆发的那一刻」。)

### 字数与压缩策略

- **目标字数**：3000-3500字（±10%可接受，叙事完整性优先于字数精确）
- **压缩策略**：超字数时优先压缩过渡场景（场景1深夜决策/场景7发完消息走路），推荐压缩至80-120字——保留1句环境切换 + 1句人物动作 + 1句情绪留白；不得因压缩删改核心场景（跟踪过程、目睹拍摄、发消息分手、张野崩溃）

### 本章驱动力

陆辞从怀疑到确认到心死的全过程——读者想知道：陆辞怎么跟踪的？在工地上看到了什么？拍了什么？发消息时说了什么？张野看到消息时是什么反应？每个问题都要给答案，落在「沉默的心碎」上。

### 信息释放节奏

情绪沉浸型——读者跟随陆辞的视角经历从怀疑到确认到分手的全程：
- 开场：深夜宿舍，陆辞翻手机里张野的照片，做了决定——跟一次
- 推进：清晨楼下等张野→公交车跟踪→工地烂尾楼→找到位置
- 转折：目睹张野跟赵铮做→打开手机录像→拍完全程→关掉手机走人
- 小高潮：走出工地→站定→打字、删除、再打——发送分手消息
- 章末：陆辞走远了，手机在裤兜里震了一次又一次（不接不回）→ 切到张野蹲在工地角落崩溃（眼泪砸在屏幕上、发了很多条消息没被拉黑也没收到回复）

### 叙事目标

1. **核心悬念**：陆辞什么时候发现？怎么发现？发现之后做了什么？
   → 本章结束时：**全部兑现**——跟踪全程→目睹拍摄→发消息分手→张野崩溃
2. **读者情绪**：从压抑（第6章末尾的松口气却更堵）到跟随跟踪的紧张感、到目睹时的窒息感、到分手时的沉默心碎——终点落在「再也回不去了」
3. **爽点设计**：跟踪过程的紧迫感 + 目睹时通过手机屏幕看到的冲击 + 发消息时的极度克制 + 张野崩溃的反差释放
   - 铺垫于场景1-4（深夜决定→清晨出发→公交跟踪→进入工地）
   - 释放于场景5-6（目睹+拍摄——本章标志性段落）
   - 延续释放于场景7-9（发消息分手→张野崩溃→章末对照）

## 写作方法论

### 本章感官策略

本章以**视觉 + 动作**为主要感官通道——不是贴近身体的触觉（那属于张野的肉戏章节），是保持距离的观察感和执行感。

- **视觉优先：** 陆辞通过手机屏幕看完了全程——这是本章最重要的视觉设计。镜头里的张野（熟悉的轮廓、陌生的姿势）、水泥柱遮挡的视野边缘、手机屏幕的边界框住了全部真相。环境视觉也要到位：工地烂尾楼的光线变化（亮→暗→亮）、空旷空间里的粉尘、水泥碎块
- **动作主导：** 陆辞的动作干净利落——拿手机、打开录像、手指在半秒停顿后按下录制、拍完关掉装进口袋、转身走。动作链就是情绪链。打字时的犹豫（打了删、删了打、再打再删）是唯一暴露情绪波动的动作
- **听觉克制：** 工地空旷的回声、踩到碎砖的声音、远处传来的模糊声响（不写清楚是什么声音）、手机在裤兜里震动的声音。不写环境音过多渲染——留白比填满更有力
- **触觉极简：** 本章几乎不使用触觉描写——只有在关键处用一下：风/阳光打到脸上的感觉（走出工地后重新感知外界）、手机在手里或兜里的触感。触觉属于张野，不属于陆辞的旁观视角

### 叙事节奏交替

一段视觉/动作推进 + 一段短停顿（陆辞站定/沉默） + 一句环境切换。跟踪段节奏紧凑（动作链短促连续），分手段节奏放慢（打字犹豫、走路、不接电话——节奏从紧到松的落差暗示心死的过程）。

本章节奏：前紧后松——跟踪段快节奏推动（步速控制、公交跟踪的回避动作），目睹段突然静止（站在柱后录像的十几个呼吸），分手段重新启动但速度变了（走的步速、打字的犹豫，都和前文的果断形成对照）。

### 信息释放控制

陆辞的情绪不直接写——通过动作外化（拿手机录像前的半秒停顿、打字时的删了又打打了又删、发完消息关屏幕没有再看一眼）。不写「陆辞心里在想什么」，写他做了什么。

张野的崩溃在章末——只写外部动作（眼泪砸在屏幕上、蹲着发抖、不停发消息直到手指发酸），不写张野的心理活动。

## 背景信息

### 前情上下文

第6章「裂缝」结尾画面：陆辞在张野手机里翻到了工地地址和跟赵铮的聊天记录（内容隐晦但足以判断张野在做什么）。他把手机放回原位，没有当面问。张野从厕所出来时陆辞正在看手机——什么都没说。两人像往常一样一起吃了个晚饭，但张野没怎么说话，陆辞也没问。——读者知道陆辞看见了、知道了，但不知道他怎么处理。

读者情绪残留：压抑的等待——裂缝已经打开，读者在等「爆发的那一刻」。第6章结束时两人吃了一顿沉默的晚饭，读者比张野更紧张——因为读者知道陆辞已经翻过手机了。

第6章留下的缺口：陆辞知道张野在卖身，但他没当面揭穿。读者在等陆辞的行动——他会不会当面质问？分手？还是装作不知道？本章给出答案：他选择跟踪亲眼确认。

### 角色初始状态

**陆辞（本章主视角）：**
  起点：深夜宿舍，室友睡了。手机屏幕亮着，他在翻相册里张野的照片——不是最近拍的，是之前在一起时攒下的。看完后关掉相册，打开地图查了一下工地位置。然后躺下。没睡——只是躺着等天亮。
  → 经历：清晨出发→跟踪→目睹→拍摄→发消息分手→走远
  → 终点：走在路上，不接电话不回消息，脚步没有停。
  本章必须完成的一件事：从怀疑到确认到分手——完成陆辞单方面终止关系的全过程。
  说话方式：本章几乎无对话，性格通过动作展现——冷静、理性、沉默、决绝。
  **关键词：** 克制、冷静、动作干净、不说话。

**张野（章末辅视角，只写外部动作）：**
  起点：清晨从宿舍出来去工地，穿旧运动服，低着头走路。
  → 经历：到工地→被赵铮按在水泥地上干→被灌精→蹲在角落拿手机→看到陆辞的消息→崩溃
  → 终点：蹲在工地角落，手机屏幕暗了又亮，没收到回复，没站起来。
  本章必须呈现的一个情绪峰值：看到分手消息时的崩溃（眼泪砸在屏幕上、发了无数条消息、蹲着很久没站起来）。
  说话方式：发消息时也是低声闷闷的语气——「对不起对不起对不起」不是求饶是认罪。
  **关键词：** 崩溃、沉默的眼泪、知道自己活该。

## 输入

### 场景原材料

**场景 1｜过渡·深夜决策｜冷静基调｜权重：低**
  核心事件：深夜，宿舍。室友的呼吸声均匀——都睡了。陆辞没睡。手机屏幕亮着，他翻到张野的照片——不是最近拍的，是两个月前一起去小吃街时他偷拍的张野吃东西的样子。屏幕光打在他脸上，看不出表情。他把相册关掉，打开地图——搜索框里有历史记录：城乡结合部的一片工地。他看了几秒，退出，关屏幕，把手机扣在枕头边。躺平。天花板上的灯影模糊。他闭上眼睛——没睡，就是躺着等天亮。
  情绪意义：读者看到陆辞已经做过功课了——他不是临时起意，他查过地点、做过决定
  信息差：陆辞已经知道答案（从手机记录）→ 读者和陆辞知道同样的信息
  情绪拐点：深夜的安静 → 陆辞的冷静决策（没有什么激烈的心理斗争，就是无声地做了决定）
  → 下一场景的衔接：清晨，陆辞已经在张野宿舍楼下
  本场景焦点：人物情绪（冷静地做决定——不写内心活动，只写动作）
  本场景权重：低（快速过渡，约80-120字）
  感官建议：视觉（手机屏幕光打在脸上/暗掉的屏幕/天花板模糊的灯影）+ 听觉（室友的呼吸声、翻身的床响）
  本场景出口情绪：冷静的确定——没有犹豫

**场景 2｜推进剧情·楼下等待+跟踪出发｜紧张基调｜权重：中**
  核心事件：清晨。天刚亮透，校园里人很少。陆辞站在宿舍楼拐角——不是正对门口的位置，是斜角，能被看到也能随时闪开的位置。他帽檐压得很低。等了二十分钟左右，张野出来了——穿旧运动服，背着个黑色斜挎包，低着头走路，没有往四周看。陆辞等他走出去三十米左右才从墙角出来，保持距离跟上去。步速控制是重点——张野走得不算快，陆辞也不快，中间隔着几个早起去食堂的学生。校园清晨的风有点凉，操场方向有跑步的人。
  情绪意义：读者跟陆辞一起进入「跟踪」状态——紧张感从这里开始累积
  信息差：张野不知道有人在后面跟着 ↦ 读者知道——紧张感来自信息不对称
  情绪拐点：清晨的平静 → 陆辞开始跟踪，紧张感出现
  → 下一场景的衔接：公交车上了同一辆车
  本场景焦点：人物动作（跟踪的保持距离、步速控制、压低帽檐、避开视线）
  本场景权重：中——跟踪氛围的起点，细节决定紧张感
  感官建议：视觉（张野走出来的身影、帽檐遮挡的视野边缘）+ 触觉（清晨的凉风）+ 动作（从墙角闪出来到保持距离的步速变化）
  本场景出口情绪：紧张但不慌乱——陆辞在控制节奏

**场景 3｜推进剧情·公交车跟踪｜紧张基调｜权重：中**
  核心事件：公交车来了。张野先上，扫了码往后走——坐到了后排靠窗的位置。陆辞停在站牌前假装看了一眼路线，等张野坐定了才上车。他刷卡后没有往后走——坐在了前门附近的第一排单人座上，侧对着车厢。公交车开了，外面的街景从校园变成城市道路。陆辞没有回头，但通过公交车左侧后视镜的反光能看到张野的位置——张野一直低头看手机，偶尔打几个字。但中途有两次，张野突然抬起头往后看了一眼——没有特定目标的那种回头看，像是一种本能的警觉。第一次陆辞正对着窗外，他保持姿势没动。第二次陆辞低头假装刷手机——余光确认张野转回去了。
  情绪意义：跟踪的细节动作让读者代入——张野在警惕（以前不会回头看），陆辞在回避（每一次被张野的警觉逼退）
  信息差：张野隐约觉得不对劲但不确定 → 陆辞知道张野心里有鬼
  情绪拐点：平稳的跟踪节奏 → 张野回头时的一瞬间紧张 → 回避成功后的继续跟踪
  → 下一场景的衔接：到站，张野下车
  本场景焦点：人物动作（陆辞跟踪时的位置选择、回避动作、通过后视镜观察）+ 张野的反常（回头两次——以前不会这样）
  本场景权重：中——跟踪细节决定真实感
  感官建议：视觉（公交车内的布局/后视镜里的反光/窗外的街景变化）+ 动作（刷卡后选择座位→面对车窗保持姿势→被回头看时低头刷手机）
  本场景出口情绪：紧张升级——张野的警觉让跟踪难度增加，读者更投入

**场景 4｜推进剧情·环境·工地｜氛围基调｜权重：中**
  核心事件：公交车停在城郊的一个站——周围没什么人，路边是围挡和杂草。张野下车了。陆辞没有马上跟——他坐在座位上等公交车门关了又开了另一侧门，才站起来下车。下车后他在站牌旁边站了一会儿，假装看手机——实际上在确认张野的方向。张野已经走远了——往围挡后面的一条土路方向走。陆辞跟上去。走了几分钟，街景变了——从有零散行人的路边变成了人烟稀少的烂尾楼片区。铁皮围挡生锈了，广告布被风撕了一半，路面是碎石和灰土。张野绕过一个缺口钻进了围挡，陆辞站在缺口处停了几秒——然后跟进去。光线从亮变暗，水泥粉尘的气味飘过来，踩到碎砖的声音很清脆。空旷——什么声音都没有的时候特别空旷。
  情绪意义：环境变化暗示关系的走向——从校园到荒芜地带，从甜蜜到终结
  信息差：张野不知道有人跟进来了 ↦ 读者知道，紧张感在累积
  情绪拐点：城市道路→烂尾楼片区→进入围挡→光线从亮到暗
  → 下一场景的衔接：陆辞听到声音，找到了位置
  本场景焦点：环境描写（感官侧重三样：水泥粉尘味、空旷、踩到碎砖的声音、光线明暗变化）
  本场景权重：中——环境是情绪的重要载体
  感官建议：视觉（围挡缺口/烂尾楼外观/光线变化）+ 嗅觉（水泥粉尘味）+ 听觉（自己的脚步声在空旷楼体里的回音、远处模糊的声音）+ 触觉（踩到碎砖的脚底感）
  本场景出口情绪：空洞的紧张——环境把一切声音放大了

**场景 5｜推进剧情·目睹｜冲击基调｜权重：高**
  核心事件：陆辞循声找到位置——声音从二楼半层平台传下来，空旷的楼体把声音略微放大了一些，但他听不清具体内容。他放轻脚步顺着没有护栏的楼梯上去，在转角处停住了脚步。从水泥柱的侧后方能看到二楼的平台——看清的那一瞬间，他没有动。张野到了。跟另一个人——一个体型比张野大一圈的男人（赵铮），背对着陆辞的方向。半层平台上铺着一张破旧的防尘布。陆辞站在水泥柱后面，没有往前也没有后退，站了大概三到五秒——然后他拿出手机，打开到录像模式。手指在屏幕的红点上停了半秒。然后按了下去。
  情绪意义：本章的转折点——陆辞亲眼确认了，读者跟陆辞一起看到了
  信息差：陆辞看到了真相——信息差清零，他知道了全部（从他的视角能知道的全部）
  情绪拐点：跟踪的紧张感 → 目睹时的一瞬间凝固 → 拿出手机录像的冷静执行
  → 下一场景的衔接：通过手机屏幕看完了全程
  本场景焦点：核心冲突（目睹的瞬间——不做心理描写，只写陆辞的动作链：找到位置→看清→站定→拿手机→开录像→手指在半秒停顿后按下录制）
  本场景权重：高——本章最重要的场景之一。注意不写赵铮的详细样貌、不写张野和赵铮的具体互动、不写肉戏。通过陆辞的动作和位置限制写
  感官建议：视觉（水泥柱后看到的半层平台/张野的熟悉轮廓/赵铮的大一圈体型）+ 动作（拿手机→开录像→手指在红点上停了半秒→按下录制——动作链要干净）
  本场景出口情绪：确认的窒息感——不是震惊也不是愤怒，是一种冷静的确认
  注意：不写陆辞的内心感受——只写他的动作链
  注意：不写赵铮的样貌、台词、名字——只交代体型比张野大一圈、动作粗暴
  注意：不让陆辞和张野面对面碰上

**场景 6｜推进剧情·拍摄｜冲击基调｜权重：高**
  核心事件：陆辞通过手机屏幕看完了全程。镜头框住了两个人——张野的背影和赵铮的大部分身体。手机屏幕的边界把画面切割成框内的真实和框外的沉默。张野被按在水泥地上的防尘布上——是他熟悉的轮廓，但姿势陌生——他没见过张野在这种位置。陆辞没有移开手机，他没有放大画面也没有缩小——就是保持一个固定的焦距录着。他的呼吸没有变快，手也很稳。不知道录了多久——可能是几分钟，可能更长。最后画面里赵铮的身体绷紧了一下（射了），然后退开。过了一会儿张野慢慢坐起来。陆辞在手机屏幕上点了一下——停止录制。他把手机从面前移开，手放下来，在裤子侧面蹭了一下屏幕上的灰，然后装回口袋。转身。没有再看一眼。
  情绪意义：通过手机屏幕这个媒介——陆辞用旁观者的视角"看完了全程"。手机屏幕既是工具也是盾牌
  信息差：陆辞在手机屏幕后见证了整个过程 → 读者通过陆辞的动作和视角感知一切
  情绪拐点：录像中（通过屏幕观看）→ 录像结束（移开手机→装口袋→转身走——动作干净）
  → 下一场景的衔接：走出工地，走到外面站定
  本场景焦点：核心冲突（拍摄的过程——通过手机屏幕看全程是本章最核心的设定。不写陆辞看到的具体画面细节（不写肉戏），写他通过屏幕观看这个行为本身——不放大不缩小、不移动不回避、保持固定焦距录完）
  本场景权重：高——通过手机屏幕隔离的观看感是本章的情绪设计核心
  感官建议：视觉（手机屏幕里的画面——不写细节，写"熟悉的轮廓、陌生的姿势"）+ 动作（固定手机不动的稳定感→录完关掉→蹭掉屏幕灰→装口袋→转身走——没有停顿没有犹豫）
  本场景出口情绪：窒息的确信——录完了，关了，走了。没有任何多余动作

**场景 7｜推进剧情·发消息分手｜爆发基调｜权重：高**
  核心事件：陆辞走出工地。站到了外面——阳光有点刺眼，风打到脸上，有温度有湿度——终于不是工地里的水泥粉尘味了。他站了一会儿，拿手机。打开微信，找到张野的聊天框。手指悬在屏幕上方——打了几个字，删掉。又打了几个字，又删掉。屏幕上的输入框空了又出现、空了又出现。最后他打了：「我以为你只是忙。原来你在忙这个。我们到此为止。」打完看了一遍，没有修改——发送。然后把刚才的视频发了过去。做完这一切，他关掉屏幕。没有把张野拉黑——只是关了屏幕，把手机放回口袋。然后走了。
  情绪意义：本章最重要的台词——「我以为你只是忙。原来你在忙这个。我们到此为止。」三句话，三段停顿，句号结尾——最后一句话是句号，不是感叹号。因为不需要喊。
  信息差：陆辞发出了分手消息 ↦ 张野还不知道（下一场景）
  情绪拐点：录像后的冷静 → 工地的空旷 → 站定 → 打字的犹豫 → 发送后的决绝
  → 下一场景的衔接：切到张野那边
  本场景焦点：核心冲突（发消息的全过程——打字的犹豫（打了删、删了打）是陆辞唯一暴露情绪的地方。发送后关屏幕、没有拉黑、放进口袋、走了——每一个动作都在说"结束了"）
  本场景权重：高——本章台词最重的场景，虽然没有声音
  感官建议：视觉（阳光刺眼/手机屏幕亮光/聊天框里的文字被打了又删）+ 动作（站定的静止→打字时的犹豫→发送后关屏幕→放口袋→走了）
  本场景出口情绪：决绝但心碎——干净利落的终结，但越干净越痛
  注意：陆辞不哭——他是冷静决绝的那一方，不是崩溃的那一方
  注意：发送后没有拉黑——不拉黑但不回复是心死，不是愤怒
  关键台词：陆辞 — 「我以为你只是忙。原来你在忙这个。我们到此为止。」

**场景 8｜推进剧情·张野的反应｜爆发基调｜权重：高**
  核心事件：[切到张野——辅视角，只写外部动作] 张野那边——刚被赵铮灌了一肚子精。他腿还在抖，蹲在工地角落的水泥台阶上拿手机——想看看时间或者叫个车。屏幕亮了——陆辞的消息弹出来，两排预览文字。张野看到发件人名字的时候还在用另一只手抹额头上的汗。然后他看清了内容——他整个人僵住了。手机差点从手里滑掉。他低头又看了一遍，看到了视频——他点开看了三秒就关掉了。然后眼泪直接砸在了手机屏幕上——没有声音、没有前兆，第一滴眼泪掉在屏幕上溅开的瞬间他还在看消息。他没站起来——蹲在那里开始打字。发了一条：「不是你想的那样」——手指发抖，打错了两个字又改。又发：「陆辞你听我说」——发送。又发：「我操我骗了你」——发送。又发：「对不起对不起对不起」——一条接一条，中间没有停顿，手指在屏幕上戳得很快，眼泪糊了一脸他也没擦。他发了很多条——每一条都没被拉黑，但每一条都没有收到回复。屏幕上的最后一条消息是自己的绿泡泡——下面没有新的灰色字了。他就蹲在那里看着屏幕，屏幕暗了，他戳了一下又亮了，又暗了。他没有站起来。不知道蹲了多久。
  情绪意义：读者知道真相（系统逼迫）→ 张野的痛苦被额外放大 → 陆辞不知道真相 → 信息差让这一幕更残忍
  信息差：张野不知道陆辞跟踪了他全程/以为是从手机记录发现的/不知道陆辞亲眼看了全程 ← 读者知道全部 → 让分手场景更痛
  情绪拐点：刚做完交易的狼狈 → 看到消息的僵住 → 掉眼泪 → 拼命发消息 → 没收到回复的安静
  → 下一场景的衔接：切回陆辞那边
  本场景焦点：核心冲突（张野的崩溃——只写外部动作：眼泪砸在屏幕上、蹲着发抖、不停发消息直到手指发酸、屏幕暗了又亮亮了又暗。不写张野的心理活动）
  本场景权重：高——全章最痛的那一下
  感官建议：视觉（眼泪砸在屏幕上的瞬间/手指在屏幕上戳的动作/屏幕暗了又亮）/ 听觉（安静——只有手指戳屏幕的声响）/ 动作（僵住→发抖→发消息→一直蹲着没站起来）
  本场景出口情绪：剧烈的、无声的心碎
  注意：只写外部动作，不写张野的内心活动
  注意：张野发消息的语气也是低声闷闷的——「对不起对不起对不起」不是求饶是认罪
  注意：不恨陆辞——张野知道自己活该
  关键台词：张野（消息）—「不是你想的那样」「陆辞你听我说」「我操我骗了你」「对不起对不起对不起」

**场景 9｜造悬念·过渡·章末｜心碎基调｜权重：中**
  核心事件：两个方向，交替镜头，像电影章节的末端。① 陆辞走在路上——没有什么目的地在走，只是走了很远。手机在裤兜里震了一次两次三次四次——他摸出来看了一眼来电显示（张野的名字），他没接也没挂，把手机放回去继续走。没有拉黑——但也不会接了。走了很久，脚步声越来越轻，人影越来越小。② 张野还蹲在工地角落——手机屏幕暗了又亮，亮又暗了。陆辞没有拉黑他，但他也等不到回复了。他知道陆辞看到了什么——那段视频他自己都只看了三秒。他把手机放在旁边的台阶上，两只手抱在膝盖上——就蹲在那里，没动。两个人都很安静。
  情绪意义：章末对照——陆辞走远了（沉默的离开）vs 张野蹲在原地没动（无声的崩溃）。两种心碎，没有高下
  信息差：陆辞不知道张野蹲在工地哭 / 张野不知道陆辞走远了 ← 读者知道一切但什么都做不了
  情绪拐点：……没有拐点了——两个人都在各自的沉默里
  → 全章结束
  本场景焦点：人物情绪（不做总结——陆辞不接不挂一直走是心死/张野蹲着不动屏幕亮了又暗是崩溃。两个画面交替，让读者自己感受）
  本场景权重：中——章末收束，破碎感拉满
  感官建议：视觉（陆辞走在路上/裤兜里的手机亮光/张野蹲在角落/旁边的台阶上的手机屏幕亮了又暗）/ 听觉（远去的脚步声/手机震动的嗡鸣/工地空旷的安静）
  本场景出口情绪：心碎——不释放、不总结。停在这里

## 不可违反规则

### 优先级（从高到低）

1. **约束红线** — 情节红线、角色禁区、边界禁止（必须写/不能写）
2. **字数** — 3000-3500字目标（超限启用压缩策略）
3. **反AI规则** — T1句式/禁用词零容忍、认知动词节制、感官分层
4. **叙事规则** — 视角控制、节奏交替（高于同层·写作规范其他规则）
5. **写作规范同级其他规则**

### 红线约束

**冲突阶梯**：第2层·升级——本章从怀疑升级到确认分手，是张野恶堕的关键转折

**情节红线（不可删减）：**
1. **陆辞跟踪从深夜决策到清晨出发到公交到工地的完整过程必须写出**——跟踪细节决定这一章的真实感和压迫感
2. **目睹+拍摄场景必须通过手机屏幕的媒介来写**——陆辞通过屏幕看完全程，不写他直接看到的画面细节
3. **发消息分手的台词必须完整写出**——「我以为你只是忙。原来你在忙这个。我们到此为止。」三句话，句号结尾
4. **视频发过去的动作必须写**——陆辞发了文字后把视频也发了过去——他不需要张野解释，他用「你看到了，我也看到了」的方式结束
5. **张野的崩溃镜头必须写外部动作**——眼泪砸在屏幕上的瞬间不可省略、发了很多条消息的动作不可省略
6. **章末两个方向的交替镜头**——陆辞走远（不接不挂）+ 张野蹲在原地（屏幕亮了又暗）

**边界禁止（不涉及）：**
1. 不写肉戏——赵铮和张野做的过程不展开，通过陆辞的拍摄镜头扫过即可
2. 不揭示赵铮的真实身份和后续走向（第三卷展开）
3. 不让赵铮和张野有超过「交易行为」的互动
4. 不让陆辞和张野在工地面对面碰上——陆辞从拍摄到离开，张野都不知道有人在拍
5. 不写张野的内心活动（章末崩溃场景——只写外部动作）
6. 不写分析性结论（如「这意味着他们关系的终结」「这一刀砍断了最后的留恋」）
7. 不写「本章讲述了」「陆辞经历了」等上帝视角概述
8. 不写陆辞的内心感受——只写他的动作和选择

**角色禁区：**
- 陆辞不会哭——他是冷静决绝的那一方，不是崩溃的那一方
- 陆辞不会在发完消息后把张野拉黑——不拉黑但不回复是心死，是「已经不想再为这个人生气了」
- 陆辞不会回头——从工地走出来的每一步都是在往前走
- 张野不会用「求求你别走」这种语气说话——发消息时也是低声闷闷的语气
- 张野不会恨陆辞——他知道是自己先脏了
- 赵铮在本章不说话或只说粗口的几个字——不展开人物

### 反AI规则覆盖

**T1句式零容忍（出现即修改）：**
- ❌ 「不是A而是B」句式 → 直接写B
- ❌ 「带着……」（万能状语） → 拆短句或换动作
- ❌ 「仿佛/犹如/宛若」 → 删除或白描
- ❌ 「让/令/使」句式 → 删掉「让」，主语直接行动
- ❌ 「当……时/的时候」 → 拆成独立动作句
- ❌ 「他意识到/他感到/他注意到」（认知动词前置） → 删掉，直接写事实
- ❌ 「与此同时」 → 直接换场景开头

**T1禁用词（出现即替换）：**
- ❌ 一丝、一抹、些许、几分 → 具体量词或删除
- ❌ 不禁、不由得、不由自主 → 直接写动作（保留有节奏功能的）
- ❌ 瞳孔一缩、身体一僵 → 用具体身体反应替代
- ❌ 倒吸一口凉气、呼吸一滞 → 用呼吸变化细节替代
- ❌ 露出（表情） → 用具体笑法/表情动作替代
- ❌ 陷入（沉默） → 写沉默后的具体行为
- ❌ 涌起、泛起（情绪） → 用身体反应替代
- ❌ 嘴角勾起一抹…… → 写具体动作（「他笑了一下」「嘴角动了一下」）
- ❌ 眼中闪过一丝…… → 用动作替代（「他把目光移开」「他低下头」）
- ❌ 内心深处 → 删掉或直接写动作
- ❌ 突然/忽然（超过4次/章） → 保留节奏功能的，控制≤4次
- ❌ 渐渐/逐渐（超过2次/章） → 用变化细节替代

**禁止的总结升华句式：**
- ❌ 「这就是……」 → 删除，让场景自然结束
- ❌ 「他终于明白了……」 → 删除，停在画面/动作上
- ❌ 「从这一刻起……」 → 删除
- ❌ 「这一夜注定……」 → 删除
- ❌ 「他不知道的是……」 → 替换为陆辞视角内的动作
- ❌ 「一切……都……」 → 删掉或改为具体状态
- ❌ 「此时的他还不明白……」 → 删除（上帝视角禁止）

**禁止的心理描写替代：**
- ❌ 「陆辞感到/意识到/觉得」 → 直接写动作/身体反应
- ❌ 「陆辞心想/想道」 → 用动作外化
- ❌ 「张野感到/意识到」 → 写外部动作，不写内心

### 风格约束

**视角控制（严格第三人称有限——陆辞视角为主，张野视角为辅）：**
- 场景1-7：全章从陆辞的视角写——只写陆辞看到、听到、感觉到的一切
- 不写陆辞的内心活动——通过陆辞的动作让读者推测他的情绪
- 不写上帝视角总结——不写「陆辞不知道更大的痛苦还在后面」
- 场景8：切换到张野——但只写外部动作（眼泪砸在屏幕上、手指戳屏幕、蹲着发抖），不写张野的心理活动
- 场景9：两个方向交替——不写内心，只写动作和环境

**段落精度分层：**
- **高权重场景（场景5、6、7、8）**：细节饱满，至少3处动作/感知层级细节。场景5细化目睹瞬间的动作链（找到→看清→站定→拿手机→开录像→按录制）；场景6细化通过手机屏幕观看的行为本身（固定焦距不动的稳定感→录完关掉→蹭屏幕灰→装口袋→转身）；场景7细化打字时的犹豫（打了删、删了打、打了又删——发送后关屏幕放进口袋）；场景8细化张野的外部反应（僵住→眼泪砸在屏幕上→不停发消息→没有收到回复）
- **中权重场景（场景2、3、4、9）**：正常叙事节奏，场景2-4跟踪细节为主，场景9章末对照渲染氛围
- **低权重场景（场景1）**：快速转场，压缩至80-120字

**陆辞的说话方式：**
- 本章几乎无对话——只有发消息时的文字
- 陆辞的性格通过动作而非语言展现：冷静、理性、沉默、决绝
- 发消息时打字果断（但也暴露了犹豫——打了删、删了打的过程）
- 风格：文字干净，三句话三个句号，没有多余的字

**张野的说话方式（消息）：**
- 即使崩溃也是低声闷闷的语气：「不是你想的那样」「陆辞你听我说」「我操我骗了你」「对不起对不起对不起」
- 不是求饶——是认罪。他知道骗了陆辞是他的错
- 不写「求求你别走」这种乞求语气

**赵铮的处理：**
- 不出名字——只写「体型比张野大一圈的男人」「那个人」
- 不写台词——如果写台词只写粗口的几个字，不展开
- 不写外貌描写
- 只为第三卷埋伏笔——体型差+动作粗暴的印象

**身体的诚实反应替代心理描写：**

| 情绪 | 不写 | 写 |
|------|------|----|
| 心碎（陆辞） | 「陆辞心碎了」 | 打字前的犹豫（打了删删了打）、发完消息关屏幕没有再看一眼、走远了不接电话 |
| 紧张（跟踪） | 「陆辞感到紧张」 | 压低帽檐、选择站立位置、假装看手机、通过后视镜观察 |
| 确认（目睹） | 「陆辞确认了」 | 站在水泥柱后没有动→拿出手机→录像键按下前的手指在半秒停顿 |
| 崩溃（张野） | 「张野崩溃了」 | 眼泪砸在屏幕上、手指发抖、蹲着不停发消息、屏幕暗了又亮 |

**禁止的AI疲劳词/句式：**
- 「陆辞感到/觉得/意识到」 → 用动作替代
- 「某种/一股/一阵」（搭配抽象情绪） → 删掉或具体化
- 「慢慢地/缓缓地」 → 删掉或换具体速度（「他走得很慢，脚步有点飘」）
- 「温暖的/柔和的」（泛光感） → 用具体光影描述
- 「坚定的/深邃的」 → 删掉或换具体状态
- 「突然/忽然」控制≤4次/章

**对话标签规则：**
- 本章几乎无对话——无需担心对话标签问题
- 写消息时不写「他说」——直接写消息内容加引号，通过上下文判断是谁发的

**结构类禁止：**
- 连续4句相同主语 → 变换主语和句式
- 连续3段相同开头 → 每段首句不重复
- 连续5句相同句式 → 陈述/短句/动作交替
- 同一情绪词重复（紧张/心碎/崩溃）→ 单章≤3次

**排比句式禁止：**
- ❌ 「也许是……也许是……也许是……」 → 保留最强一条
- ❌ 「不是……不是……而是……」 → 直接写「而是」部分
- ❌ 「既没有……也没有……更没有……」 → 只保留最具体一条

**标点规则：**
- 全文禁止破折号（——）作停顿（控制≤3次/章）
- 对话中声音中断用「…」或直接断句
- 不使用感叹号收尾段落
- 不使用升华句式（「这一夜注定……」「从这一刻起……」）

### 本章注意点

- **陆辞的情绪不写**——全章不写陆辞的内心活动。所有情绪通过动作暴露：打字前的犹豫（打了删删了打）、录像键前的半秒停顿、发完消息关屏幕没再看一眼、不接电话但也没挂。读者通过这些动作自己感受。
- **手机作为媒介**——本章手机是核心道具：深夜看照片做决定用手机、跟踪时假装刷手机、目睹时通过手机录像、发消息分手用手机、张野崩溃时盯着手机屏幕。手机分隔了两个人——也隔开了陆辞和直接的冲击（通过屏幕看）。
- **「句号」的隐喻**——陆辞发消息用句号结尾（「我们到此为止。」），不是感叹号不是问号不是省略号。句号意味着陈述事实、没有商量余地、不需要回应。张野的消息后面没有句号——是一条接一条的绿泡泡，像抓住最后一根稻草。
- **陆辞的脚步声**——章末陆辞走远了，脚步声越来越轻。不写他去了哪里、在想什么。走了就是走了。
- **张野的眼泪**——不写「失声痛哭」不写「嚎啕大哭」——写眼泪无声地砸在手机屏幕上。安静比大声更痛。
- **不拉黑的理由**——拉黑是愤怒之下的动作，不拉黑但不回复是心死。陆辞已经不想再为这个人生气了。
- **赵铮的伏笔**——出场但不展开：体型比张野大一圈、动作粗暴。不写名字不写样貌不写台词。留给第三卷「绑架迷奸」的伏笔。
- **信息差贯穿全章**：读者知道张野是被系统逼的 → 陆辞不知道 ← 这个根本性的误会不会在本章被消解。陆辞的立场（他被背叛了）是合理的，张野的痛苦（被系统逼迫身不由己）也是真实的——两个真相都是真的，这让分手场景更残忍。
