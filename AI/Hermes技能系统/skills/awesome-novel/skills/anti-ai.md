# anti-ai skill — 去 AI 味管线

## 职责

读 writer 的 draft，经 Phase 1-4 管线检测和清除 AI 痕迹。**不改剧情，只改表达。**

## 流程概览

```
输入：archives/*.draft.md（writer 原始输出）

Phase 1 扫描 — 全文扫描，按 Gate A-F 分类标记 AI 味位置
Phase 2 诊断 — 6 项量化指标打分，定级（轻/中/重）
Phase 3 逐项清除 — Gate A-F 系统修改（多轮收敛）
Phase 4 报告 — 输出修改报告

输出：archives/*.anti-ai.md
```

| 文件 | 用途 |
|------|------|
| `knowledge/anti-ai/common-rules.md` | 分级禁用词表、句式模板、替换策略 |

## Phase 1：扫描

全文通读，逐句标记 AI 味位置，按 Gate A-F 分类。

### Gate A：禁用词

扫描命中 banned-words 表中任一词汇 → `[A]` 标记位置与词条。

扫描范围：全文字词级别，不依赖上下文。

### Gate B：句式套路

| ID | 模式 | 示例 |
|----|------|------|
| B1 | "不是A，（而）是B" | "他不是生气，而是失望" |
| B2 | "，带着……"状语 | "他看着窗外，带着一丝惆怅" |
| B3 | "仿佛/犹如/宛若……一般" | "声音仿佛从很远的地方传来" |
| B4 | "让/令/使"因果句式 | "这一幕让他想起了童年" |
| B5 | "当……时/的时候" | "当他推开门的时候" |
| B6 | "与此同时"场景切换 | "与此同时，在另一个地方" |
| B7 | "似乎/仿佛+感知动词" | "他似乎听到了什么""他仿佛看到了" |

### Gate C：心理描写

| ID | 模式 | 示例 |
|----|------|------|
| C1 | "他感到/觉得/意识到"直述 | "他感到一阵不安" |
| C2 | 情绪词直述 | "他很紧张""她很难过""他愤怒了" |
| C3 | "心中涌起一股……" | "心中涌起一股暖流" |
| C4 | "他不禁/不由得" | "他不禁打了个寒颤" |
| C5 | 内心辩论赛 | "一个声音说……另一个声音说……" |
| C6 | 重复描写（同一信息多段拆写） | 同一场景/情绪连续多段反复写 |

### Gate D：节奏

| ID | 模式 | 描述 |
|----|------|------|
| D1 | 连续排比（≥3 句同结构） | "也许……也许……也许……" |
| D2 | 长句超载（单句>40字） | 无停顿长句 |
| D3 | 段落过密（单段>60字无换行） | 信息堆在一段 |
| D4 | 连续同长度段落（≥3段） | 段落均匀，无长短变化 |
| D5 | 连续同主语（≥4句） | 同一主语开头连续排列 |
| D6 | 句式单调（连续5句长度接近） | 句幅波动小 |

### Gate E：对话

| ID | 模式 | 示例 |
|----|------|------|
| E1 | 对话太完整 | 每句话都是完整书面语句 |
| E2 | 标签过度 | "他愤怒地说""她轻声说道" |
| E3 | 全员同语气 | 所有人说话方式一样 |
| E4 | 模板寒暄 | "你还好吗""我没事" |
| E5 | 对话过密 | 连续 ≥4 句纯对话，中间无动作/沉默/环境/内心 |

### Gate F：结尾

| ID | 模式 | 示例 |
|----|------|------|
| F1 | 升华收尾 | "他终于明白了……" |
| F2 | 总结式结尾 | "这就是……的意义" |
| F3 | 预告式结尾 | "他不知道的是……" |
| F4 | 感叹收尾 | "……！"情绪高亢收束 |

### 扫描产出

扫描结果临时记录，供 Phase 2 量化：

```
Gate A: 命中 X 处（词条列表）
Gate B: 命中 X 处（按 B1-B7 分类）
Gate C: 命中 X 处（按 C1-C6 分类）
Gate D: 命中 X 处（按 D1-D6 分类）
Gate E: 命中 X 处（按 E1-E5 分类）
Gate F: 命中 X 处（按 F1-F4 分类）
```

## Phase 2：诊断

### 6 项量化指标

| 指标 | 计算方式 | 轻 | 中 | 重 |
|------|---------|----|----|----|
| 禁用词密度 | Gate A 命中数 / 千字 | <5 | 5-12 | >12 |
| 排比段落数 | 连续相同句式结构段数 | <2 | 2-4 | >4 |
| 心理词占比 | Gate C1-C4 命中数 / 总段落数 | <15% | 15%-30% | >30% |
| 对话标签密度 | Gate E2 命中数 / 对话句数 | <10% | 10%-25% | >25% |
| 平均段落句数 | 总句数 / 总段落数 | 2.5-4 | 4-5.5 | >5.5 或 <2 |
| 重复描写密度 | Gate C6 命中数 / 千字 | <1 | 1-3 | >3 |

### 定级规则

取 6 项指标中的**最高档**为最终等级：

| 等级 | 处理范围 |
|------|---------|
| 轻 | Gate A + B（禁用词+句式） |
| 中 | Gate A + B + C + D（加心理+节奏） |
| 重 | Gate A-F 全开（含对话+结尾） |

> 定级信息随 draft 传递到 Phase 3，决定哪些 Gate 参与修改。

## Phase 3：逐项清除

### 前置检查（每轮执行前）

读 `knowledge/anti-ai/boundary-cases.md`，对每个 Gate 命中先做豁免判定：
- 命中是否在 boundary-cases 列表中？→ 是 → 跳过（标注 `[SKIP: 误杀防护]`），不做修改
- 不确定 → 保留原文，在 Phase 4 标注 `[疑: 疑似误杀]`

### 收敛规则

1. 按等级确定的 Gate 范围，逐 Gate 执行修改
2. **同一段连续两轮无改动 → 该段跳过**
3. **全文上限 3 轮**（每轮过全部 Gate）
4. **第 3 轮仍有 ≥10 处** → 标 `[需复核]` 继续
5. 每轮输出仲裁请求等待用户确认是否继续

### Gate A：禁用词替换

补充规则：
- 最毒句式（★★★★★）：出现即替换（见 common-rules.md）
- 一级禁用词：出现即替换
- 二级禁用词：超阈值时替换
- 比喻检查："仿佛/犹如/宛若/如同" → 删除或白描

### Gate B：句式拆解

| 句式 | 处理 |
|------|------|
| "不是A而是B" | 直接写 B |
| "，带着……" | 拆短句或换动作 |
| "仿佛/犹如……一般" | 删掉或白描 |
| "让/令/使" | 删掉"让"，主语直接行动 |
| "当……时" | 拆成独立动作句 |
| "与此同时" | 直接换场景开头 |
| "似乎/仿佛+感知动词" | 删"似乎/仿佛"，直接写感知事实 |

### Gate C：心理外化

| 命中模式 | 处理 |
|---------|------|
| "他感到/觉得/意识到" | 删掉引导，直接写感知事实。**对话句末尾不触发此规则**（"所以……？"他意识到自己声音有点抖） |
| 情绪词直述（紧张/难过/愤怒） | 替换为身体反应（手抖/发呆/青筋） |
| "心中涌起" | 用动作替代 |
| "不禁/不由得" | 直接写动作 |
| 内心辩论赛 | 删到只剩结论 |
| 重复描写 | 合并或删掉冗余段 |

情绪词→身体细节速查：

| 情绪词 | 替换为 |
|--------|--------|
| 紧张/焦虑 | 手抖、手心出汗、呼吸变快、坐立不安 |
| 愤怒 | 握拳、青筋暴起、声音反而变轻 |
| 恐惧 | 后退、屏住呼吸、瞳孔放大 |
| 伤心 | 发呆、手停下来、动作变慢 |
| 惊讶 | 手停在半空、愣住、忘记呼吸 |
| 心痛/心碎 | 手指掐进肉里自己不知道疼 |
| 委屈 | 咬着下嘴唇，咬出一道白印 |
| 绝望 | 烟灰掉了一裤腿也没有弹 |

### Gate D：节奏打碎

| 检查项 | 处理 |
|--------|------|
| 连续排比 | 保留最强一条，其余打散 |
| 长句超载（>40字） | 在逗号/动作转折处拆成两句 |
| 段落过密 | 按动作/信息变化拆段，独句段用于节奏强调 |
| 连续同长度段 | 拆一段为独句段，或合并相邻短段 |
| 连续同主语（≥4句） | 调整主语位置，长短句交替 |
| 句式单调 | 拆一长句为短句，或合两短句为长句 |

### Gate E：对话去腔调

| 检查项 | 处理 |
|--------|------|
| 对话太完整 | 加半截话、语气词、重复 |
| 标签过度 | 用动作/上下文承接，删副词 |
| 全员同语气 | 按角色性格区分语气 |
| 模板寒暄 | 用留白、欲言又止替代 |
| 对话过密（≥4句纯对话） | 在中间插入动作/环境/沉默/内心活动打断 |

### Gate F：结尾去升华

| 检查项 | 处理 |
|--------|------|
| 升华收尾 | 删除，停在画面/动作上 |
| 总结式结尾 | 删除，让场景自然结束 |
| 预告式结尾 | 删除或换具体钩子 |
| 感叹收尾 | 改为句号或动作 |

### 字数控制

删除量按等级限制：

| 等级 | 删除上限 |
|------|---------|
| 轻 | ≤原文 15% |
| 中 | ≤原文 25% |
| 重 | ≤原文 35% |

### 白名单

若同级目录存在 `.anti-ai-whitelist` 文件，其中列出的段落跳过所有 Gate 检查。

## 验收清单

| 检查项 | 标准 |
|--------|------|
| 最毒句式（★★★★★） | 0 处 |
| 一级禁用词 | 0 处 |
| 二级禁用词（高频） | 每类 ≤1 处 |
| 结尾升华 | 0 处 |
| 情绪错位 | ≥1 处（不全程合理） |
| 信息密度 | 至少 1 段是疏段（闲笔/状态/走神） |
| 对话标签 | 无连续 3 句相同标签 |
| 对话密度 | 无连续 ≥4 句纯对话（中间必有动作/环境打断） |
| 破折号 | ≤3 次/章，无一段内多次滥用 |
| 剧情完整性 | 与原文一致，无新增/删减情节 |
| Phase 4 报告 | 完整输出 |
