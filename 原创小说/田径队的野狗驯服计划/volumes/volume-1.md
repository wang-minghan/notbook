# 卷1·蚕食

> 卷纲——从"高价值身份建立"到"不可逆沦陷"的完整弧线。

## 卷信息

- **情绪走向**：自信（展示身份）→ 意外崩塌（第一次被侵犯）→ 被迫接受（威胁+打赌）→ 身体觉醒（耻辱快感）→ 公开（被见证）→ 彻底沦陷（主动+群交）
- **核心冲突**：沈骁的雄性自尊 ∧ 赵叔的蚕食策略 + 身体的本能背叛
- **主导驱动力**：威胁驱动（赵叔掌握把柄）+ 欲望驱动（身体觉醒后）
- **冲突阶梯**：
  - 第1层（尊严动摇）：被触碰但不敢反抗（第2章）
  - 第2层（身体背叛）：硬了→射了→开始想（第3-4章）
  - 第3层（领域入侵）：在学校/公开场合做→被其他人看到（第4-5章）
  - 第4层（身份粉碎）：从操别人的人变成被别人操的狗（第6章）
- **预估章节**：6章
- **核心毒源公式**：每章必须满足——1个新羞辱层级 + 1个新目击者/公开范围扩大 + 1句刺人台词 + 1个不可逆改变

## 章节列表

### 1. 野狗（已写·存档）

- **冲突事件**：沈骁的日常——训练/队友/恋人/被器材管理员注视。林越察觉到异常。
- **情绪锚点**：阳光、自信、让人羡慕——为后面的崩塌建立高起点
- **信息差**：读者知道赵叔看沈骁的眼神不对 ↦ 沈骁不知道这个人在等他等了三年

### 2. 猎物（已写·存档）

- **冲突事件**：省赛前庆祝喝酒 → 被赵叔以锁门为诱饵骗到器材室 → 第一次身体触碰 → 沈骁没有反抗 → 赵叔说"你硬了"
- **情绪锚点**：从开心到不安到恐惧到屈辱——被扒开面子的瞬间
- **信息差**：沈骁以为自己能控制局面 ↦ 赵叔知道他不敢反抗（因为林越）

### 3. 打赌（待写 → 千问提示词）

- **冲突事件**：省赛当天，赵叔在赛场看台上出现（威胁落地）；赛后赵叔叫沈骁来器材室"谈谈"；打赌：沈骁如果能忍住不被操射，就放过他；沈骁输，被操，被剃毛。
- **情绪锚点**：紧张→绝望→耻辱→身体背叛（被操射）→仪式化屈服（剃毛）
- **信息差**：沈骁以为打赌有赢的可能 ↦ 赵叔从打赌规则设计上就没打算让他赢

### 4. 哨声（待写）

- **冲突事件**：赵叔开始在白天/学校厕所找沈骁；口交升级到灌尿；从抗拒到主动赴约；林越发现沈骁开始冷淡。
- **情绪锚点**：屈辱升级→恐惧消退→身体快感开始出现→厌恶自己但还是来了
- **信息差**：林越觉得沈骁不对劲但不知道原因 ↦ 沈骁瞒着林越

### 5. 第三者（待写）

- **冲突事件**：林越想给沈骁惊喜，偷偷去器材室找→从门缝/窗外看到了沈骁骑在赵叔身上的画面；崩溃。
- **情绪锚点**：好奇→震惊→恶心→心碎→复杂的身体反应
- **信息差**：林越视角—他以为的沈骁 vs 他看到的沈骁

### 6. 野狗之夜（待写）

- **冲突事件**：林越和沈骁对峙→分手；赵叔叫来工友/队友群交；沈骁从"被操"变成"求操"；开放式结局——他还得继续当队长，但所有人都知道他是什么了。
- **情绪锚点**：悲伤→解脱→羞耻→彻底放开→宿命般的平静
- **信息差**：沈骁以为自己还能回到从前 ↦ 读者（和赵叔）都知道回不去了
