# 张威 — 肉体征服者

## 基础信息

- **角色定位**：攻方②——健身房老板，张野的第一个正式客人，纯肉体征服者
- **年龄**：34岁
- **身份**：健身房老板（体育学院南门对面）
- **外形**：满身腱子肉，粗犷豪放，花臂，寸头。一看就是不好惹的社会人
- **鸡巴**：20cm / 龟头直径5.5cm / 直粗型 / 像涂了油的健身哑铃杆
- **核心癖好**：操尿控 + 中出上瘾

## 人物特点

- **出场时机**：陆辞分手后一周。张野接同城任务，到了才发现约自己的人就是办过年卡那家健身房的老板
- **张野反应**：尴尬想跑——但张威认出他了，笑了一声说"办了卡不来练？今天来了就别走了"
- **功能**：让张野第一次体验到被粗暴征服的爽。陆辞操的是心，张威操的是底线。第一次被操到失禁、第一次被内射到满肚子精、第一次发现原来被粗暴地干也可以很爽

## 精神内核

"体面人？体育生？被老子操到尿裤子的时候都一样骚。"

## 经典模式

每次必内射，射完不拔出来，堵着继续操，等精液被搅成白沫从穴口溢出。喜欢在张野快到高潮的时候掐住龟头不让射，逼他求自己。张威的玩法粗暴直接，没有调教和技巧，就是纯粹的体能压制——用体重和蛮力让张野屈服。
