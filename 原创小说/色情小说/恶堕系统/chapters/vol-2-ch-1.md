# 第二卷·肉体征服 → 第1章「约到了老板」

## 章纲概要

分手后一周。张野接了系统推送的第一个同城任务，到了才发现约自己的人就是办过年卡那家健身房的老板——张威。张威认出他了："办了卡不来练？今天来了就别走了。" 张野尴尬得想跑，但系统惩罚倒计时逼他留下。章末落在张威休息室的床上——张野攥着床单，指节发白。

---

## 关键点（Outline Key Points）

1. [过渡·造悬念] 分手后一周。张野在出租屋的状态——没有哭天抢地，就是空。手机里和陆辞的聊天记录停在一周前最后那条「视频我删了。以后别联系了。」他没删聊天记录，但也没再打开看过。系统面板弹出一个新任务推送：同城，一个地址，限24小时内确认。他看了几秒，按了「接单」。

2. [推进剧情·过渡] 傍晚出门。初夏天黑得晚，街上还有亮。张野穿旧T恤和运动短裤，不打扮，低着头走路。路过学校南门时他停了一下——隔着路就是那家健身房。他没多看，收回目光继续走。路上手机震了一下，系统提示客人已确认接单时间。

3. [推进剧情·情绪] 按地址到了才发现——就是南门对面那家健身房。张野站在门口愣住了。三万的年卡，一次都没来练。他站在玻璃门前，能看到里面器械区的灯光和跑步的人影。他想转身走，系统面板立刻弹出倒计时——红色数字在视网膜上闪：「00:05:00」。不走的话有惩罚。他站了几秒，推门进去了。

4. [推进剧情·对话] 前台没人，柜台后面传来一个声音："找谁？" 张野转头，看到张威从里面走出来——34岁，满身腱子肉把黑色背心撑得很紧，花臂从肩膀铺到手腕，寸头，下巴留一圈胡茬。张威看到他就笑了，认出来了："操，你不就是那个办了年卡的体育生？" 张野张了张嘴说不出话。张威走近两步打量他："办了卡一次不来练，今天怎么有空了？"

5. [推进剧情·对话·情绪] 张野不知道该说什么，站在器械区旁边，手插在裤兜里攥着手机。系统倒计时还在走，提醒他任务目标就在面前。张威没急着干嘛，靠在柜台边上跟他说了几句话——说那张年卡，说三万块不是小数目，说"你们这些学生就是被销售忽悠的"。语气带调侃但没有恶意。张野低着头嗯了几声，耳朵红了。

6. [推进剧情·过渡·环境] 张威带张野往里走——穿过器械区，经过几个正在练的人，有人跟张威打招呼叫他"威哥"。健身房的气味——汗味、铁锈味（器械）、消毒水混在一起。张威走在前面，背心勒着肩背肌肉线条，花臂在灯光下纹路清晰。张野跟在后面，这段路比他想象的长。

7. [推进剧情·情绪] 休息室在健身房最里面。推开门，十几平的房间——一张单人床靠墙，床单深色，床头柜上放着烟灰缸和打火机，角落有个独立淋浴间，墙上挂着一副拳击手套。门关上之后外面的器械声变小了。系统倒计时还在视网膜右下角跳。张野站在门口没往里走。

8. [过渡·造悬念] 张威说了句"先去冲一下"，张野进了淋浴间关上门。他没马上开水——站在瓷砖地上，手撑着洗手台边缘，低头看了一会儿自己的手指。水开了，温度很热。他站在水下没有动，让热水从头顶浇下来。淋浴间里全是水蒸气的时候他才关水，擦干，裹着浴巾出来。

9. [推进剧情·造悬念·章末] 张威坐在床边，手里转着打火机。看张野出来，上下扫了一眼——张野185的体育生身材，宽肩窄腰，浴巾裹在腰上。张威拍了拍床沿。张野走过去坐下，床垫陷了一点。他的手指下意识攥着床单边沿，指节发白。系统倒计时归零——绿色提示：「任务进行中」。张威的手按在他肩膀上，虎口卡着肩胛骨，粗糙有力。张野没有躲，也没有回头，闭着眼。章末落点：攥着床单的手，指节发白，窗外的天已经全黑了。

## 当前任务

建立张野第二次卖身（系统第一次正式任务）的开端，通过健身房场景引出张威的人物形象——粗犷、豪放、体能压制型，同时展示张野分手后一周的状态：空了但还在运转。本章是连接第一卷心碎结局和第二卷肉体征服的桥梁。

## 读者期待

- **情绪状态**：混合——第一卷结束在心碎的余烬里，读者知道张野正式进入卖身阶段，期待看到"他怎么开始"以及"第一个客人是谁"
- **策略**：给部分信息但吊胃口——让读者看到张威的外形和气质（满身腱子肉、花臂、寸头、粗犷），看到张野的紧张和被迫，但不给肉戏——肉戏留到第2章
- **具体说明**：读者想看的是"系统的第一个任务长什么样"、"第一个客人是谁"、"张野怎么面对"以及"他有没有退路"——本章回答前三个，第四个留到床上揭晓

## 兑现计划

- **必须了结**：
  - 第一卷结尾的铺垫——张野在第8章打开系统接了任务，本章展示这个任务的具体执行过程
  
- **必须压住**：
  - 不能写任何肉戏——不写插入、不写鸡巴尺寸、不写被操的细节，所有性描写留给第2章
  - 不能让张野逃跑——系统惩罚倒计时逼他留下，他必须跨过这道门槛
  - 不能写张野哭或情绪外露——他的心碎是沉默的，不是歇斯底里的
  - 不能写张威的正面性描写——只写他的外形（肌肉、花臂、寸头）和说话方式（粗犷、带脏），不写鸡巴、不写性暗示动作

- **推进不收束**：
  - 张野的肉体征服正式启幕——这是系统安排下的第一次正式卖身
  - 建立张威粗犷征服者的第一印象——为第2章"无前戏粗暴插入"做铺垫
  - 新钩子：系统惩罚机制在本章首次亮出（倒计时强制），读者看到系统不是只发任务，还有执行层面的压迫力
  - 新钩子：健身房年卡被骗三万块的背景——张野经济困境的延续，也是他和张威之间建立联系的入口

## 过渡功能

- 「分手后一周·张野的状态」 → 承接第一卷结局，建立张野的当下心理状态（不是崩溃，是空）
- 「路上→发现是健身房」 → 空间和情绪的同步切换——从日常步态到发现目的地时的本能退缩
- 「穿过器械区→去休息室」 → 从公共空间到私密空间的过渡，为性场景做空间铺垫，同时通过健身房环境描写展示张威的社会角色
- 「站在淋浴间不开水→热水浇下来」 → 从"还在犹豫"到"知道逃不掉了"的心理转折，用水的温度变化暗示情绪
- 「张威拍床沿→张野坐下攥床单」 → 从"还有退路的可能性"到"已经接单"的仪式感过渡

## 关键抉择

- 「张野为什么接了系统任务？——分手后一周，他没有像第6章那样挣扎抗拒。陆辞的消息像一刀切断了什么——不是切断了关系，是切断了"我还能回头"的错觉。他接任务不是因为不怕了，是因为觉得无所谓了。符合人设：他逃避不了的时候就会顺着力最大的方向走」
- 「张野站在健身房门口为什么还是进去了？——系统惩罚倒计时是表层原因，深层原因是他没有力量对抗系统。门不是他推开的，是被系统性倒计时推开的。符合第一卷建立的角色处境：他在系统中没有自主权」
- 「张野为什么没有在浴室里待更久逃避？——他不是不想，是他知道外面的人在等。逃不掉的。他裹上浴巾出来，不是准备好了，是放弃了。符合张野的被动性格：他不会主动推进，也不会主动拒绝，只是在"不得不"的时候向前迈一步」

## 角色信息状态

- **张野**：知道这是系统的第一个正式任务 / 知道客人是办过年卡的健身房老板张威 / 知道自己还喜欢陆辞但已经不属于他了 / 知道没有退路 / 不知道张威在床上是什么风格（第2章才展开）、不知道以后会和这个男人发生多少交集

- **张威**：知道张野是办过年卡的学生 / 知道张野是接了约炮/卖身单子来的（系统背后的机制他不一定了解全貌，他只知道自己下单了） / 知道张野很紧张 / 不知道张野是被系统逼来的、不知道张野刚刚分手、不知道张野的第一任是谁

**信息差关系**：
- 张威以为张野是自愿来接单的 ↦ 张野是被系统逼迫的 ← 核心信息差，制造不适感——读者知道张野没得选，但张威不知道
- 张威还记得张野办年卡的事 ↦ 张野尴尬到想死 ← 这条关系（被骗钱的健身房老板现在成了买身的客人）增加了张野的心理负担
- 读者知道张野刚从心碎期走出来 ↦ 张威不知道张野的过去 ← 读者比张威更清楚张野此刻有多脆弱

**信息差变化**：
- 开场：张野知道任务来了但不知道是谁 ↦ 读者和张野一样未知
- 中段：张野发现客人是张威 ↦ 信息差转为"张威不知道张野是被逼的"
- 结尾：张威准备动手 ↦ 读者知道痛苦要来了（第2章）但张野不知道具体有多痛

**对话**：有

- 张威调侃年卡的事：「三万块就当交学费了」

## 硬约束

- 不要在任何地方展开肉戏——不写性交过程、不描写张威的鸡巴、不写插入、不写被操的细节。本章核心是「场景铺垫和人物建立」
- 不要写张野在浴室里哭——他的状态是「不知道该怎么办」的空，不是「想哭但又憋回去」的悲伤——后者在第1卷已经用过了，第2卷开始他进入新阶段
- 不要让张野逃跑——系统惩罚倒计时必须让他留下
- 不要让张威变成调情高手——他粗犷、直接、没有花哨，不是陆辞那种温柔型
- 不要对比陆辞——本章不提陆辞的名字（只在关键点1中暗示「聊天记录停在一周前」），不用张威的动作去对照陆辞
- 张野的说话方式保持低声闷闷的——本章几乎不说话，即使说话也是嗯/好/知道了
- 不要出现「本章讲述了」「张野经历了」等上帝视角概述
- 不要出现分析性结论（如「这标志着张野正式进入恶堕的第二阶段」）

## 情绪设计

**卷走向位置**：第1层（入门阶段）——本章是第二卷的起始章，基调是「尴尬、紧张、被迫」。

**章内微弧线**：
- 空（分手后一周的平静——不是悲伤，是什么都感觉不到）→ 意外（发现目的地是健身房时的停顿）→ 尴尬（被张威认出来说年卡的事——恨不得钻地缝）→ 被迫（系统倒计时亮出来，想走也走不了）→ 紧张（穿过器械区走向休息室——越来越近）→ 放弃抵抗（坐在床上，攥着床单，闭眼）
- 从「空了」到「被迫承受到极限」，终点落在「无路可退的紧张感」——支撑卷走向的下一步（第2章·肉体折磨的开始）

**主情绪**：尴尬、紧张、被迫

**强度峰值**：关键点9（张野坐下攥床单的时候——一切退路都关上了）

**强度等级**：6（第一卷结尾第8章约5~6——心死后平静地接了任务；本章强度同为6——尴尬紧张但还未到性交的剧烈冲突；第2章「第一次被干」的痛感冲击约7~8）

**情绪钩子**：紧张与不适——张野坐在张威的床上，攥着床单，读者知道下一章会发生什么——一个内向的体育生即将被一个粗犷的肌肉男粗暴操干，没有前戏，没有温柔。读者想知道张野能不能扛住，以及——他会不会像上次（第6章对赵铮）一样再次在被迫中找到身体的反应。

## 读者获得

- **类型**：信息（info）——读者确认了张野的第一次系统正式任务对象是张威（健身房老板），以及他的外形气质：肌肉男、花臂、寸头、粗犷豪放
- **描述**：读者通过张野的视角完整经历了从接任务到发现客人=健身房老板的整个过程——知道了他有多尴尬、多紧张、多没有选择
- **位置**：前段—中段（关键点3-6，发现健身房+初遇张威）

## 情绪锚点与微弧线设计

**微弧线**：空（分手后）→ 意外（发现健身房）→ 尴尬（被认出）→ 被迫（系统倒计时）→ 紧张（走向休息室）→ 放弃（坐在床上）

**情绪锚点**：
1. 张野站在健身房门口，手已经按在玻璃门上又缩回来——系统倒计时在闪，他站了三秒，推门进去。玻璃门在身后关上的声音比他想象的重。
2. 张威靠在柜台上说「办了卡不来练」——语气是调侃的，但张野耳朵红到脖子根。不是害羞，是难堪。三万块，一次没来，现在站在这里是因为要卖身还债。
3. 张野站在淋浴间里，开了水没马上洗——水声很大，他盯着瓷砖墙缝里发黑的霉点发了一会儿呆，然后才开始洗。不搓，只是站在水下。
4. 张威拍床沿说「过来坐」——张野走过去，坐下，手指攥住床单边沿。指节发白。他没有抬头看张威的脸，也没有说话。床单的布料被攥得比手还紧。

## Memo

- **immersion_type**：冲突紧迫——系统倒计时的强制力和张威的粗犷气场给读者制造压迫感，张野的紧张通过动作细节传染给读者
- **key_point 数量**：9 条（约4500字）
- **张野的说话方式**：小声、闷闷的、语速慢——本章几乎没有主动说话，都是被问才答几个字（嗯、好、知道了）
- **本章核心**：从心碎过渡到肉体征服的第一块跳板。建立张威的形象，展示张野被系统逼到绝境的处境，为第2章的粗暴性爱做场景和情绪铺垫
- **张威的塑造要点**：粗犷但不猥琐，豪放但不油腻。说话带脏字是自然的，不是刻意耍流氓。他的力量感通过体型、动作（拍肩膀、按肩膀）和空间占用来体现
- **无肉戏**：这是第二卷唯一没有肉戏的章节。重要的事情都在场景和对话里完成——角色建立、关系建立、权力格局建立
