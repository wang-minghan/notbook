# 图书清单(最近录入的15本)
```dataview
TABLE without id
	"![bookcover|50]("+封面+")" AS "封面",
	file.link as "书名",
	author as "作者",
	elink(豆瓣页面, string(豆瓣评分)) AS "豆瓣评分",
	阅读状态 AS "状态",
	我的评级,
	file.etags AS "标签"
FROM "300 Resources/330 Books/331 BookInfo"
Sort created desc
limit 15
```

## 评分标准：
⭐很差，浪费时间；
⭐⭐未达期望，没有增值，没有收获，消磨时间；
⭐⭐⭐一般般，有收获，但不大；
⭐⭐⭐⭐超出期望，有收获；拓展知识/可获得精神满足或愉悦，但无认知上的颠覆
⭐⭐⭐⭐⭐收获很大；有认知上的改变/可以反复阅读。
