## 电脑版（全年）
脚本文件：[[task-heatmap.js]]

>[!warning] 注意事项：
>热力图任务统计的范围是“500 Journal/540 Daily”里的日志。如需使用此功能，不要修改日志路径。另外，不在日志里的任务，不在统计范围内。

### 使用方法：

1. **默认使用（暗色模式）**：

````
```dataviewjs
dv.view("task-heatmap")
```
````

效果：
```dataviewjs
dv.view("task-heatmap")
```

1. **使用亮色模式**：
   
````
```dataviewjs
dv.view("task-heatmap", { theme: "light" })
```
````

效果：
```dataviewjs
dv.view("task-heatmap", { theme: "light" })
```


3. **同时设置多个参数**：
   
````
```dataviewjs
dv.view("task-heatmap", { 
  theme: "light",
  showTitle: true,
  color: "blue",
  year: 2025
   })
```
````


效果：
>*颜色可选red, blue, green, purple*
```dataviewjs
dv.view("task-heatmap", { 
  theme: "light",
  showTitle: true,
  color: "blue",
  year: 2025
   })
```
