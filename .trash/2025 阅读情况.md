## ✅已读书籍

```dataview
Table 
	without id
	 "![bookcover|80]("+封面+")" AS "封面", 
	 file.link AS "书籍名称" , 
	 我的评级 ,  
	 短评 AS "我的短评" , 
	 豆瓣评分 AS "豆瓣"
	 From "300 Resources/330 Books/331 BookInfo"
Where date(开始时间).year = 2025 AND (阅读状态 = "已读" or 阅读状态 = "✅")
Sort 读完时间 desc
```

>*阅读状态 = "已读"; 阅读开始时间和读完时间不能为空；开始阅读时间为2025年。*

## 在读书籍
```dataview
List 
	without id
	"📙" + file.link + "最后一次阅读是" + date(读完时间) + ""
From "300 Resources/330 Books/331 BookInfo"
Where created >= date(2025-01-01) AND (阅读状态 = "📙" or 阅读状态 = "在读")
Sort 结束时间
```

>*阅读状态 = "在读"；书籍录入时间为录入时间2025年。*

## 待读书籍

List 
From "300 Resources/330 Books/331 BookInfo"
