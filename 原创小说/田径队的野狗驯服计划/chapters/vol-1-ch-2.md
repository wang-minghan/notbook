# 第二章·猎物

- **memo:** 第一次身体触碰。赵叔展示"我知道你的一切"→沈骁的软肋被捏住→不敢反抗→身体背叛(勃起)→赵叔以退为进"今天就到这儿"
- **stage:** archived
- **字数:** 约3400字（完整正文已归档）

## 场景

### 场景1｜器材室｜深夜｜权重：高

- **入口情绪**：警觉、恐惧
- **核心事件**：门锁落下 → 赵叔掐住沈骁后颈 → 开灯对峙 → 赵叔说出沈骁的一切(家庭/成绩/林越) → 沈骁的拳头松开 → 赵叔抚摸 → 乳头被揉 → 沈骁勃起
- **信息差**：沈骁以为赵叔只是个普通器材员 ↦ 赵叔已经观察了他三年
- **情绪拐点**：听到"你硬了"时的崩溃感
- **出口情绪**：屈辱、恐惧、自我厌恶

### 场景2｜宿舍厕所｜深夜｜权重：低

- **核心事件**：沈骁冲回宿舍楼的厕所 → 看镜子里潮红的脸 → 裤子隆起还没消 → 骂自己
- **情绪意义**：独白式的自厌，第一次意识到身体的背叛

### 场景3｜宿舍床上｜深夜｜权重：中

- **核心事件**：林越发消息问到了没 → 沈骁只回"到了" → 躺床上睁眼到天亮 → 梦里全是赵叔的手
- **情绪拐点**：最恐惧的是——赵叔说得对，他确实在想

## 人物状态

- **沈骁起点→终点**：被诱捕前的放松 → 被摸后崩溃、自我厌恶、开始恐惧
- **赵叔策略**：展示"我知道你的软肋"→第一次动手测试反应→以退为进留悬念
- **赵叔状态**：完全掌控节奏的猎手，不急于一次性吃干抹净
