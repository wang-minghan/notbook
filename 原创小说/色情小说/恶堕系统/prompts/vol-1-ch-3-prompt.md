# Prompt — 第一卷·第3章「破处」

## 角色

你是一个都市题材（urban）小说作家。现代都市背景，所有场景发生在真实城市空间——出租屋、暖黄灯光、拉着窗帘的傍晚房间。超自然元素仅限「阿鼻系统」本身，其余世界观完全写实。不涉及修仙、穿越、异能战斗等非都市元素。

## 任务指示

创作第一卷·第3章正文（在全卷中的角色：第1层·入门→第2层·升级——本章以身体亲密度的推进为主要冲突。这是张野的破处章——从接吻、抚摸、口交、扩张到首次插入式性交的完整过程。陆辞全程温柔耐心，张野从紧张到疼痛到被爱着的安心。这是卷内唯一的「纯甜高潮」节点，情绪达到「被爱着的安心」，为后续系统的介入积蓄最大的反差势能。）

### 字数与压缩策略

- **目标字数**：建议 4500 字（±10% 可接受，叙事完整性优先于字数精确）
- **压缩策略**：超字数时优先压缩低权重场景（关键点1过渡/关键点9章末收束），推荐压缩至150-200字以内——保留1句环境切换 + 1句人物动作 + 1句情绪/氛围留白；不得因压缩删改红线内容或破坏情绪主线

### 本章驱动力

张野在陆辞的出租屋里经历人生的第一次完整性行为——从接吻到抚摸到口交到扩张到插入。这不是GV式的粗暴破处，而是温柔耐心的恋人式第一次。陆辞用一整晚的前戏让张野放松，在插入时张野疼得眼眶红但没叫出声，陆辞立刻停下来抱着他等他适应。张野缓过来后小声说「没事……你动吧……」。章末落在张野被搂在怀里的安心感——身体还酸胀着，但心里是满的。

### 信息释放节奏

情感推进型——从紧张到放松到疼痛到安心。从进屋的局促（过渡）→建立安全感的对话和触碰（前戏开始）→接吻和抚摸（亲密度升级）→口交初体验（第一次被人口射）→扩张（耐心适应）→首次插入的疼痛（情绪强度峰值）→适应和开始抽送（转折）→事后拥抱的安心（章末）。终点落在张野把脸埋进陆辞怀里「不想说话，不想动，就想一直这么待着」。

### 叙事目标

1. 核心悬念：张野的第一次性交能顺利完成吗？他能承受插入的疼痛吗？
   → 本章结束时：**解答了**——他在陆辞的耐心和温柔下承受了，但读者知道这是「痛但被爱着」的第一次，不是「爽」

3. 爽点设计：温柔前戏的情感张力 + 口交初体验 + 首次插入的痛感与温柔并存的时刻 + 事后拥抱的安心

   - 释放于关键点3-6（前戏升级→口交→扩张的渐入佳境）
   - 高潮释放于关键点7-8（首次插入的疼痛→适应→第一次完整的性结合）
   - 余韵于关键点9（事后拥抱清理的安心感）

### 写作方法论

1. **焦点锁定**：对每一场景，识别其唯一的「核心冲突」或「人物情绪」或「信息差」（三选一，不叠加）。将70%以上笔墨集中于此焦点对象，次要元素简笔带过。高权重场景细化到动作与感知层级；低权重场景允许快速转场，但不得违反焦点锁定规则。

2. **感官分层取舍**：以 2 种感官为主进行细化，氛围允许时增至 3 种（如触觉+听觉），其余一笔略写。禁止全维度均匀堆砌。
   - 本章感官优先顺序：**触觉 > 听觉 > 视觉**
   - 触觉：嘴唇的触感、舌头的温度、手指抚过皮肤的痒/麻、龟头被含住的湿热、后穴被扩张的异物感、龟头顶开括约肌的撕裂般撑开感、被填满的饱胀感、事后拥抱的体温

3. **叙事节奏交替**：一段密集动作/触感描写 + 一段短留白 + 一句轻环境过渡循环。禁止连续长描写、连续对话、通篇等长段落。性爱场景要有层次推进——挑逗→试探→插入→适应→余韵的完整节奏。

4. **情绪外化，不直述**：张野的紧张不写「他感到紧张」，写「他站在门口搓手指，不敢乱看」；疼痛不写「他很疼」，写「眼眶瞬间红了，腹肌绷到发抖，闷闷地哼了一声」；安心不写「他感到幸福」，写「他把脸埋进陆辞怀里，不想说话，不想动」。

## 背景信息

### 前情上下文

第2章结尾：陆辞和张野确认了恋爱关系。陆辞在校园路灯下第一次亲了张野（嘴唇轻轻碰了一下），然后隔衣摸了张野的下面——张野硬了，陆辞隔着裤子掌住揉了揉。张野红着脸把头埋进陆辞肩膀。两人确立了恋人关系，张野知道陆辞是温柔的。

## 案例

**【温柔前戏写作示例 · 感官分层取舍（触觉+听觉）】**

> 陆辞的嘴唇落在他额头上。软的，温的，像在碰什么易碎的东西。张野闭着眼，睫毛一直在抖，呼吸乱了节奏。陆辞的嘴唇从额头滑到鼻尖，停了一下，最后落在嘴唇上——很轻，先是上唇碰下唇，然后舌尖沿着唇缝划过，轻轻撬开他的齿关。舌头探进来的那一刻张野的手不知道往哪放——悬在半空，最后轻轻搭在陆辞腰侧。陆辞的舌头很慢、很耐心，一下一下地舔过上颚，勾着他的舌头打转。张野的呼吸越来越重，喉咙里泄出一点闷闷的气音。

（演示：感官分层取舍·触觉+听觉——以嘴唇的触感（软/温）、舌头的温度、呼吸的声音为主，省略视觉描写，符合场景焦点在「第一次接吻的触感与声音」）

**【首次插入写作示例 · 焦点锁定（核心冲突）】**

## 不可违反规则

### 优先级（从高到低）

1. **约束红线** — 情节红线、角色禁区、边界禁止
2. **字数** — 4500字目标（超限启用压缩策略）
3. **T1词 / 认知动词 / 感官** — 修饰阈值、认知动词节制、感官分层
4. **叙事规则**（高于同层·写作规范其他规则）
5. **写作规范同级其他规则**

### 红线约束

**冲突阶梯**：第1层·入门→第2层·升级——本章以身体亲密度的推进为主要冲突（张野的身体紧张 vs 陆辞的耐心引导），不涉及外部对抗。

**情节红线（不可删减）：**
1. **前戏必须写充分**——从接吻到抚摸到口交到扩张，每一步都是逐级递进的，不能跳步。前戏篇幅应占全文约60%
2. **扩张环节必须写润滑油的使用**——不可跳过扩张直接插入
3. **首次插入时的疼痛必须写足**——龟头顶开括约肌时的热辣撕裂感、张野眼眶红、腰弓起来、手指攥拳、腹肌绷到发抖的反应
4. **陆辞在插入时停下来的画面必须写**——顶进去不到一个龟头就停住，抱住张野亲他眼角等他适应——这是本章最动人时刻
5. **张野说「没事……你动吧……」必须写**——这是张野主动交付信任的关键时刻。说话方式：小声、闷闷的、不娘
6. **章末落在安心感**——事后拥抱清理、张野把脸埋进陆辞怀里「不想说话，不想动，就想一直这么待着」

**边界禁止（不涉及）：**
1. 不要写张野从此不怕性爱——这是第一次，留下的感受是「痛但被爱着」，不是「爽」
2. 不要写陆辞说脏话——本章的陆辞是温柔耐心的恋人形象，脏话是后续攻方的标签，不是本章陆辞的
3. 不让系统在本章出现——系统的寒意属于第5章，本章必须纯甜
4. 不要写「张野觉得爽」——第一次的感觉是「痛→酸胀被填满→不习惯但因为是陆辞所以可以忍」，不是快感高潮
5. 不要让张野在疼痛时叫出声——他疼得眼眶红、闷哼、深呼吸，但不哭不叫不喊停
6. 不要让陆辞在插入时太快——全程「停下来确认反应→继续」的节拍
7. 不要让张野在事后说骚话——他此时还是内向处男，射完把脸埋进陆辞怀里不说话才是他的反应
8. 不要写成分体位的大段机械描述——前戏和情绪推进比体位重要

**角色禁区：**
张野不会主动要求性行为（他接受但不主动）
张野不会说「还要」或「再来」（首次性交的消耗感让他只想被抱着）
张野不会在事后问「我们是什么关系」（他已经知道）
陆辞不会说脏话（温柔恋人设定，脏话属于后续攻方）
陆辞不会急促推进（全程「等张野反应→继续」的节奏）

### 网文风格基线（任何章节适用）

1. 认知动词节制使用（≤2次/章——「感到」「觉得」「意识到」等）
2. 因果关系自然呈现，不强行解释
3. 叙事自然有温度——禁止极端白描（纯动作描写无情感流动）
4. 全程线性叙事
5. 拒绝完美叙事——允许留白和思绪跳跃
6. 禁止刻意堆砌技法
7. 禁止诗词腔

### 风格建议（可偏离）

- **视角**：第三人称有限，紧贴张野。不写他人内心。禁止全知解释。
- **描写**：用动作感知推动叙事，感官服务于情绪。本章以**触觉（嘴唇的温度/被抚摸的触感/被撑开的痛感和饱胀感/事后的体温）和听觉（呼吸声/闷哼声/陆辞的安抚声）** 为主要感官。
- **节奏**：段落自然起伏，句式长短交替。性爱场景要有递进：试探→渐入→突破→适应→余韵。前戏部分节奏舒缓，疼痛部分节奏收紧（短句+动作的快速切换），安心部分节奏又放缓。
- **张野的说话方式**：小声、闷闷的、不娘、不扭捏、语速慢。疼的时候也是闷哼不是尖叫。事后的沉默比说话更有力量。
- **量化写实**：陆辞的鸡巴17cm/匀称直挺/粉褐色，写尺寸但不夸张——这是温柔的第一次，不是后续的粗暴大鸡巴。不搭配粗俗比喻（棒球棍/驴屌等），用温和的描写。
- **不与后续攻方对比**：本章不暗示「以后会有更大的/更粗的」，保持纯甜的语境完整。
- **张野的生理反应**：写但不极端——眼眶红、闷哼、呼吸变重、手指攥床单、腹肌绷紧，但不写翻白眼/流口水/痉挛等（那些属于后续堕落阶段的反应）。
- **陆辞的温柔**：通过动作细节展示——不是「他很温柔地」，而是「停了一下，没有急着推进，低头亲了亲他的额头。」

### 场景写作指引

本章涉及3种场景类型（first-time-sex / dialogue / transition）：

**首次性交场景（first-time-sex）—— 本章主体（关键点3-8）：**
1. **感官分层取触觉+听觉优先**——张野第一次体验性行为，触感（嘴唇/手/龟头/被撑开的感觉）和声音（呼吸/闷哼/安抚低语）是核心描写通道。视觉辅助（灯光下的影子/眼眶泛红/攥床单的手指）。
2. **节奏控制**——从接吻（关键点3）到抚摸（关键点4）到口交（关键点5）到扩张（关键点6）到插入（关键点7）到开始抽送（关键点8），逐级递进，每一步不能跳。前戏的节奏是舒缓的、试探性的；插入的节奏是紧张-疼痛-停顿-适应-继续的节拍。
3. **情绪推进**——比身体推进更重要。每一阶段的情绪变化：紧张→被安抚→刺激→疼痛→被爱着的安心。身体动作服务于情绪线，不是反过来。

**对话场景（dialogue）：**
1. 本章对话量少但每个对话都有功能——关键点2的「紧张？」「没事，慢慢来」建立安全感；关键点6的「放松」是安抚；关键点8的「没事……你动吧……」是信任交付。
2. 对话标签全部用动作替代（倒水、搭手、轻拍大腿、抱紧、亲额头），不写「陆辞说」「张野说」。
3. 张野的对话简短（3-5个字），闷闷的低声，不主动开启话题。

### 质感要求

- **中高权重场景（关键点3、4）**：正常叙事节奏，接吻和抚摸的逐级递进感为主

- **低权重场景（关键点1、9）**：快速转场和章末收束

**额外约束（本章特有）：**
1. **首次插入的痛感必须写到**——不是「有一点不舒服」就带过去，是要写出龟头顶开括约肌时的热辣撕裂感、张野的生理反应（腰弓起来、眼眶红、腹肌绷到发抖），以及陆辞停下来的温柔回应
2. **「没事……你动吧……」的情感分量**——这是张野主动交付信任的瞬间，说话方式必须对（小声、闷闷的、不娘），后续抽送开始时的节奏必须保持陆辞的耐心风格

5. **不要使用粗俗比喻**——陆辞的17cm鸡巴不需要用棒球棍/矿水瓶等比喻（那是后续攻方的写法）。本章是温柔的第一次，用温和的描写——「匀称直挺」「温热的硬物」「不粗不细刚好能填满」
6. **前戏占全文至少60%**——关键点2-6（对话建立安全→接吻→抚摸→口交→扩张）的篇幅总和应该超过关键点7-8（插入到抽送）的篇幅
