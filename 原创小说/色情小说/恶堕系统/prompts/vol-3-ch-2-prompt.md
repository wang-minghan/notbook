# Prompt — 第三卷·第2章「工地板房」

## 角色

你是一个都市题材（urban）小说作家。现代都市背景，所有场景发生在真实城市空间——工地板房、破旧行军床、节能灯管发黄的10平米空间。超自然元素仅限「阿鼻系统」本身（本卷系统暂不出现），其余世界观完全写实。不涉及修仙、穿越、异能战斗等非都市元素。

本章是**第三卷（底线碾碎期）第2章**——本书全书最暴力的肉戏章节。张野从乙醚昏迷中醒来，被呈大字型绑在工地板房的破行军床上，嘴被封箱胶带缠死。赵铮坐在床边喝白酒，24cm的粗黑鸡巴没有任何前戏直接捅进干涩的屁眼。没有润滑、没有温柔、没有试探、没有快感——只有疼痛、恐惧和被彻底支配的无力感。

**风格定位：** 全书最暗的章节——暴力＞＞肉度。本章不是GV硬核风格（第三卷还不是GV期），而是「纯暴力性侵」的写实描写。张野的反应不是被操出快感的闷哼，是被封住嘴从喉咙最深处挤出来的惨叫。笔触要**冰冷、直接、不美化**——用短句、硬动词、真实痛感。不给读者任何「也许还有点爽」的错觉。这就是强奸，写清楚。

**与第三卷第1章风格的区别：** 第1章「路边摊」是绑架章（恐惧在未知和悬念中），基调是「日常→猝不及防的黑暗」。本章从第1章的悬念出发——读者知道张野被绑架了但不知道被谁绑的——直接给出最残酷的答案：被工地板房里的包工头绑了，接下来是暴力性侵。笔触要从第1章的「冰冷绑架感」转为「粗暴性暴力感」——写身体的痛、绳子的勒、胶带的窒息、直肠被强行撑开的撕裂感。

**与第二卷任何肉戏章节的区别：** 第二卷（肉体沦陷期）的肉戏虽然粗暴（张威），但张野的身体在逐渐适应并获得快感。本章完全相反——张野被绑着、被堵着嘴、没有前戏、没有润滑、24cm直接捅进干屁眼。反应不是闷哼是惨叫，身体没有「被操开」的适应过程，只有「被强行撑开」的撕裂痛感。本章没有一丝快感，没有一丝享受，没有一丝「身体背叛意识」的张力——只有纯粹的、单向的暴力。

**全卷基调（第三卷）：** 本书最暗卷。张野将经历绑架、迷奸、内尿、公开露出——这些经历的核心是「尊严被碾碎」，不是「性快感」。本章作为第三卷第一个肉戏章节，必须建立这个基调：这不是第二卷那种「被操服」的进阶——这是底线被彻底碾压的开始。

### 叙事目标

1. **核心悬念**：赵铮是谁？他要干什么？会持续多久？
   → 本章结束时：**部分兑现**——读者知道了绑匪是工地包工头赵铮，知道了他的意图（暴力性侵），知道了他的尺寸（24cm/极粗/无前戏），但不知道他要操多久、他还要做什么（后续还有颜射口爆/内尿/露出）。悬念从「他是谁」转为「接下来还有多残忍」。

2. **读者情绪**：刚醒来的困惑（和主角一起）→ 确认被绑的恐惧 → 直面赵铮的压迫感（酒味烟味汗味）→ 看到24cm鸡巴的视觉冲击（不可能塞进去的绝望）→ 白酒喷龟头时的残忍细节（酒精刺尿道）→ 插入瞬间的剧痛（视线发白/惨叫被封在胶带里）→ 持续操干中的破碎意识（不知道还要多久的恐惧）→ 章末的无力感（被彻底支配的定调）
   → 终点落在「纯粹的恐惧和疼痛」——毫无快感，只有被彻底支配的无力感。支撑卷的下一步（第3章「颜射」——屈辱的进一步升级）。

3. **新钩子建立**：
   - 赵铮说了"干一夜"或类似暗示——这远没有结束，读者对接下来的章节产生持续的紧张预期
   - 赵铮的性暴力范式不是陆辞的温柔，不是张威的粗暴但有控制——赵铮是纯粹的、无差别的泄欲暴力，读者不知道他还会有哪些更过分的动作
   - 张野被绑着的状态将持续——他没有逃走的能力，后续章节会在同样的绑缚状态下展开


## 写作方法论

### 本章感官策略

本章以**触觉（外部压迫）+ 视觉（压迫感）** 为主要感官通道——不同于vol-2-ch-4内感受优先的策略，本章要写的是「被外部力量强行侵入」的痛感。

- **触觉（外部压迫）优先：** 绳子的勒感（手腕脚踝被粗麻绳勒住皮肉的疼——不是装饰性的绑缚，是真的勒进肉里的疼）、胶带的紧绷感（脸颊被胶带拉紧的疼、鼻呼吸时胶带内壁的胶水味）、白酒喷在龟头上的冰凉和刺疼（酒精进尿道口的烧灼感）、龟头顶在肛口的异物感（干涩的、没有润滑的强行撑开）、24cm鸡巴在直肠里捅穿的痛感（不是撑开是捅穿的感觉——脊椎通电般的痛、腹肌痉挛、视线发白）。所有触觉都是「痛」和「不适」，没有任何「舒服」或「爽」的触觉。

- **视觉关键（压迫感）：** 24cm鸡巴的视觉压迫——小孩手腕那么粗的龟头、黑红发亮的柱身像工地钢管、青筋盘虬的样子——这些视觉必须写，因为视觉是「不可能塞进去」的绝望的最直接来源。张野看到那根东西时的本能恐惧——身体往后缩、被封住的嘴发出恐惧的闷声——视觉触发恐惧。此外：节能灯管的昏黄光线下的板房环境映射被困的氛围、赵铮黑壮的轮廓在灯光下的压迫感、行军床铁架在灯光下的影子。

- **触觉（内部）：** 只写「痛」——肠道被强行撑开的撕裂感（不是快感的撑开，是干涩的、没有润滑的、异物入侵的痛）、龟头撞到深处的钝痛（像被什么撞到了腹腔里的某一处，不是前列腺的快感酸胀，是撞到内脏一样的闷痛）、括约肌被撑到极限的灼烧痛（肛口像是被撕开了一道口子）。

- **听觉补充：** 肉体撞击的啪啪声（比vol-2-ch-4的更响、更干——因为没有润滑的水声）、行军床铁架的吱呀声（铁架快散架的噪音）、张野被封住嘴的惨叫（不是闷哼，是喉部和鼻腔挤出来的气流声和破碎音——像有人捂着你的嘴让你尖叫的效果）、赵铮的脏话和喘气声。没有水声——因为没有润滑，只有干涩的肉体碰撞。

### 叙事节奏交替

一段密集触感/痛感描写 + 一段短停顿（视野发白/缺氧的空白）+ 一句环境切换。禁止连续长描写、连续对话、通篇等长段落。

### 本章注意点

- **「惨叫不是闷哼」是本章的第一核心区分**：整章张野的反应必须始终是「被封住嘴的惨叫」——从喉咙最深处挤出来的、被胶带压碎的低沉呜咽声。即使到了章末体力耗尽后声音变小变碎，本质上也是「惨叫被碾碎后的残响」，不是「舒服的闷哼」。这个区分是本章与第二卷任何肉戏的本质区别——第二卷张野在张威身下发出的是闷哼（带快感），本章发出来的是惨叫（纯痛）。

- **白酒喷龟头细节的「残忍感」来自它的随意性**：赵铮喷白酒不是经过深思熟虑的虐待手段——他只是在工地上随手拿起一瓶白酒，"消毒"一下就开始干了。这种随意性比刻意虐待更让人不寒而栗——对赵铮来说，喷白酒跟他吐口唾沫抹龟头是同一性质的动作，都是"弄一下就行"的工具性工序。写的时候不要给这个动作加任何意义——它就是赵铮干活前的惯例步骤。

- **24cm鸡巴的视觉描写不要写成黄色小说式的「硬起来」描写**：不要用「青筋暴起好有男人味」「粗长的鸡巴让人腿软」这类被操服后的视角。张野看到24cm时的反应是——「那根东西像小孩手腕那么粗。不可能。不可能塞进去。不可能。」——是拒绝式的认知冲击，不是「好大」的惊叹。笔触要维持张野的恐惧视角，不要滑向GV式的炫耀描写。

- **「被绑着」的绝望感需要贯穿全章的每一个动作描写**：张野的所有本能反应——想蜷缩、想合腿、想翻侧身体、想往后缩——都在「被绑着」这个前提下被打断。每写一个「他想……」后面都要跟一个「但被绑着做不到」。这是章内重复出现的递进式绝望——越写越深。

- **不要追求「疼痛的文学化」——用生理证据代替修辞**：痛感不需要形容词堆砌——视线发白、耳鸣、呼吸断裂、肌肉痉挛、反射性的弓起和弹动——这些客观生理反应比任何形容词更有力量。当张野痛到视线发白的时候，不需要说「他疼到无法忍受」——白光的画面已经说了所有的话。

- **全程保持「张野的有限感知」，不要跳到赵铮视角**：不写赵铮插进去时的感受（紧不紧、爽不爽、暖不暖）、不写赵铮看到张野身体时的想法（好不好看、值不值得）、不写赵铮的性快感（他射了没有、他爽不爽）——赵铮的一切都通过张野的感知过滤：张野听到他的喘息和脏话、张野感觉到他的动作节奏、张野看到他身上的汗和粗金链子的光——仅此而已。

- **章末悬置感的技术实现**：章末落在「持续的、没有结束迹象的」动作中——不写赵铮射精、不写张野晕倒、不写有人敲门或电话响、不写天快亮了。最后一个画面可以是灯管、行军床的吱呀声、赵铮的粗喘——三者的持续。让读者合上这一章时的感觉是憋着气的——下一章再呼吸。

- **乙醚醒来的真实感**：张野醒来的过程不是一睁眼就清醒——是碎片化的感知先恢复（嗅觉→触觉→听觉→视觉），意识在几行文字内逐步拼接完整。写清楚嘴里塞着的东西（布团/胶带）、鼻腔呼吸时胶水的气味、脑袋还沉着的乙醚残留感。
