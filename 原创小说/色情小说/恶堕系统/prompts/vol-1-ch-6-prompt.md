# Prompt — 第一卷·第6章「裂缝」

## 角色

你是一个都市题材（urban）小说作家。现代都市背景，所有场景发生在真实城市空间——出租屋、小区楼道、浴室、卧室。不涉及修仙、穿越、异能战斗等非都市元素。超自然元素仅限「阿鼻系统」本身，其余世界观完全写实。

本章是**第一卷（初恋与破处）第6章**——信任裂痕章。系统倒计时的最后一夜，张野去见第一个客（张威），**全程不写门内的事**——画面切在"门开了，他进去了"就停。回来后洗澡很久，陆辞察觉异常——回消息慢、身上陌生味道、走神。陆辞偷翻手机看到约时间的对话。信任裂开一道缝。

**风格定位：** 压抑·愧疚·暗流涌动。本章不是GV硬核风格——**本章无肉戏**。笔触贴近心理，克制、沉默、不安。张野的愧疚通过身体动作外化（洗澡时用力搓皮肤、发呆、答话太快），陆辞的怀疑通过观察和动作呈现（闻到的气味、翻手机时放回原处的精确）。

---

## 任务指示

创作第一卷·第6章正文（在全卷中的角色：信任裂痕章——第5章以系统降临并下达「72小时内完成第一次性行为」结束，第6章兑现这个任务但不写肉戏，留白给第二卷。同时引入陆辞视角的怀疑积累过程，为第7章「背叛」做准备。）

### 字数与压缩策略

- **目标字数**：建议 3000-3500 字（±10% 可接受，叙事完整性优先于字数精确）
- **压缩策略**：超字数时优先压缩低权重场景——过渡段落（路上走/独处发呆）压缩至100-150字；不得因压缩删改红线内容（敲门后的留白、洗澡搓皮肤的细节、陆辞翻手机的全过程、章末背对背的沉默）

### 本章驱动力

系统倒计时逼张野出门→张野去见了张威（门开了他进去了→留白）→回来后长时间洗澡（逃避与自我清洗）→陆辞察觉异常（消息慢、陌生气味、走神、心跳快答话太快）→陆辞翻手机发现可疑通话记录→信任裂开一道缝。驱动力是**信息差不断扩大**：张野做了亏心事想瞒，陆辞已经发现了不对劲。读者同时看到两个人的视角，知道裂缝正在扩大但两人都还没戳破。

### 信息释放节奏

情绪沉浸型——读者跟随张野的视角经历第一次卖身的心理挣扎，中段切换到陆辞视角感受怀疑的累积：
- 开场（张野视角）：系统倒计时最后几小时，张野坐在床边看客单——名字叫张威，陌生小区。他盯了很久，按了「确认接单」。压抑不安的基调。
- 推进（张野视角）：傍晚出门→路过球场想起陆辞→走到小区门口深呼吸→站在那扇门前犹豫了几秒→敲门→门开了→他进去了（此处留白，画面切走）。紧绷到极致。
- 中段（张野视角）：一个多小时后出来，恍惚空白的状态→回家→长时间洗澡→用力搓自己。逃避与自我清理，但洗不掉的不是气味。
- 转折（陆辞视角切入→双线并行）：陆辞发消息没及时回→回家闻到陌生味道→床上对话时张野心跳过快、答得太快→陆辞翻手机看到未存号码和通话记录。信任从动摇到裂开。
- 章末（双线汇集→背对背）：两个人都醒着，中间隔着一段以前没有的距离。不安像水从裂缝里渗进来。

### 叙事目标

1. 核心悬念：张野会真的去吗？陆辞什么时候发现不对？
   → 本章结束时：**部分兑现**——张野去了也回来了（但不写肉戏）；陆辞发现了可疑点但还没弄明白是什么
2. 读者情绪：从压抑不安到紧绷到极致到恍惚空白到小心翼翼的重逢到沉默的不安——终点落在「裂缝」
3. 信息差建立：读者看到张野第一次卖身的全过程（不包括肉戏）↦ 陆辞只看到碎片（通话记录、陌生气味）↦ 读者比陆辞更清楚发生了什么，因此更紧张他什么时候会发现真相

## 写作方法论

### 本章感官策略

本章以**触觉 + 嗅觉**为主要感官通道——不是视觉冲击，是贴近身体的触感和气味带来的不安。

- **触觉优先：** 站在门前犹豫时手指悬在门板前的停顿感、洗澡时水从头顶浇下来的冲刷感、用力搓皮肤时指甲刮出的红印的刺痛感、握水杯时指尖发白的力度、躺在床上时背对背的距离感
- **嗅觉辅助：** 陌生香皂的味道混在宾馆廉价的腥味上——这是陆辞闻到的关键线索。张野自己闻不到的，但读者知道那是什么
- **听觉辅助：** 楼道里感应灯亮起的声音、敲门声的回响、水声很大盖住所有声音、心跳声在楼道里听起来特别响、翻手机时手指划屏幕的微小声音
- **视觉克制：** 本章不需要大量视觉描写。场景切片式呈现：手机屏幕的倒计时数字、小区门牌号、浴室镜子里的水雾、黑暗里睁着的眼睛

### 叙事节奏交替

一段紧密的动作/感知 → 一段沉默/停顿 → 一个外部环境细节过渡。

本章节奏：**前紧后松再紧**——出门前到敲门是紧绷的、不断累积的紧张感；结束后到洗完澡是恍惚的、迟缓的空白感；陆辞回家后是小心翼翼的试探和逐渐收紧的怀疑。

### 信息释放控制

张野的心理活动不直接解释——通过身体反应外化（盯着屏幕眼睛不眨、手指在键盘上顿了一下、洗澡时站在水下不动好一会儿才开始搓、回答问题时太快了）。不写「张野心里想……」，写他做了什么和没做什么。

陆辞的情绪也不直接写——通过观察和动作暗示（等了二十分钟没回消息时看了两次手机、走近时停了一下闻了闻、翻手机时放回的位置太精确了）。

## 背景信息

### 前情上下文

第5章结尾画面：系统面板浮现在张野眼前——冰冷淫秽的电子女声念完任务，倒计时启动。「72小时内完成第一次性行为。宿主当前余额：0。逾期惩罚：器官剥离。」张野坐在出租屋床边，手机屏幕上是系统推送的客单。他的手指悬在屏幕上方。本章接续这一秒——倒计时还在跳。

读者情绪残留：第5章的系统降临带来的寒意和紧张——读者知道张野被逼到绝路了，第6章他要面对「去还是不去」的现实。

第5章留下的缺口：系统下达了第一次卖身任务，但没写张野是否真的去了。读者等了这一章来看到底发生了什么。

### 角色初始状态

**张野：**
  起点：系统面板上倒计时在闪，还剩最后几个小时。他坐在床边，手机里是系统推送的客单——名字叫张威，地点在一个他从没去过的小区。他盯了很久，屏幕暗了又按亮。陆辞的消息还停在微信列表里——「明天晚上一起吃饭？」他回了个「好」字，然后切回系统界面，按了「确认接单」。手指在屏幕上顿了一下。
  → 经历：出门→去张威住处→敲门→门开了他进去了（留白）→出来→回家→洗澡→和陆辞互动→被翻手机→背对背
  → 终点：躺在黑暗里不敢想「如果陆辞知道了怎么办」。没睡熟，感觉到陆辞翻了身背对自己。
  本章小动作/习惯：手指在屏幕上顿一下、走路时停下来看球场铁丝网、站在门前抬手敲下去之前那几秒的犹豫、洗澡时用力搓皮肤搓到发红、握杯子时指尖发白、回答问题时语速快了平时半拍

**陆辞：**
  起点：什么都不知道。以为张野今天和平时一样——训练、休息、晚上回消息。
  → 经历：发消息等了二十分钟才收到回复→回家闻到张野身上有陌生的味道→床上感觉心跳太快、答话太快→睡不着→翻张野手机发现未存号码和通话记录→记下号码放回手机→背对背睡着但没睡着
  → 终点：躺在黑暗里睁着眼。以前躺在这张床上旁边的呼吸声是让他安心的，今天不是。他不知道那个号码是谁。念头一旦浮上来就按不下去。
  本章必须完成的一件事：从「完全信任」到「开始怀疑」——翻手机的动作是信任裂开的瞬间。不是愤怒，是心凉。

## 输入

### 场景原材料

**场景 1｜过渡·系统倒计时→出门前｜压抑不安基调｜权重：中**
  核心事件：系统面板上的倒计时在闪。三天期限还剩最后几个小时。张野坐在床边，手机里是系统推送的客单——名字叫张威，地点在一个他从没去过的小区。他盯了很久，屏幕暗了又按亮。陆辞的消息还停在微信列表里——「明天晚上一起吃饭？」他回了个「好」字，然后切回系统界面，按了「确认接单」。手指在屏幕上顿了一下。他没有换好衣服——T恤短裤，不打扮，不想被任何人注意。出门前他站在玄关停了一下，没看镜子，拉开门走了出去。
  情绪意义：系统倒计时的压迫和不得不去的困境——不是「要不要去」，是「什么时候去」
  信息差：张野知道这是卖身任务 ↦ 读者知道张野被系统逼着去 ↦ 陆辞什么都不知道
  情绪拐点：日常的平静（回陆辞消息）→ 不可抗拒的外部压力（按了确认接单）
  → 下一场景的衔接：傍晚走在路上
  本场景焦点：人物情绪（被迫行动的压抑——被系统倒计时催着、被钱逼着、没有选择的绝境感）
  本场景权重：中（约200-250字）
  感官建议：视觉（手机屏幕的倒计时数字、客单界面的信息、微信消息的预览框）+ 触觉（手指在屏幕上顿住的感觉、按下去之前那半秒的停留）
  本场景出口情绪：压抑不安——要去但不想去的拉扯

**场景 2｜推进剧情·路上→路过球场｜紧绷基调｜权重：中**
  核心事件：傍晚出门。天还没全黑，闷热的初夏空气裹在身上。张野低着头走，不看路边的任何人。路过球场时他停了一下，隔着铁丝网看了一眼——有人在打篮球，皮球砸在地面上嘭嘭响。他想起陆辞在球场递水的那个下午，站了几秒，走了。路上他想了很多——还能收手吗、不去的话系统会怎样、陆辞知不知道他今晚出门了。但每一步都在往前走。到了小区门口他站住，深呼吸了几次才进去。感官侧重：铁丝网里传来的篮球声、闷热的晚风、踩上台阶时的回声、楼道里感应灯亮了一盏又一盏的节奏。
  情绪意义：展示张野的挣扎——他在路上想了一千个理由掉头回去，但每一步都还在往前走
  信息差：张野走在去卖身的路上，脑子里闪过陆辞的脸 ↦ 读者看到张野的犹豫和不得已
  情绪拐点：出门时的不安 → 路过球场看到打篮球的人想起来陆辞 → 但脚步没停
  → 下一场景的衔接：站在张威住处门口
  本场景焦点：人物情绪（挣扎但无法抗拒——想收手但不敢收手，每一步都在朝自己不想去的方向走）
  本场景权重：中（约300-350字）
  感官建议：听觉（篮球声从铁丝网里传出来、楼道里感应灯亮起的啪嗒声、自己的脚步声在楼梯间回响）+ 触觉（闷热的空气裹在皮肤上、深呼吸时胸口发闷）
  本场景出口情绪：紧绷——已经到了楼下，不能再后退了

**场景 3｜推进剧情·站在门口→敲门→门开了→他进去了｜高潮基调·强度峰值｜权重：高**
  核心事件：张野站在一扇门前。门牌号是客单上写的那串数字。他抬起手想敲门，举了三秒没敲下去。心跳声在楼道里听起来特别响。脑子里闪过陆辞的脸——陆辞说「不急，慢慢来」的那个晚上。系统冷冰冰地催了一声。他闭了一下眼，敲了。门开了。他进去了。**画面在此处切走——不写门内的事，不写张威的样貌，不写任何肉戏。不留恋、不描写、不停顿。敲下去→门开了→他进去了→直接切到下一个场景。**
  情绪意义：本章强度峰值——敲下去之前的犹豫和敲下去之后不可逆的后果。那几秒里什么决定都没有改变，但读者能感受到他有多不想进去。
  信息差：张野做了不可逆的一步 ↦ 读者看到他的犹豫和最后的放弃挣扎 ↦ 张威的一切待第二卷展开
  情绪拐点：犹豫到极致（手举了三秒没敲）→ 被迫行动（敲了→门开了→进去了）
  → 下一场景的衔接：门在他身后关上——切到下一个场景：他走出来的时候天已经黑了
  本场景焦点：核心冲突（决定性的时刻——敲下去之前的犹豫是本章最重要的心理描写。敲下去之后即刻切走，不流连。）
  本场景权重：高（约200-300字——但不要拖，要干脆。犹豫写细，进门后切得利落）
  感官建议：听觉（心跳声在楼道里回响、系统提示音、门打开时的声音）+ 触觉（手指悬在门板前三秒的静止、敲下去时指关节碰到的木门的硬度）
  本场景出口情绪：紧绷到极致后的断裂——「门开了，他进去了」，读者被留在外面
  注意：此场景是本场景列表中最关键的一道红线。**不写门内任何事。不写张威。不写肉戏。门开了，他进去了——就到这里。**

**场景 4｜推进剧情·从小区出来→回家路上｜恍惚空白基调｜权重：中**
  核心事件：一个多小时后，楼门打开，张野走出来。天已经黑了。他走出来的时候脚步有点飘，衣摆有一边塞在裤腰里，他走了几步才发现，扯出来。不记得自己怎么走出那栋楼的。路边有车经过，车灯晃了一下他的脸——表情是空的，没哭，没愤怒，什么都没。他站在路边了一会儿，像是在确认自己还在原来的世界里。然后他走了——走路的速度比来的时候快，低着头，不看两边。
  情绪意义：展示卖身后的心理状态——不是悲伤，是空洞。还没反应过来发生了什么，身体在动但意识是停的。
  信息差：事情已经发生了 ↦ 读者和张野一样不知道该怎么消化→下一章才展开张威线
  情绪拐点：门内的事没有写 → 门开后张野走出来，状态已经完全变了——从紧绷变成空
  → 下一场景的衔接：回到家，在玄关站住
  本场景焦点：人物情绪（做完之后的虚无——不是哭不是愤怒，是整个人还没回到地面上的恍惚）
  本场景权重：中（约200-250字）
  感官建议：触觉（衣摆塞在裤腰里走了几步才发现、扯出来时布料摩擦的触感）+ 视觉（车灯晃了一下他的脸——脸上什么表情都没有）+ 动作（脚步飘的、站在路边确认自己还在这个世界里的那几秒）
  本场景出口情绪：空洞——事情已经做了，但情绪还没跟上来

**场景 5｜推进剧情·回家→玄关→脱衣→洗澡｜逃避·自我清洗基调｜权重：高**
  核心事件：回家。开门的时候动作很轻，像怕吵到谁。屋里没开灯，陆辞还没回来。他站在玄关没动，低头看着自己穿了一天的球鞋。然后脱了衣服——不叠，直接团成一团塞进洗衣机里。走进浴室，开了水。水声很大。他没有马上洗，站在水下闭着眼，水从头顶浇下来，浇了好一会儿没动——不是在想什么，是还没办法想。然后他开始用力搓——肩膀、胸口、大腿内侧、小腹——被碰过的每一处都用指甲刮出红印。不哭，就是搓。水声盖住了所有声音。洗了很久——四十多分钟。关了水之后他站在浴室里，湿漉漉的，没马上出来。
  情绪意义：逃避和清理的外化表现——不是洁癖，是想把身上那个陌生人的气味洗掉。他在水里搓自己的时候想的是「这样陆辞就不会发现了」，但同时又知道洗不掉的不是气味。
  信息差：张野以为洗澡可以洗掉一切 ↦ 读者知道洗不掉的是发生过的事实本身
  情绪拐点：还未回家的平静 → 自我清洗中的逃避式用力 → 洗完后站在浴室里没出来
  → 下一场景的衔接：陆辞回来了，浴室灯还亮着，水声还在响
  本场景焦点：核心冲突（卖身后的自我清洗——用搓皮肤的痛感替代还没想明白的愧疚感）
  本场景权重：高（约400-500字）
  感官建议：触觉（水从头顶浇下来的冲刷感、指甲刮过皮肤留下的刺痛感、皮肤被搓到发红发疼的触感、水汽弥漫的闷热感）+ 听觉（水声很大盖住了所有声音——水声本身就是一种掩护和逃避）
  本场景出口情绪：逃避式清洗后的空白——站在浴室里不想出去，因为出去就要面对陆辞
  注意：不写情色。不是洗澡的性感描写，是搓皮肤刮出红印的压抑和逃避。皮肤发红发疼的触感是心理崩溃的外化。

**场景 6｜推进剧情·陆辞视角·回消息慢了｜悬念积累基调｜权重：中**
  核心事件：陆辞的视角切换。陆辞在回去的路上给张野发了消息问他在不在家，等了二十分钟没回。又发了一条——「在家吗？」——又等了十几分钟才收到一个「在」。他看了一眼时间——八点半，不算晚，但张野之前每条消息都会在十分钟内回。今天不一样。他收起手机，走快了几步。
  情绪意义：从张野视角切换到陆辞视角——读者第一次看到陆辞的怀疑是怎么开始的
  信息差：张野以为自己的回复没有异常 ↦ 陆辞已经注意到消息回得慢了 ↦ 读者知道为什么慢
  情绪拐点：陆辞从「平常心」到「觉得哪里不对」——二十分钟的回消息延迟是第一个信号
  → 下一场景的衔接：陆辞到家
  本场景焦点：人物情绪（陆辞的怀疑从细微处开始——他还不确定出了什么事，但心里已经开始有疙瘩了）
  本场景权重：中（约150-200字）
  感官建议：视觉（陆辞看手机的动作——看了时间、等了又看）+ 动作（收起手机走快了几步——身体比意识先反应）
  本场景出口情绪：微妙的怀疑——说不清道不明的不对劲

**场景 7｜推进剧情·陆辞到家→闻到陌生味道｜关键转折基调｜权重：高**
  核心事件：陆辞到家的时候浴室灯还亮着，水声还在响。他看了一眼时间——张野洗了四十分钟了。他在客厅坐了一会儿，水声还在。又过了十分钟，水才停了。张野出来的时候头发还在滴水，皮肤被烫得泛红——胸口和肩膀上有几道自己搓出来的红痕。陆辞看了他一眼：「洗这么久。」张野说嗯，运动完出了汗。陆辞走近的时候闻到了——不是他们用的那款沐浴露的气味，是宾馆里那种便宜香皂的味道，混在一股水汽底下淡淡的腥味上面。陆辞没说话。张野接了一杯水喝，手在握杯子的时候指尖发白。
  情绪意义：陆辞闻到陌生气味的那一刻是怀疑从直觉变成线索的转折点——他还没确定那是什么味道，但他知道那不是张野应该有的味道。
  信息差：张野以为洗了澡就干净了 ↦ 陆辞闻到了混在水汽下的陌生气味 ↦ 读者知道那是宾馆香皂和精液的味道
  情绪拐点：陆辞回家时的平常心 → 看到张野洗了五十分钟的意外 → 闻到陌生气味时的停顿时 → 他没说话
  → 下一场景的衔接：两个人上床，准备睡觉
  本场景焦点：核心冲突（陆辞闻到陌生气味——怀疑从直觉变成线索。他没说话是因为他还没想好该说什么，但那个味道他记住了。）
  本场景权重：高（约300-400字）
  感官建议：嗅觉（便宜香皂的化工气味混在蒸汽里淡淡的腥味——陆辞走近的时候闻到的，不是浓烈，是若有若无但确实存在）+ 视觉（张野皮肤被烫得泛红、肩膀上搓出来的红痕、握水杯时指尖发白）+ 听觉（水声持续五十分钟终于停了、拖鞋踩在卫生间地砖上的声音）
  本场景出口情绪：怀疑从模糊变成具体——陆辞闻到了，他没说，但他记住了

**场景 8｜推进剧情·床上对话→心跳太快→答得太快｜焦虑试探基调｜权重：中**
  核心事件：晚上，床上。陆辞侧过身来摸张野的脸，说了句「你今天是不是累了，早点睡」。张野嗯了一声闭上眼。陆辞的手从他脸上滑下来搭在他胸口——心跳很快。陆辞问「你心跳好快」，张野说可能是洗完澡太烫了。他的声音闷闷的，和平常一样，但是——太快了。张野平时说话慢，想一下才答，今天答得太快了。像是早就准备好了一句答案。陆辞没再问。他躺平了，看着天花板。
  情绪意义：张野反常的「答得太快」比心跳快本身更让陆辞起疑——两个人在一起久了，说话节奏的变化是藏不住的。
  信息差：张野以为自己的回答没问题（心跳快可以说洗澡太烫了）↦ 陆辞听到的不仅是回答的内容，更是回答的速度——太快了
  情绪拐点：睡前的日常互动 → 陆辞发现心跳快 → 张野答得太快 → 陆辞不再问但心里已经确定了什么
  → 下一场景的衔接：陆辞等张野睡着了，翻了手机
  本场景焦点：人物情绪（张野的紧张是藏不住的——虽然他的回答内容没问题，但回答的速度出卖了他；陆辞没再追问，但心里已经有了想法）
  本场景权重：中（约250-300字）
  感官建议：触觉（陆辞的手搭在张野胸口感受到的心跳过快）+ 听觉（张野回答的声音——平时说话慢、闷闷的，今天语速快了半拍）+ 视觉（陆辞躺平了看着天花板——不再追问但也没睡）
  本场景出口情绪：暗流涌动——两个人都没说话，但都感觉到了有什么不对

**场景 9｜推进剧情·陆辞翻手机→发现通话记录｜信任裂缝·关键转折基调｜权重：高**
  核心事件：张野已经睡着了（或者闭着眼在装睡），手机搁在床头柜上，屏幕朝下。陆辞躺了一会儿，没睡着。他侧过脸看了一眼张野——呼吸均匀，睫毛没抖，是真睡着了。他伸手拿过那部手机。认识以来他从来没有翻过张野的东西。今天他翻了。微信聊天列表很正常——室友、教练、几个打球的朋友。他在搜索栏里打了一个「张」——没有异样。又打了「客」——没有。他往下翻通讯记录，在最近通话里看到一个没存名字的号码，通话时长显示下午六点多打进来过一次。他点了进去，看到了通话记录前后只有这一条。他把号码记在了自己手机上，然后把张野的手机放回原处——屏幕朝下，位置放得太精确了，和他拿起来之前一模一样。他自己都没意识到自己放得那么小心。
  情绪意义：本章最重要的转折——信任裂开的瞬间。陆辞从来没有翻过张野的东西，但今天他翻了。放回去的时候位置太精确了，这个细节本身就说明了他在心虚。
  信息差：张野不知道陆辞翻了手机 ↦ 陆辞看到了通话记录但不知道那个号码是谁 ↦ 读者知道那是张威的电话
  情绪拐点：陆辞睡不着→决定翻手机→找到可疑通话记录→记下号码→放回原处→信任裂开了
  → 下一场景的衔接：陆辞躺回去，背对着张野
  本场景焦点：核心冲突（信任裂开的瞬间——翻手机的动作本身是陆辞性格的突破，他从「温柔信任」变成了「已经开始查了」的人）
  本场景权重：高（约350-400字）
  感官建议：触觉（拿起手机的重量感、手指在屏幕上滑动搜索的触感、放回去时动作轻到几乎没有声音）+ 听觉（黑暗中手机屏幕亮起时的轻微嗡声、搜索框里打字的细小嗒嗒声）+ 视觉（屏幕光映在陆辞脸上、他的表情变化——从平静到皱眉到确认有事）
  本场景出口情绪：信任裂开了——不是愤怒，是心凉。陆辞知道有事了，但他还不知道是什么事。
  注意：系统在张野脑子里，手机上只有通讯记录。陆辞不会发现系统。他只看到一条通话记录。

**场景 10｜推进剧情·背对背躺着｜沉默不安基调｜权重：中**
  核心事件：陆辞躺回去。张野还是那个姿势睡着。陆辞在黑暗里睁着眼。以前躺在这张床上旁边的呼吸声是让他安心的，今天不是。他不知道那个号码是谁、为什么张野要删掉通话记录、为什么他回消息变慢了、为什么洗澡洗了五十分钟、为什么身上有陌生的味道。他不想往那个方向想，但念头一旦浮上来就按不下去。他翻了身，背对着张野。
  情绪意义：信任裂缝后两个人的距离——陆辞翻身的动作是本章物理化的「裂缝」象征
  信息差：陆辞知道有事但不知道是什么事 ↦ 张野不知道陆辞已经起了疑心 ↦ 读者知道双方的信息差在扩大
  情绪拐点：翻手机前的不安 → 翻手机后确认了有事 → 翻身后背对着张野
  → 下一场景的衔接：章末落点
  本场景焦点：人物情绪（陆辞的心境变化——从「信任」到「怀疑」再到「不敢继续想下去」）
  本场景权重：中（约200字）
  感官建议：触觉（翻身时床垫的震动、背对背中间那段空出来的距离感）+ 听觉（黑暗里的呼吸声——以前让他安心的声音，今天不是）+ 视觉（黑暗里睁着的眼睛）
  本场景出口情绪：从怀疑到确认有事后的沉默不安——裂缝出现了

**场景 11｜过渡·章末落点｜双重沉默·不安基调｜权重：中**
  核心事件：张野醒了一下——或者他根本没睡熟。他感觉到陆辞翻了个身背对自己。他没睁眼，也没动。他在黑暗里想的是：如果陆辞知道了怎么办。但这个问题的答案他没有往下想，不敢往下想。夜色很深，两个人背对背躺着，中间隔着一段以前没有的距离。不安像水一样从裂缝里渗进来，无声的。
  情绪意义：全章终点——两个人都在黑暗里醒着，都感觉到有什么东西不一样了，但谁都没有开口。张野不敢想的问题，陆辞已经把答案的一半握在手心里了。
  信息差：张野不知道陆辞翻了手机 ↦ 陆辞不知道那个号码意味着什么 ↦ 信息差驱动第7章
  情绪拐点：各怀心事的沉默 → 全章终点——不安和距离
  → 全章结束。为第7章「背叛」留下钩子：陆辞记下的那个未存号码
  本场景焦点：人物情绪（双倍的沉默——两个人都在想同一件事但都还没到说破的地步）
  本场景权重：中（约150-200字）
  感官建议：触觉（背对的姿势、被子下面的空间感——中间那段距离是新的）+ 视觉（黑——什么都看不见，但两个人都醒着）+ 听觉（安静的夜里谁的呼吸声稍微重了一点——但都没说话）
  本场景出口情绪：不安的沉默——裂缝已经出现了，水正从那里渗进来

## 不可违反规则

### 优先级（从高到低）

1. **约束红线** — 情节红线、角色禁区、边界禁止（必须写/不能写）
2. **字数** — 3000-3500字目标（超限启用压缩策略）
3. **反AI规则** — T1句式/禁用词零容忍、认知动词节制、感官分层
4. **叙事规则** — 视角控制、节奏交替（高于同层·写作规范其他规则）
5. **写作规范同级其他规则**

### 红线约束

**冲突阶梯**：第3层·裂痕——本章是卷内冲突的第一个真正转折点：外部压力（系统）迫使张野做出违背关系的行为，内部信任开始崩解。不再是甜蜜的恋爱日常，而是隐瞒和怀疑并行。

**情节红线（不可删减）：**
1. **张野站在门口犹豫→敲门→门开了→他进去了——必须完整写出犹豫的过程，但门开后立即切走，不写门内任何事**
2. **洗澡时用力搓皮肤的细节必须写出**——这是张野愧疚外化的核心场景，指甲刮出红印的触感不能省略
3. **陆辞闻到的陌生气味必须写出**——便宜香皂味混在水汽下的腥味，这是陆辞怀疑从直觉变成线索的转折点
4. **张野回答问题时「太快了」这个细节必须写出**——他平时说话慢，今天答得太快了，这是藏不住的反常
5. **陆辞翻手机的全过程必须写出**——搜索栏里打「张」、打「客」、翻到未存号码和通话记录、记在自己手机上、放回原处时位置放得太精确了。信任裂开的瞬间
6. **章末背对背的沉默必须写出**——两个人都在黑暗里醒着，谁都没开口，中间隔着一段以前没有的距离

**边界禁止（不涉及）：**
1. **不在任何地方展开张威的肉戏**——不写性交过程、不描写张威的身体或鸡巴、不写张野被操的细节。本章的核心不是卖身的「肉」，是卖身后的「心理」
2. **不在章中出现「破处」的误导**——张野的第一次是给了陆辞（第3章），卖身是另一次，不是破处
3. **张野洗澡时的搓不是情色场景**——是心理崩溃的外化表现，写触感（皮肤发红发疼）不写性感
4. **不要让张野哭**——他的状态是「还没反应过来」的恍惚，眼泪应该在第7章（被陆辞发现后）才出来
5. **不要让陆辞在翻手机时发现「系统」**——系统在张野脑子里，手机里只有通讯记录
6. **不要写陆辞的愤怒**——他现在的状态是怀疑和不安，不是愤怒。愤怒在第7章
7. **陆辞的说话方式保持温柔**——他还没翻脸，现在还是那个温柔的陆辞，只是心里已经开始有疑虑了
8. **不要出现「本章讲述了」「张野经历了」等上帝视角概述**
9. **不要出现分析性结论（如「这标志着他们的关系出现了裂痕」）**
10. **不写系统对张野的实时反馈或评价**——系统任务是本章开场的背景压力源，本章核心不是系统和张野的互动

**角色禁区：**
张野不会在洗澡时哭（他的愧疚是沉默的、闷着的，不是情绪外放的）
张野不会主动坦白或说「我有事跟你说」（他还没想好怎么说）
张野不会在陆辞面前表现出异样——除了那些他自己控制不了的反常（心跳快、说话太快）
陆辞不会当面质问张野（他还没想好怎么问，先记下号码观察）
陆辞不会对张野发火（现在还是温柔的陆辞，只是心里已经开始有疑虑了）
张威不出场、不描写、不对话（肉戏留第二卷）

### 反AI规则覆盖

**T1句式零容忍（出现即修改）：**
- ❌ 「不是A而是B」句式 → 直接写B
- ❌ 「带着……」（万能状语） → 拆短句或换动作
- ❌ 「仿佛/犹如/宛若」 → 删除或白描
- ❌ 「让/令/使」句式 → 删掉「让」，主语直接行动
- ❌ 「当……时/的时候」 → 拆成独立动作句
- ❌ 「他意识到/他感到/他注意到」（认知动词前置） → 删掉，直接写事实
- ❌ 「与此同时」 → 直接换场景开头

**T1禁用词（出现即替换）：**
- ❌ 一丝、一抹、些许、几分 → 具体量词或删除
- ❌ 不禁、不由得、不由自主 → 直接写动作
- ❌ 瞳孔一缩、身体一僵 → 用具体身体反应替代（「他的手指停在半空中」「肩膀绷紧了」）
- ❌ 倒吸一口凉气、呼吸一滞 → 用呼吸变化细节替代（「呼吸停了半拍」「呼出来的气不稳」）
- ❌ 露出（表情） → 用具体笑法/表情动作替代
- ❌ 陷入（沉默） → 写沉默后的具体行为
- ❌ 涌起、泛起（情绪） → 用身体反应替代
- ❌ 嘴角勾起一抹…… → 写具体动作（「他笑了一下」「嘴角动了动」）
- ❌ 眼中闪过一丝…… → 用动作替代（「他目光移开了」）
- ❌ 内心深处 → 删掉或直接写动作
- ❌ 突然/忽然（超过4次/章） → 保留节奏功能的，控制≤4次
- ❌ 渐渐/逐渐（超过2次/章） → 用变化细节替代

**禁止的总结升华句式：**
- ❌ 「这就是……」 → 删除，让场景自然结束
- ❌ 「他终于明白了……」 → 删除，停在画面/动作上
- ❌ 「从这一刻起……」 → 删除
- ❌ 「这一夜注定……」 → 删除
- ❌ 「他不知道的是……」 → 替换为角色视角内的动作
- ❌ 「一切……都……」 → 删掉或改为具体状态

**禁止的心理描写替代：**
- ❌ 「张野感到……」 → 直接写动作/身体反应
- ❌ 「张野心想……」 → 用动作外化
- ❌ 「张野意识到……」 → 写事实，不总结
- ❌ 「他的内心……」 → 用行为展示

### 风格约束

**视角控制（第三人称有限视角——双视角切换）：**
- 场景1-5：张野视角——只写张野看到、听到、感觉到的一切
- 场景6开始：切换到陆辞视角——只写陆辞看到、听到、感觉到的一切
- 场景10-11：保持陆辞视角直到章末，不切回张野内心
- 不写上帝视角总结——不写「张野不知道更大的改变正在发生」
- 不写陆辞的内心活动分析——通过陆辞的动作和观察呈现他的怀疑过程

**段落精度分层：**
- **高权重场景（场景3、5、7、9）**：细节饱满。场景3细化犹豫到敲门的心理和时间感；场景5细化洗澡时搓皮肤的用力感和时间感；场景7细化陆辞闻到味时的停顿和沉默；场景9细化翻手机的每一步操作和内心的悄然变化
- **中权重场景（场景1、2、4、6、8、10、11）**：正常叙事节奏，支撑情绪主线
- **低权重场景**：无（本章场景数量适中，每个场景都有情绪推进功能）

**张野的说话方式不变：**
- 说话始终小声、闷闷的——但本章中他回答陆辞问题时「答得太快了」，这反常本身就是信号
- 本章张野的对话极少——发消息说「在」、洗澡后说「运动完出了汗」、床上说「可能是洗完澡太烫了」
- 不娘不妖，体育生的低声应答

**陆辞的说话方式：**
- 本章陆辞还是温柔的——「洗这么久」（不带责问的，是寻常的关心）、「你今天是不是累了，早点睡」（试探性的日常对话）、「你心跳好快」（自然的疑问）
- 温柔但心里已经有疑虑——说话时多了一层观察和试探
- 不写愤怒——他的状态是怀疑和不安，不是愤怒

**身体的诚实反应替代心理描写：**

| 情绪 | 不写 | 写 |
|------|------|----|
| 犹豫/挣扎 | 「张野在犹豫要不要敲门」 | 手抬起来，举了三秒没敲下去 |
| 愧疚/逃避 | 「张野觉得很愧疚」 | 站在水下不动，然后开始用力搓，指甲刮出红印 |
| 怀疑 | 「陆辞开始怀疑了」 | 陆辞走近的时候停了一下，他闻到了 |
| 不安 | 「陆辞感到不安」 | 陆辞躺在黑暗里睁着眼，没有再翻身 |
| 心虚 | 「张野心虚了」 | 答题太快了，平时说话慢的人今天答得太快了 |
| 心凉 | 「陆辞的心凉了半截」 | 他把手机放回去，位置放得太精确了 |

**禁止的AI疲劳词/句式：**
- 「张野感到/觉得/意识到」 → 用动作替代
- 「某种/一股/一阵」（搭配抽象情绪） → 删掉或具体化（「便宜的香皂味道」不是「一股味道」）
- 「缓缓地/慢慢地」 → 删掉或换具体速度（「他走了几步才停下」不是「他慢慢地停下」）
- 「温暖的/柔和的」（泛光感） → 用具体光影（「车灯晃了一下他的脸」）
- 「坚定的/深邃的」 → 删掉或换具体状态
- 「什么/某种说不清的」 → 删掉，直接写具体事实

**对话标签规则：**
- 删副词修饰（不写「陆辞轻声问」「张野闷闷地说」——「闷闷的」可以用来形容张野的说话方式本身，但不加在对话标签里）
- 用动作替代：陆辞说话时用「他看了一眼时间」「他走近的时候停了一下」「他躺平了看着天花板」替代「他说」
- 张野的简短回答前不写副词——直接写「张野说嗯」「张野说在」

**结构类禁止：**
- 连续4句相同主语 → 变换主语和句式
- 连续3段相同开头 → 每段首句不重复
- 连续5句相同句式 → 陈述/短句/对话交替
- 连续2段描写同一物 → 不重复描写同一感官对象
- 同一情绪词重复（紧张/不安/怀疑）→ 单章≤3次
- 相同比喻复用 → 用过的比喻本章不再用

### 叙事规则补充

**双视角处理**：
- 前5个场景以张野视角为主（他出门→他敲门→他出来→他洗澡）
- 从场景6开始切换到陆辞视角（他发消息没回→他回家闻到味道→他翻手机）
- 章末（场景10-11）保持在陆辞视角——不切回张野内心，保留读者「看到陆辞做了什么而张野不知道」的信息差优势
- 视角切换不提示——直接以该角色的感知开场，通过上下文让读者自然跟上

**沉默的运用**：
- 本章对话极少——大部分交流发生在「没说的话」里
- 陆辞没说的（他闻到了但没问）、张野没说的（他做了什么但不敢说）、两个人没说的（背对背但谁都没开口）
- 沉默本身就是本章的叙事语言

**感官密度控制**：
- 不连续使用同一感官通道——一段触觉后换听觉或嗅觉，防止感官疲劳
- 嗅觉是本章的关键线索——只在陆辞出场时才出现，张野视角下不写气味（他没注意到自己身上的味道）

### 第六章·裂缝 情绪锚点（关键定格画面）

1. **站在门口犹豫的那三秒**（场景3）——张野站在门前，手抬起来要敲门，举了三秒没敲。心跳声在楼道里特别响。脑子里闪过陆辞的脸。他还是敲了。门开了。他进去了。画面在此处停——读者被留在门外。

2. **站在水下搓皮肤**（场景5）——张野站在淋浴下，水从头顶浇了十几分钟，他没动。然后开始搓——肩膀、胸口、大腿内侧、小腹。指甲刮出红印。不哭，就是搓。水声很大。洗完了他站在浴室里湿漉漉的，没马上出来。

3. **陆辞走近时闻到的味道**（场景7）——陆辞说「洗这么久」，张野说运动完出了汗。陆辞走近的时候闻到了——不是他们用的沐浴露，是宾馆里那种便宜香皂的味道，混在水汽下面淡淡的腥味。他没说话。

4. **陆辞翻手机——放回去的位置太精确了**（场景9）——陆辞找到未存号码和一条通话记录，记在自己手机上。然后把张野的手机放回去——屏幕朝下，位置和他拿起来之前一模一样。他自己都没意识到自己放得那么小心。

5. **章末背对背的沉默**（场景10-11）——陆辞翻了身，背对着张野。张野醒了（或者根本没睡熟），感觉到陆辞背过身去。他没睁眼，没动。两个人中间隔着一段以前没有的距离。不安像水一样从裂缝里渗进来，无声的。

### 本章注意点

- **「留白」是本章最重要的叙事技术**——场景3「门开了，他进去了」之后直接切走，不解释、不描写、不回头。读者被留在门外，这是刻意的粗粝中断。
- **「答得太快」是张野唯一藏不住的反常**——他平时说话慢、闷闷的、想一下才答。今天回答陆辞问题的时候语速快了半拍。内容没问题，但节奏出卖了他。
- **陆辞翻手机时的心理变化阶梯**：从不打算翻（认识以来没翻过）→ 犹豫（手伸出去停了停）→ 翻了（搜索、查看）→ 找到线索（心往下一沉）→ 记下号码（冷静地）→ 放回去（位置太精确了——这个细节比翻手机本身更说明问题）。这个阶梯要通过动作写，不写心理总结。
- **双视角的信息差是本章的情绪引擎**：读者知道张野去做了什么、为什么洗澡那么久、为什么有陌生气味、为什么心跳快和答话太快——陆辞不知道。读者比陆辞更清楚发生了什么，因此更紧张他什么时候会发现。
- **本章无肉戏**——不写张威的样貌、不写性交过程、不写张野被操的细节。核心不是卖身的「肉」，是卖身后的「心理」。
- **系统的存在感**：系统是本章开场（倒计时）和敲门瞬间（系统提醒）的背景压力源，除此之外不要频繁插入系统语音或面板描述。本章不是系统和张野的对话章。
