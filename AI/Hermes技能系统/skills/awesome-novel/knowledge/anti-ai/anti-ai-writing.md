# 去 AI 味完整指南

> anti-ai 各 Phase 的方法论基础。涵盖 AI 写作指纹识别、系统性修改方法和改写范例。

---

## 一、AI 写作指纹

AI 生成的文本在统计层面有稳定的特征模式，以下是指纹识别依据。

### 1.1 高频词指纹

AI 偏好使用特定高频词，这些词在人类写作中分布稀疏：

| 类别 | 典型词 |
|------|--------|
| 情态副词 | 突然、仿佛、不禁、依然、逐渐 |
| 判断弱化 | 似乎、也许、大概、有些、某种 |
| 书面连接 | 然而、与此同时、由此可见、综上所述 |
| 万能动词 | 露出、陷入、涌起、泛起、闪过 |
| 心理引导 | 感到、觉得、意识到、明白 |

### 1.2 章末总结体

AI 在章节结尾处有强烈的总结和升华倾向：

- "他不知道自己即将面对……"
- "从这一刻起，一切都将不同"
- "这一夜，注定……"
- "也许，这就是成长的意义"

### 1.3 叠加式描写

AI 倾向用多层修饰叠加来制造"文学感"：

- 多层定语："他眼中闪过一丝复杂难言的、连他自己也说不清楚的情绪"
- 情绪+动作+环境三件套："他深吸一口气，望着远方的天际线，心中五味杂陈"
- 副词堆叠："他似乎隐隐约约地感觉到……"

### 1.4 均匀分布

AI 在微观层面有稳定的均匀分布倾向：

- **句长均匀**：长短句波动幅度小，很少出现 2-3 字超短句
- **段落均匀**：段落长度分布集中，缺少极短短段（1-2 句）和极长段（10+ 句）
- **词频均匀**：各词汇出现频率分布平坦，缺少核心词的合理重复
- **标点均匀**：句号和逗号比例稳定，缺少破折号、顿号等非常规标点

## 二、核心规则

### 规则 1：段落密度

| 原则 | 说明 |
|------|------|
| 单段只讲一件事 | 动作、环境、对话、情绪，不塞进同一段 |
| 信息密度递减 | 关键信息在前，补充信息在后 |
| 短段优先 | 能用 2 句说完不用 5 句 |

理想段落长度分布（以句数为单位，定性指引非硬性指标）：

| 段落类型 | 建议比例 | 作用 |
|---------|---------|------|
| 独句段 | 建议 10%-30% | 节奏变化、强调（不要整章通篇独句段） |
| 2-5 句段 | 主体叙事 | 常规段落，信息密度适中 |
| 6+ 句段 | 按需使用 | 复杂场景允许，注意信息密度不要过于均匀 |

### 规则 2：动作 + 对话 + 情绪

情绪描写的优先级：

1. **动作**（最好）："他把杯子往桌上一顿"
2. **对话**（次好）："你什么意思"
3. **身体反应**（再次）："他的手在抖"
4. **直述**（最差，仅在关键情绪节点允许1-2句简短直述）："他很愤怒"

### 规则 3：长短句结合

- 长短句自然交替，避免通篇长句或通篇短句
- 关键信息可用短句突出，氛围描写可用长句铺陈
- 长句（>40 字）之间建议有短句隔开调节节奏

### 规则 4：口语化

- 对话要短、要碎、要有留白
- 叙述语言偏向口语节奏，不是书面作文
- 允许"然后""反正""其实"这类日常口语词，但低频

## 三、Show Don't Tell 原则

### 3.1 核心公式

```
Tell: 情绪词/判断句 → 读者被告知
Show: 身体反应/动作/环境/对话 → 读者自己感受
```

### 3.2 对照表

| Tell（禁止） | Show（推荐） |
|-------------|-------------|
| 他很紧张 | 他手指在桌沿敲了三下，又三下 |
| 她很生气 | 她把手机屏幕朝下扣在桌上 |
| 这是个破旧的小镇 | 墙皮剥落的地方露出红砖，巷子里晒着被单 |
| 他感到不安 | 他走两步就回头看一眼 |
| 她很开心 | 她脚尖在地上点了两下 |
| 气氛很尴尬 | 他低头喝水，喝完发现她还在看他 |
| 他心里很乱 | 他把烟点起来，吸了一口又掐灭 |

### 3.3 五感检查

每段场景描写问自己：

- [ ] 视觉——他看到了什么？（不是"美丽"，是"墙上有裂缝"）
- [ ] 听觉——他听到了什么？（不是"安静"，是"只有钟在走"）
- [ ] 触觉——他感觉到了什么？（不是"冷"，是"门把手冰了一下"）
- [ ] 嗅觉——他闻到了什么？（不是"香味"，是"药汤苦得他皱了整张脸"）
- [ ] 动觉——他的身体在做什么？（不是"站着"，是"他换了个脚撑着重心"）

## 四、7 种 AI 写作模式检测

### 模式 1：总结式开头/结尾

```
信号：段首/段尾出现总结性判断
特征："这就是""这意味着""他明白了""总的来说"
修复：删掉总结，停在具体画面/动作上
```

### 模式 2：内心分析报告

```
信号：大段规整心理描写（>3 句连续心理）
特征："他内心深处知道……""一方面……另一方面……"
修复：打碎为 1-2 句，余下的用动作外化
```

### 模式 3：完美逻辑链

```
信号：每句话都是上一句的直接因果结果
特征：没有闲笔、没有走神、没有无关细节
修复：在两句因果之间插入一句无关观察
```

### 模式 4：均匀信息密度

```
信号：每段都在推情节，没有疏密变化
特征：从头到尾信息浓度一致，没有喘息段
修复：选一段插入生活化细节（走神/看窗外/想起无关的事）
```

### 模式 5：模板情绪反应

```
信号：情绪反应是"最合理的选择"
特征：该哭时哭、该怒时怒、该紧张时紧张，全程无可挑剔
修复：改一处为错位反应——该哭时笑、该怒时发呆、该紧张时跑题
```

### 模式 6：全线平滑

```
信号：所有段落转换都顺畅自然
特征：没有突兀跳跃，每段开头都有平滑承接
修复：如果整章所有段落转换都平滑无变化，考虑删掉1-2段过渡句制造节奏变化。
注意：不要为硬切而硬切——自然的过渡是必要的。
```

### 模式 7：标点规整

```
信号：句号和逗号比例稳定，缺少非常规标点
特征：几乎不用破折号、顿号、省略号；句幅变化小
修复：加 1-2 处用顿号/省略号的碎句，制造标点不规则感
```

### 模式 8：对称填充

```
信号：句子结构对称工整，前后分量均衡
特征："既X又Y""不仅X更Y""从A到B"
修复：打破对称，只保留强的一侧，弱侧直接删掉或改成独立的短句
      不是"既紧张又兴奋"→"他紧张得手心出汗。又有点兴奋。"
      不是"从清晨到日暮"→"天亮了。又黑了。"
```

### 模式 9：机械列举

```
信号：显式的顺序或平行结构，读起来像提纲
特征："首先…其次…最后""第一…第二…第三""一方面…另一方面…"
修复：打碎为自然叙述，用时间/因果/感受承接代替序号词
```

### 模式 10：反问与假设引导

```
信号：先抛一个反问/假设再回答，制造不必要的悬念感
特征："如果我告诉你……""你有没有想过……""想象一下……"
修复：删掉引导句，直接陈述。读者不需要先被问再被告知。
```

### 模式 11：正能量收尾

```
信号：章末出现鸡汤/励志/希望感总结
特征："未来可期""拭目以待""来日方长"（含变体）
修复：删除，停在具体的画面/动作上。情绪留给读者自己消化。
```

### 模式 12：无源引用

```
信号：提到研究/数据/证据但不给出处
特征："研究表明""据统计""调查显示"
修复：删掉"研究表明"或补具体来源。小说中绝大部分可以直接删掉。
```

## 五、系统性去 AI 三遍法

### Pass 1：去泛化（~80% 的 AI 味集中在此）

**目标**：替换抽象/套话表达为具体细节。

步骤：
1. 全文扫描禁用词（★★★★★ 最毒句式 → 一级禁用词）
2. 替换情态词（仿佛/一丝/不禁 → 删除或白描）
3. 替换情绪词（愤怒/难过 → 身体反应）
4. 替换模板描写（眼中闪过一丝 → 动作）
5. 替换句式（不是A而是B → 直接写B）

验证：禁用词命中率下降 80% 以上。

### Pass 2：去书面化

**目标**：书面腔/翻译腔 → 口语和动作。

步骤：
1. 扫全知叙述（他不知道…… → 删除）
2. 扫心理分析（他内心深处 → 动作外化）
3. 去内心辩论赛（一个声音说…… → 保留结论）
4. 简化对话标签（他愤怒地说 → 动作承接）
5. 去副词修饰（他缓缓地 → 删掉）

验证：对话标签密度 <10%，无全知叙述。

### Pass 3：回自然感

**目标**：让文本有人写的痕迹——自然的停顿和节奏变化。

步骤：
1. 检查节奏是否过于均匀——如有，制造自然的起伏（长短交替、信息密度变化）
2. 加闲笔（选一段插入生活化细节，不服务于主线）
3. 加错位（建议至少 1 处情绪反差或异常反应）
4. 修结尾（升华/总结/预告 → 画面/动作）

验证：段落长度有自然变化，至少 1 处闲笔，至少 1 处情绪反差。避免刻意碎句。

## 六、改写范例库

### 6.1 情绪外化

**愤怒：**

❌ "他心中涌起一股无法遏制的怒火"

✅ "他把杯子往桌上一顿。水溅出来，顺着桌沿往下淌。他没擦。"

**紧张：**

❌ "他紧张得手心冒汗"

✅ "他在裤腿上蹭了一下手。又蹭了一下。桌沿被他摸得发亮。"

**悲伤：**

❌ "她感到一阵难以言说的悲伤"

✅ "她把外套叠了三叠，放回衣柜最里面。拉链没拉到头。她又拉开了，重新拉好。"

**惊讶：**

❌ "他震惊得说不出话来"

✅ "他张了张嘴。没出声。又张了一下，还是没出声。"

### 6.2 场景描写

**环境：**

❌ "夕阳西下，天边的云彩被染成了金红色，美得令人窒息"

✅ "六点一刻，太阳还没落完。影子拉得很长，他的影子比他自己高两个头。云不红，是灰的，像没洗干净的抹布。"

**天气：**

❌ "雨下得很大，仿佛天空破了一个洞"

✅ "雨打在铁皮棚上，声音大得听不见人说话。他的裤脚湿透了，鞋里也进了水。"

**建筑：**

❌ "城堡巍峨耸立，塔尖直入云霄"

✅ "城堡墙上有裂缝。他路过的时候数了数，有七条。最长的那条能伸进去两根手指。"

### 6.3 打斗

❌ "他身形一闪，躲过对方的攻击，随即一个回旋踢正中对手胸口"

✅ "对方一拳过来。他偏头，慢了半拍，拳风擦过耳朵，耳朵嗡了一声。他没回旋踢——他没那么利索——他往前一撞，脑门磕在对方鼻梁上。"

### 6.4 结尾

**错误示范：**

❌ "他望着远方的天际线，心中充满了对未来的期待。他不知道的是，更大的挑战正在前方等着他。这一夜，注定不平凡。"

✅ "他把烟掐灭。鞋底碾了一下。转身往回走。"

**错误示范：**

❌ "从这一刻起，他终于明白，真正的勇气不是无所畏惧，而是心怀恐惧却依然前行"

✅ "他站起来。膝盖上沾了两片草叶。他拍了一下，没拍掉。算了。"

### 6.5 节奏对比

**AI 均匀节奏：**

"他走进房间，环顾四周，房间里很安静，只有钟在走。他拉开窗帘，阳光照进来，灰尘在光柱里飘。他在床边坐下，床垫弹了一下。"

**真人碎节奏：**

"他推开门。钟在走。没别的声音。
窗帘拉开的瞬间，灰在光柱里翻了一下，又沉下去了。
他往床上一坐。弹了两下。"

### 6.6 身体反应替换

**瞳孔一缩：**

❌ "他的瞳孔微微一缩"

✅ "他看了三秒。低头翻了一页文件。又翻回来看了一遍。"

**倒吸一口凉气：**

❌ "他倒吸一口凉气"

✅ "他往后靠了靠。椅背硌得后背疼。他又往前坐了回来。"

**身体一僵：**

❌ "他身体一僵"

✅ "他脚钉在原地没动。旁边有人绕开他走过去，他也没注意到。"

**心中涌起：**

❌ "他心中涌起一股暖意"

✅ "他把那杯茶捧在手心里。没喝。茶凉了也没放下。"

**露出/脸上露出：**

❌ "他脸上露出一个微笑"

✅ "他笑了一下。笑到一半，发现没人接，又把笑收了回去。"

## 七、质量检查清单

### 修改后自查

- [ ] 无 ★★★★★ 最毒句式
- [ ] 无一级禁用词
- [ ] 二级禁用词未超阈值
- [ ] 每段只讲一件事
- [ ] 段落长度有自然变化，避免全线均匀
- [ ] 至少 1 处闲笔（走神/看窗外/无关细节）
- [ ] 至少 1 处情绪反差（反常反应/错位反应）
- [ ] 对话标签不密集（无连续 3 句相同标签）
- [ ] 结尾停在画面/动作上，不升华不总结不预告
- [ ] 无"仿佛/犹如/宛若"类比喻
- [ ] 破折号 ≤3 次/章，无一段内多次滥用
- [ ] 段落有长短变化，无连续同长度段
- [ ] 情绪靠动作/对话/身体反应，不靠心理直述
- [ ] 与原文剧情一致，无新增/删减情节
