# 降阶3D物理模型训练系统

**时间：** 2025.04 — 2025.07  
**归属：** [[北京灵易数智-工作经历]]

---

## 项目概述

基于多种降阶与代理建模方法，构建面向 CAE 仿真的 3D 物理模型训练系统，实现高维工程数据的快速推理与预测。

## 核心技术亮点

### 1. 算法突破
- 首次将 **Kolmogorov-Arnold Network（KAN）** 应用于 CAE 仿真降阶
- 突破传统神经网络表达能力限制

### 2. 多模型融合（Ensemble）
- 集成四种算法：RBF、Kriging、DNN、KAN
- 构建 ensemble 预测框架，根据数据特征智能选择最优模型

### 3. POD 降维优化
- 实现**本征正交分解（POD）**技术
- 通过能量阈值控制降维
- 保持 **99%+ 能量**的同时将维度压缩至原来的 **5%**

### 4. 数据处理 Pipeline
- 支持 **VTU 格式解析**（Fluent/Comsol/Mechanical）
- 自动化数据清洗、归一化处理

### 5. 分布式训练
- 设计异步训练框架
- 支持多 GPU 并行
- **训练效率提升 12 倍**

### 6. 微服务部署
- RESTful API + Docker 容器化
- 支持水平扩展
- **平均推理延迟 < 50ms**

## 技术栈

`KAN` `POD` `RBF` `Kriging` `DNN` `CAE` `VTU` `Docker` `RESTful API` `GPU并行`
