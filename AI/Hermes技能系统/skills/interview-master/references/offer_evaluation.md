# Offer 评估与决策框架

---

## 一、单个Offer的评估清单

拿到一个offer后，不要只看薪资数字。用以下5个维度全面评估：

### 维度一：经济回报

```
□ 年度总包是多少？（月薪×月数+年终奖+签字费+股票）
□ 和你的市场价值相比，是高于、等于、还是低于？
□ 每年的调薪机制是什么？（普调比例、绩效调薪幅度）
□ 晋升后的薪资涨幅通常是多少？
□ 公积金缴纳比例是多少？（直接影响实际收入）
□ 有没有住房补贴、餐补、交通补等？

综合判断：
  3年累计收入预估 = 第一年总包 + 第二年（考虑调薪） + 第三年（考虑晋升）
  不要只看第一年，看3年累计更有参考价值。
```

### 维度二：成长价值（3年后的你值多少钱）

```
□ 这个岗位能让你积累什么新技能/新经验？
□ 这些技能在市场上的稀缺性如何？需求趋势如何？
□ 公司品牌在简历上的溢价有多大？
  （大厂经历 vs 创业公司经历，对下一份工作的影响不同）
□ 3年后从这里离开，你的选择面是更宽还是更窄？
□ 公司内部有没有成体系的培训和晋升通道？

综合判断：
  如果这份工作让你3年后在市场上的身价提升50%+，
  即使当前薪资低10-20%也值得考虑。
```

### 维度三：业务前景

```
□ 这个业务/产品线是在增长期、成熟期、还是衰退期？
□ 团队是在扩招还是在缩减？
□ 行业整体趋势如何？有没有结构性风险？
□ 公司的财务状况健康吗？（上市公司看财报，非上市看融资轮次和现金流）
□ 竞争对手的情况如何？公司有没有明确的竞争壁垒？

综合判断：
  选择上升期的业务 > 选择高薪但衰退的业务。
  因为上升期意味着更多机会、更快晋升、更有说服力的简历。
```

### 维度四：日常体验

```
□ 平均工作强度如何？加班是偶尔还是常态？
□ 通勤时间多长？有没有远程办公的灵活度？
□ 直属leader的风格和你匹配吗？
  （如果面试中有和leader交流的机会，认真感受）
□ 团队氛围如何？（可以通过脉脉、认识的人了解）
□ 年假多少天？请假方便吗？
□ 公司的管理风格：扁平还是层级？OKR还是KPI？

综合判断：
  日常体验直接影响你每天的状态和心情。
  不要低估这个维度——很多人离职不是因为钱，而是因为"受不了了"。
```

### 维度五：风险因素

```
□ 试用期多长？淘汰率高吗？
□ 有没有竞业协议？范围和期限是什么？有补偿吗？
□ 公司最近有没有裁员新闻或组织调整？
□ 劳动合同的条款有没有不合理的地方？
□ 如果这个offer不如预期，你有退路吗？

综合判断：
  风险不代表不能去，但你需要清楚自己在承担什么。
  特别注意竞业协议——它可能限制你未来1-2年的择业自由。
```

## 二、多Offer对比决策

### 对比模板

当你手里有2-3个offer时，用以下表格做对比：

```
|              | Offer A      | Offer B      | Offer C      |
|:-------------|:-------------|:-------------|:-------------|
| 公司/岗位     | [名称]       | [名称]       | [名称]       |
| 年度总包      | [X万]        | [Y万]        | [Z万]        |
| 3年累计收入    | [估算]       | [估算]       | [估算]       |
| 成长价值      | [高/中/低]    | [高/中/低]    | [高/中/低]    |
| 业务前景      | [上升/稳定/下行] | ...       | ...          |
| 工作强度      | [描述]       | [描述]       | [描述]       |
| 通勤时间      | [X分钟]      | [Y分钟]      | [Z分钟]      |
| 主要风险      | [描述]       | [描述]       | [描述]       |
```

### 决策框架

**如果各维度差不多，用"3年后检验法"：**
> 想象3年后的你回头看今天的选择，哪个选择让你更不后悔？

**如果薪资差距大但其他维度不同，用"薪资差异年化法"：**
> 薪资差的部分 ÷ 3年 = 每年多/少的金额。
> 这个金额对你的生活影响大吗？值得牺牲[其他维度]吗？

**如果纠结于"大厂低薪" vs "小公司高薪"：**
```
选大厂的理由：
  - 品牌溢价让你下一跳更容易
  - 系统化的培训和成长体系
  - 更规范的项目管理经验
  - 社交圈和人脉资源

选小公司的理由：
  - 更高的即时收入
  - 更广的职责范围（一人当三人用 = 快速成长）
  - 可能有股权/期权的高倍回报
  - 更扁平的架构，更快的晋升速度

没有标准答案，取决于你当前最需要什么。
```

## 三、特殊情况处理

### 只有一个offer，要不要接？

```
问自己：
  1. 这个offer是否达到了我的底线薪资？
  2. 我拒绝后还有多少时间和精力继续面试？
  3. 市场上类似机会多不多？
  4. 我目前的紧迫程度如何？（在职/离职/裸辞多久了）

如果：达到底线 + 市场机会少 + 比较紧迫 → 接受是合理的
如果：低于底线 + 市场机会多 + 不紧迫 → 可以拒绝继续找
```

### 已经口头接受了但收到更好的offer

```
这是一个棘手但不罕见的情况：

选项A：遵守口头承诺，去第一家
  - 职业声誉最安全
  - 但可能错过更好的机会

选项B：礼貌地反悔，选择更好的offer
  - 经济上更优
  - 但会损害和第一家公司/HR的关系
  - 某些行业/城市HR圈子小，可能有后续影响

建议：
  - 如果差距不大（<15%），遵守承诺
  - 如果差距很大（>30%）且是职业方向的重大差异，慎重考虑反悔
  - 反悔时的措辞要诚恳，不要编理由，直接说"有了一个更匹配我方向的机会"
  - 书面offer签了之后再反悔，比口头承诺反悔的代价大得多
```

### Offer有效期快到了但还在等另一家

```
策略：
  1. 和催促的这家谈延期
     "我对这个offer非常重视，但我需要做一个慎重的决定，
      能否将offer有效期延长到[日期]？"
     （通常可以延3-5天，超过一周比较难）

  2. 同时催促另一家加快
     "我目前手里有一个offer需要在[日期]前回复，
      想了解一下贵公司这边的流程进度，
      因为贵公司是我的优先选择。"
     （这不是施压，是提供信息帮对方也做决策）
```

## 四、接受Offer后的行动清单

```
□ 确认书面offer的所有条款与口头沟通一致
□ 签署并返回offer/劳动合同
□ 如果有竞业协议，仔细阅读条款
□ 和现公司提出离职（如果在职）
□ 确认入职日期、报到流程、需要准备的材料
□ 向未选择的公司礼貌回复拒绝
   "感谢您的offer和整个面试过程中的时间。
    经过慎重考虑，我决定接受另一个更匹配我当前职业方向的机会。
    希望未来有机会合作。"
□ 入职前做好知识准备（了解公司业务、团队结构、行业动态）
```
