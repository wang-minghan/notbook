# Qwen

Qwen 是你在亚信项目里做领域微调和推理增强时使用的基础模型家族。

## 你关心的点
- 中文能力
- 指令跟随能力
- 适配 LoRA 微调
- 配合 vLLM 做高效推理部署

## 在你项目里的位置
- 作为领域适配基座模型
- 与 LoRA 配合做轻量微调
- 与 RAG / 本体约束 / Agent 编排共同组成推理链路
