# Prompt — 第二卷·第1章「约到了老板」

## 角色

你是一个都市题材（urban）小说作家。现代都市背景，所有场景发生在真实城市空间——出租屋、大学南门、健身房、器械区、休息室。超自然元素仅限「阿鼻系统」本身（任务推送、惩罚倒计时、视网膜面板），其余世界观完全写实。不涉及修仙、穿越、异能战斗等非都市元素。

本章是**第二卷（肉体征服）第1章**——第二卷的开端，也是张野正式进入系统卖身阶段的第一章。本章是铺垫章，无肉戏。基调是**紧张、尴尬、被迫**。

**风格定位：** 过渡铺垫章·紧张＞肉度。本章不是GV硬核风格——是情绪铺垫和人物建立。张野分手后一周的状态（空了、空转）、发现客人是认识的人的尴尬、被系统惩罚倒计时逼着留下的被迫感、以及面对张威（满身腱子肉/花臂/寸头/34岁粗犷社会人）的紧张。所有情绪不靠心理描写直述，靠动作和细节外化。

**与第一卷风格的区别：** 第一卷（第1-8章）基调是恋爱/心碎，张野有陆辞的温柔支撑。第二卷从这里开始，基调转向肉体征服——张野唯一能依靠的人不在了，他独自面对系统任务，面对一个完全陌生且压迫感极强的客人。写作笔触要从暖色转为中性/冷感，从「把情绪写软」转为「把情绪写硬」——张野的难受不是哭出来的，是攥着床单指节发白那种。

## 任务指示

创作第二卷·第1章正文（在全卷中的角色：第二卷肉体征服的起始章——第一卷结尾在第8章「抉择」中张野心死接了系统任务，本章展示这个任务的具体执行过程。读者等了这一章——从第8章结尾张野点下「接单」开始，读者就在等「第一个客人是谁」和「张野怎么面对」。）

### 字数与压缩策略

- **目标字数**：建议 3000-3500 字（±10% 可接受，叙事完整性优先于字数精确）
- **压缩策略**：超字数时优先压缩低权重场景（场景1过渡/场景5走路过渡），推荐压缩至100-150字以内——保留1句环境切换 + 1句人物动作 + 1句情绪/氛围留白；不得因压缩删改红线内容（发现客人是张威、系统倒计时、章末攥床单）或破坏情绪主线

### 本章驱动力

张野接系统推送的第一个同城任务→出门→发现目的地是办过年卡的健身房→被老板张威认出→尴尬想跑但系统惩罚倒计时逼他留下→穿过器械区走向休息室→浴室犹豫→坐在床上攥床单。驱动力是**从「接了任务」到「已经坐在客人床上」的不可逆推进**：读者想知道第一个客人是谁、张野怎么发现的、他有没有退路、最后怎么样了。每个问题都要给答案，落在「没有退路了」上。

### 信息释放节奏

紧张沉浸型——读者跟随张野的视角体验一步步踏入陷阱的过程：
- 开场：分手后一周。出租屋里安静空荡。张野的状态——不是悲伤，是空。手机和陆辞的聊天记录停在一周前。系统面板推送同城任务。他看了几秒，按了「接单」。
- 推进：傍晚出门。初夏天黑得晚。他穿旧T恤运动短裤，不打扮，低着头走。路过学校南门时停了一下——隔着路就是那家被骗了三万年卡的健身房。他没多看。路上手机震了一下，系统提示客人已确认。
- 转折：按地址到了才发现——就是南门对面那家健身房。站在玻璃门前愣住。想转身走，系统倒计时在视网膜上闪红色数字：「00:05:00」。不走有惩罚。他推门进去了。
- 小高潮：前台没人。张威从里面走出来——34岁，满身腱子肉把黑色背心撑到绷紧，花臂从肩膀铺到手腕，寸头，下巴一圈胡茬。张威认出了他：「操，你不就是那个办了年卡的体育生？办了卡不来练？今天来了就别走了。」张野耳朵红到脖子根，一句话说不出来。
- 继续推进：张威带他穿过器械区走向休息室——健身房的气味（汗味、铁锈味、消毒水），有人跟张威打招呼叫「威哥」，张威走在前面背心勒着肩背肌肉线条。这段路比张野想象的长。
- 章末：休息室里一张单人床靠墙。张威说「先去冲一下」。张野进了淋浴间关上门，没马上开水——站在瓷砖地上看着墙缝里发黑的霉点发了一会儿呆。热水浇下来他才开始洗。出来时张威坐在床边转打火机。他拍了拍床沿。张野走过去坐下，手指攥住床单边沿——指节发白。系统倒计时归零。绿色提示：「任务进行中」。张威的手按在他肩膀上。窗外的天已经全黑了。

### 叙事目标

1. **核心悬念**：第一个系统任务的对象是谁？张野有没有退路？他会被怎么样？
   → 本章结束时：**部分兑现**——第一个客人是张威（健身房老板/满身腱子肉/花臂/寸头），张野没有退路（系统倒计时逼他留下），但肉戏留到第2章
2. **读者情绪**：从空（分手后状态）→意外（发现目的地是健身房）→尴尬（被认出）→被迫（系统倒计时）→紧张（走向休息室）→放弃抵抗（坐在床上攥床单）——终点落在「无路可退的紧张感」
3. **新钩子建立**：
   - 张威的形象（粗犷/肌肉/花臂/寸头/社会人）→ 读者知道第2章他要对张野做什么
   - 系统惩罚机制首次亮出（倒计时强制）→ 读者看到系统的压迫力
   - 年卡被骗三万块的背景 → 张野经济困境的延续，也是他和张威之间尴尬的联系入口

## 写作方法论

### 本章感官策略

本章以**动作 + 触觉**为主要感官通道——不靠心理描写，靠身体行为展示张野的状态。

- **动作优先：** 张野的空和紧张不靠心理描写，靠动作——接任务时盯着屏幕看了几秒、站在健身房门口手按在玻璃门上又缩回来、淋浴间里开了水不洗先发呆、坐在床上攥床单指节发白。每一个动作都是一句情绪。
- **触觉补充：** 浴巾裹在身上的触感、床单布料的粗糙感、张威按在肩膀上虎口的温度——为第2章身体接触做触觉铺垫。
- **视觉克制：** 本章只写必要的视觉——健身房的灯光和器械区人影、张威的花臂纹路、淋浴间瓷砖墙缝发黑的霉点、攥床单发白的指节。视觉服务于气氛（冷感/压迫/失落），不做细致风景描写。
- **听觉：** 系统倒计时在视网膜上无声地跳（只写张野看到面板闪动，不写声音）、健身房器械碰撞声、热水浇地的声音、张威转打火机的咔嗒声——声音是张野在不同空间切换的标志。

### 叙事节奏交替

一段密集动作 + 一段短停顿（发呆/沉默/呼吸） + 一句空间氛围切换。禁止连续长描写、连续对话、通篇等长段落。

本章节奏：中慢——张野的动作之间有空隙（发呆、停顿、犹豫），这些空隙是情绪写生的空间。不赶节奏，让读者感受到「每一步都走得很勉强」。

### 信息释放控制

张野的心理活动不直接解释——通过身体反应和动作外化（盯着手机发呆、按在玻璃门上的手缩回来、淋浴间里开着水不动、攥床单的指节发白）。不写「张野心里想……」，写他做了什么。

张威的情绪也不直接写——通过体型压迫（满身腱子肉撑紧背心）、说话方式（带脏字的调侃、简短指令）和空间占用（走在前面挡光、拍床沿的随意感）来暗示他的主导地位。

## 背景信息

### 前情上下文

第8章结尾画面：张野在出租屋里打开系统面板——任务列表多了一条同城推送。他看了几秒，指腹按在屏幕上方停了一下，然后点下去了。系统提示「已接单」。他放下手机，躺回床上，盯着天花板。手机上陆辞最后一条消息还亮着：「视频我删了。以后别联系了。」

读者情绪残留：心碎 + 对未来卖身之路的未知和担忧——读者知道张野走投无路，系统任务是他唯一的经济来源，但不知道第一个任务会把他推给一个怎样的人。

第8章留下的缺口：系统同城任务接单了但不知道对象是谁——读者和张野一样在等。

### 角色初始状态

**张野：**
  起点：分手后一周。坐在出租屋里。聊天记录停在一周前陆辞最后那条消息。他没删。空——不是悲伤，是什么都感觉不到了，但身体还在运转。
  → 经历：接任务→出门→发现是健身房→被张威认出→尴尬被迫留下→穿过器械区→进浴室→坐在床上
  → 终点：坐在张威休息室的床上，攥着床单，指节发白。知道没有退路了。
  本章小动作/习惯：发呆、低头、手攥东西（手机、布料的边、床单）、耳朵红到脖子根、说话只有单字、走路慢、站在一个地方不动

**张威：**
  起点：在健身房前台后面，听到门响走出来——看到了一个办了年卡一次没来的学生。
  → 经历：认出张野→调侃年卡的事→带着穿过器械区→让张野去冲一下→坐在床边等
  → 终点：手按在张野肩膀上——准备开始。
  本章必须完成的一件事：建立粗犷征服者的第一印象——满身腱子肉、花臂、寸头、说话带脏但不猥琐、豪放直接、体型压迫感强。
  语气：粗犷直接、带脏字但自然（不是刻意耍流氓）、简短指令、调侃但不恶意。

## 输入

### 场景原材料

**场景 1｜过渡·分手后一周→接任务｜空荡基调｜权重：低**
  核心事件：分手后一周。出租屋里很安静。张野坐在床边，手里拿着手机——没在刷，就是拿着，屏幕亮着但没在看。手机里和陆辞的聊天记录停在一周前最后那条「视频我删了。以后别联系了。」他没删聊天记录，但也没再打开看过。系统面板弹出一个新任务推送——同城任务，一个地址，限24小时内确认。面板悬在视野右上角，浅蓝色的字。张野看了几秒。然后他按了「接单」。动作很平静——不是因为准备好了，是因为觉得无所谓了。他把手机装进口袋，没再看聊天记录。
  情绪意义：展示分手后一周张野的状态——不是悲痛欲绝，是「空了」。接任务不是勇敢，是无所谓了。
  信息差：张野不知道任务对象是谁 ↦ 读者和张野一样未知
  情绪拐点：静止的空白 → 按下接单的动作——第一块多米诺骨牌倒了
  → 下一场景的衔接：傍晚出门
  本场景焦点：人物情绪（空——不是悲伤，是什么都感觉不到了。通过动作写：拿着手机没在刷、看了几秒就接单、装进口袋的动作太平静）
  本场景权重：低（快速开场，约150-200字）
  感官建议：动作（拿手机→看屏幕→按接单→装进口袋——几个动作说清楚状态）+ 视觉（手机屏幕的光在暗房间里的一小片亮）
  本场景出口情绪：空的平静——张野不是在「抉择」，他只是在执行一个已经没有所谓的步骤

**场景 2｜推进剧情·傍晚出门→路上｜过渡基调｜权重：低**
  核心事件：初夏天黑得晚，傍晚六点多天还是亮的。张野出门——穿旧T恤和运动短裤，球鞋，不打扮，低着头走。路上的人不多，迎面走来的人他不看。路过学校南门的时候他停了一下——隔着路就是那家健身房，玻璃门面，能看到里面器械区的灯光和跑步的人影。三万的年卡，一次都没来练。他没多看，收回目光继续走。路上手机震了一下——系统提示客人已确认接单时间。他没拿出来看，把手机在口袋里攥了一下又松开了。
  情绪意义：从日常步态到发现健身房时的本能停顿——身体知道那里有不好的记忆，但还在往前走。
  信息差：张野还不知道客人就是健身房老板 ↦ 读者也不知道——但健身房的出现已经是一个暗示（张野生活中有限的人际关系和场所）
  情绪拐点：走路放空 → 路过健身房时停了一下 → 继续走
  → 下一场景的衔接：到了地址才发现就是那里
  本场景焦点：人物情绪（放空走路中的短暂停顿——张野站在路对面看了健身房一眼，那个停顿里有一秒的犹豫）
  本场景权重：低（快速过渡，约150-200字）
  感官建议：视觉（初夏傍晚的光线亮度、健身房玻璃门里漏出的灯光）+ 动作（停了一下、收回目光继续走、口袋里攥手机又松开）
  本场景出口情绪：平静中带一丝不安——健身房的出现像一颗小石子扔进水里

**场景 3｜推进剧情·发现目的地是健身房｜转折基调｜权重：中**
  核心事件：按地址走到的——就是南门对面那家健身房。张野站在门口愣住了。玻璃门擦得很干净，能看到前台和器械区。三万的年卡从来没来练过，现在站在这里是因为要卖身还债。他站在门口，手抬起来按在玻璃门上——然后缩了回来。就在他缩手的瞬间，系统面板弹出来了——红色倒计时数字在视网膜右下角闪：「00:05:00」。没有声音，但数字在跳动，每跳一下颜色深一点。不走的话有惩罚。张野站在门口没动——手还贴在玻璃门上，能感觉到玻璃的温度，比他手掌凉一点。他站了三秒——三秒里他脑子里转了很多东西但又好像什么都没想。然后他推开门进去了。玻璃门在身后关上的声音比他想象的重。
  情绪意义：读者和张野一起发现目的地——尴尬的确认（原来就是骗了钱的那家健身房），以及系统第一次展示强制力（倒计时亮了，不走不行）
  信息差：张野确认了地址就是这家健身房 ↦ 读者和张野一样紧张——他还不知道老板就是张威
  情绪拐点：发现目的地→本能想缩手→系统倒计时亮了（没有退路了）
  → 下一场景的衔接：进门见到张威
  本场景焦点：核心冲突（站在健身房门口——手按在玻璃门上又缩回来的本能反应 + 系统倒计时亮起来的强制确认 + 推门进去的动作）
  本场景权重：中——本章第一个关键转折点。细化这几个动作：走过去→抬头看→愣住→手按玻璃→缩回→倒计时亮→站了三秒→推门
  感官建议：触觉（玻璃门的温度——比手掌凉一点、门推开的阻力）+ 视觉（玻璃门里透出的灯光和跑步人影、视网膜上闪烁的红色倒计时数字）
  本场景出口情绪：紧张 + 被迫——门不是自己推开的，是被倒计时推开的
  对话要点：无——张野没说话

**场景 4｜推进剧情·初遇张威｜高潮基调｜权重：高**
  核心事件：门推开了。前台没人——柜台后面的灯开着但椅子上没有坐人。张野站在入口处没往里走，能听到里面器械区有人在练的声音——杠铃落架的哐当声、跑步机的电机声。过了一会儿，柜台后面传来一个声音——带着点意外和随意：「找谁？」张野转头。张威从里面走出来——34岁，满身腱子肉把黑色背心撑得很紧，花臂从肩膀铺到手腕，纹路在灯光下清晰。寸头，下巴留一圈胡茬。他走出来的时候前台的灯光被他挡了一部分，影子落在柜台上。张威看到张野就笑了——不是客气的笑，是认出来了：「操，你不就是那个办了年卡的体育生？」张野张了张嘴，说了一个字：「呃。」——然后就说不出话了。张威走近了两步，上下打量了他一下。张威比他矮一点，大概180左右，但肩宽体厚，站在面前有压迫感。「办了卡一次不来练，今天怎么有空了？」语气是调侃的，没有恶意，但张野不知道该接什么——手插在裤兜里攥着手机，低着头，耳朵红了。从耳朵尖红到脖子根。不是害羞，是难堪。三万块，一次没来，现在站在这里是因为要卖身还债。
  情绪意义：读者等到张威出场——满身腱子肉、花臂、寸头、粗犷直接，与陆辞（温柔阳光学生）形成鲜明反差。同时展示张野最尴尬的瞬间——被认出来了，而且是被骗了钱那家店的老板认出来了。
  信息差：张威不知道张野是被系统逼来的，以为他是自愿接单的 ↦ 读者知道真相——这个信息差制造不适感
  情绪拐点：走进门的紧张 → 被认出时的尴尬 → 被调侃年卡时的难堪
  → 下一场景的衔接：张威说「今天来了就别走了」——带他往里走
  本场景焦点：核心冲突（张威出场的画面——视觉为主：满身腱子肉撑紧背心、花臂铺满、寸头胡茬；张野的反应：张了张嘴说不出话、耳朵红到脖子根、手在兜里攥手机）
  本场景权重：高——本章第一个标志性场景。张威的出场必须让读者「看到」他——肌肉的撑感、花臂的纹路、寸头胡茬的粗犷感。不是唯美描写，是白描式的视觉冲击。张野的尴尬也必须写足——不是害羞，是难堪。
  感官建议：视觉（张威从柜台后走出来——灯光被他挡住一部分的光影变化、花臂纹路的细节、寸头胡茬）+ 听觉（张威说话的调子——粗犷带脏但不恶意）+ 动作（张野张了张嘴只说了一个字、手在兜里攥手机的动作）
  本场景出口情绪：尴尬到想钻地缝 + 压迫感——张威站在面前的时候，张野觉得自己矮了一截
  对话要点：张威「操，你不就是那个办了年卡的体育生？」「办了卡一次不来练，今天怎么有空了？」「今天来了就别走了」——语气带脏但自然，不是刻意猥琐
  张野「呃」——只有这一个字

**场景 5｜推进剧情·穿过器械区→去休息室｜过渡基调｜权重：中**
  核心事件：张威没急着干嘛，靠在柜台边上跟张野说了几句话——说那张年卡的事，说三万块不是小数目，说「你们这些学生就是被销售忽悠的」。语气是调侃的，没有真的责备的意思。张野低着头嗯了几声——声音闷闷的，很小。然后张威说「走吧，进去说」，转身就往里走了。张威走在前面，背心勒着肩背的肌肉线条——斜方肌从脖子两侧鼓起来延伸到肩膀，背阔肌在背心下面形成倒三角。花臂在走廊灯光下一清二楚，从肩膀到手腕全是彩色的纹身，图案密到几乎没有空隙。张野跟在后面，穿过器械区——经过几个正在练的人，有人抬头跟张威打招呼「威哥」，张威点了下头算是回应。健身房里的气味——汗味、铁锈味（器械的金属味）、消毒水味混在一起，温热又有点闷。这段路比他想象的长。前台的灯在身后越来越远，休息室的门在前面。张野跟在后面走，低着头看着张威的脚跟——没有多余的动作，就是跟着。
  情绪意义：从公共空间到私密空间的过渡——穿过器械区这一路展示张威在健身房的社会角色（老板、大家都叫他威哥），同时空间越来越深、越来越安静，压迫感在累积。
  信息差：张野不知道休息室里是什么样的 ↦ 读者和张野一样——从器械区往里走的时候越来越没底
  情绪拐点：被调侃的尴尬 → 沉默地跟在后面走 → 越来越深的健身房内部
  → 下一场景的衔接：推门进休息室
  本场景焦点：人物情绪（跟着走的沉默——张野没有选择，只能跟在后面。这段路有多长不重要，重要的是他走在这段路上的感觉：越来越远、越来越深、越来越没退路）
  本场景权重：中——空间过渡，但也是张威力量的侧面展示（别人叫他威哥、他的体型在灯光下的轮廓）
  感官建议：视觉（张威背心下肩背肌肉的线条、花臂纹路、器械区的人影灯光）+ 嗅觉（汗味+铁锈味+消毒水——健身房特有的混合气味）+ 触觉（空气的温度——比外面闷一点、比外面热一点）
  本场景出口情绪：压迫感累积——越往里走越紧张
  对话要点：张威简短——「走吧，进去说」；有人打招呼「威哥」，张威点头回应

**场景 6｜推进剧情·休息室｜紧张基调｜权重：中**
  核心事件：休息室在健身房最里面。推开门——十几平的房间，一张单人床靠墙，床单是深色的（深灰或者深蓝），床头柜上放着烟灰缸和一个塑料打火机，角落有个独立淋浴间，玻璃门半开着。墙上挂着一副拳击手套——旧的，皮面有点磨损。门关上之后外面的器械声变小了——不是完全消失，是变成了从远处传来的背景音。房间里有淡淡的烟味和空调的凉气。张野站在门口没往里走。系统倒计时还在视网膜右下角跳——还剩三分多钟。张威已经进去了，走到床边弯下腰按了一下床垫（像在确认床垫的硬度或者干净程度），直起身看了张野一眼：「先去冲一下。」语气不是商量，是指令。说完他侧过身让了让——留出了通向淋浴间的通道。张野站了两秒，然后往淋浴间走。
  情绪意义：进入私密空间——休息室是本章最后的空间场景。这里不像教室或操场，这里有一张床。读者看到床就知道接下来会发生什么。
  信息差：张威不知道张野有多紧张 ↦ 读者看到张野站在门口没往里走，能感觉到他在往后退的那个瞬间
  情绪拐点：走廊的安静 → 推门看到床 → 站在门口犹豫 → 张威的指令推了他一把
  → 下一场景的衔接：进淋浴间
  本场景焦点：人物情绪（站在门口看到床的那个瞬间——张野知道这张床是用来干什么的，但他的脚步钉在门口没动，是张威说了「先去冲一下」才动起来的）
  本场景权重：中——空间和情绪的同步切换
  感官建议：视觉（房间的布局——单人床、深色床单、烟灰缸、打火机、拳击手套、淋浴间）+ 嗅觉（淡淡的烟味+空调凉气）+ 听觉（门关上后器械声变小、空调的嗡嗡声）
  本场景出口情绪：紧张 + 不得不——看到了床就知道没退路了
  对话要点：张威「先去冲一下」（简短指令，不是商量）——张野没有说话

**场景 7｜推进剧情·浴室犹豫｜转折基调｜权重：高**
  核心事件：张野进了淋浴间关上门。门是磨砂玻璃的——外面能看到人影轮廓但看不清细节。他站在瓷砖地上没有马上开水。手撑着洗手台边缘，低头看了一会儿自己的手指——指甲剪得很短，指节上有一道浅疤（打球的时候蹭的）。他站在那里的姿势不是放松的——肩膀微微向前缩，后背的肌肉绷着，能看出他在调整呼吸。水开了——温度很热，热水砸在瓷砖地上的声音很大，瞬间充满了整个小隔间。白气升起来。张野站在水下没有动——让热水从头顶浇下来。水很热，热到皮肤发红，但他没有调凉，就站在那被水冲。淋浴间里全是水蒸气的时候他才关水——把手。擦干的时候动作很慢。然后裹上浴巾（偏小，只够裹住腰到大腿根），推开门出来了。外面的冷气让他缩了一下肩膀。
  情绪意义：张野在浴室里的时间是他最后一次「一个人的时刻」——站在水下不动的那几十秒里他什么都没想，但又好像什么都想了一遍。出来之后就没有退路了。
  信息差：读者知道张野在浴室里站了多久（水从热到蒸汽弥漫）↦ 张威不知道他在里面是什么状态
  情绪拐点：站着不动（犹豫/调整）→ 开水（热水浇下来）→ 关水出来（准备好了/放弃了）
  → 下一场景的衔接：走出来看到张威坐在床边
  本场景焦点：核心冲突（张野在浴室里的状态——不写心理活动，写动作：撑洗手台看自己的手指、站着不动让热水浇、擦干动作很慢、出来时被冷气缩了缩肩膀。每个动作都写「他在想什么」，但不直接说）
  本场景权重：高——本章最重要的情绪段落。张野站在热水下的那段时间是本章的「无声核心」——他在这段时间里完成了从「还在犹豫」到「知道逃不掉了」的转变。
  感官建议：触觉（瓷砖的温度、热水很烫的温度、毛巾摩擦皮肤的触感、出来后冷气缩肩膀）+ 听觉（水声很大、淋浴间里水汽升起的声音）+ 视觉（磨砂玻璃门、蒸汽弥漫时变得乳白色的空气、瓷砖墙缝发黑的霉点）
  本场景出口情绪：放弃了——不是准备好了，是知道没有别的选择了
  对话要点：无——全程无对话

**场景 8｜推进剧情·章末·坐下攥床单｜高潮基调｜权重：高**
  核心事件：张野走出来的时候，张威坐在床边。他的姿势很放松——背靠着墙，一条腿搭在另一条腿上，手里转着塑料打火机，咔嗒、咔嗒，一下一下。他看到张野出来，上下扫了一眼——张野185的体育生身材，宽肩窄腰，浴巾裹在腰上，上半身还挂着没擦干的水珠，胸肌和腹肌的线条在水汽里还没完全干透。张威的目光在他身上停了大概一两秒，然后移开了——没有一直盯着看，就是扫了一眼确认了一下。他拍了拍床沿：「过来坐。」语气还是那样——简短，直接，像叫一个人坐下吃饭一样自然。张野站在那里停顿了一拍——一两秒的停顿，然后走过去了。他走到床边，在张威旁边坐下。床垫在他坐下去的时候陷了一点——弹簧发出一声很轻的吱响。他的手指下意识攥住了床单的边沿——深色床单，布料有点粗糙，边沿缝了一条车线。他攥着那里，指节发白。他没有看张威，目光落在自己面前的床单上——不知道在看什么，就看着那里。系统倒计时在视网膜上归零——00:00:00，数字从红色变成绿色：「任务进行中」。然后一只手按在他的左肩上——张威的手，虎口卡着他的肩胛骨，粗糙、有力、温热。张野没有躲。也没有回头。他闭了一下眼——很短，一两秒，然后睁开，还是看着床单。窗外的天已经全黑了。玻璃窗上能看到室内灯光的倒影和一个模糊的人影轮廓。
  情绪意义：本章强度峰值——张野坐在张威的床上，攥着床单，指节发白。这是他第一次正式系统任务的终点（起点在第8章「按了接单」），也是下一章的起点（第2章会从这里继续——第一次被粗暴插入）。读者猜到接下来会发生什么，但看不到——要等下一章。
  信息差：张威不知道张野是被系统逼来的 ↦ 读者知道——这条信息差让「他拍了拍床沿」这个动作充满压迫感
  情绪拐点：从浴室出来 → 被扫了一眼 → 坐下 → 攥床单 → 倒计时归零 → 肩膀被按住 → 闭眼 → 结束
  → 全章结束
  本场景焦点：核心冲突（本章的终点画面——张野坐在床上攥床单指节发白。所有情绪浓缩在攥床单的手指上。不写张野在想什么，写他攥着床单的动作、发白的指节、闭了一下眼、没有躲也没有回头）
  本场景权重：高——本章的收束画面，也是连接第2章的关键帧。必须细化：坐下时床垫陷下去的细节、手指攥床单的动作顺序（先捏住然后收紧用力直到指节发白）、张威的手按上肩膀的触感（虎口卡着肩胛骨、粗糙的皮肤、温热）、张野闭眼的时长（很短，一两秒）、窗外的天色（全黑）
  感官建议：触觉（床垫陷下去的触感、床单布料的粗糙、攥紧时布料的拉力、张威手掌按上肩膀的温度和粗糙感）+ 听觉（打火机转动的声音——咔嗒咔嗒、床垫弹簧的吱声）+ 视觉（张威扫了张野一眼——不是色情的看，是确认式的看、张野攥床单指节发白的手、窗外全黑的天空和玻璃上的倒影）
  本场景出口情绪：没有退路的紧张——章末落点：攥着床单的手，指节发白，窗外的天已经全黑了
  对话要点：张威「过来坐」（简短指令）

## 不可违反规则

### 优先级（从高到低）

1. **约束红线** — 情节红线、角色禁区、边界禁止（必须写/不能写）
2. **字数** — 3000-3500字目标（超限启用压缩策略）
3. **反AI规则** — T1句式/禁用词零容忍、认知动词节制、感官分层
4. **叙事规则** — 视角控制、节奏交替（高于同层·写作规范其他规则）
5. **写作规范同级其他规则**

### 红线约束

**冲突阶梯**：第1层·入门——本章是第二卷的开端，以场景铺垫和人物建立为主，不涉及正面对抗或性交。

**情节红线（不可删减）：**
1. **张威出场的全身描写必须写出**——满身腱子肉把背心撑很紧、花臂铺满（从肩膀到手腕）、寸头、下巴胡茬、34岁。让读者「看到」这个人的外形
2. **张威认出张野的对话必须写出**——「操，你不就是那个办了年卡的体育生？」「办了卡不来练？今天来了就别走了。」语气带脏但自然，有调侃无恶意
3. **系统惩罚倒计时必须出场**——张野站在健身房门口想走时视网膜上弹出红色倒计时数字（00:05:00），逼他留下。让读者首次确认系统不仅有任务发放功能，还有强制执行机制
4. **穿过器械区走向休息室的段落必须写出**——展示张威的社会角色（别人叫他威哥）、空间的递进感（从公共空间越来越深到私密空间）
5. **浴室犹豫段落必须写出**——张野站在水下不动的几十秒是本章情绪核心。不写心理活动，写动作（站着不动让水浇、擦干很慢）
6. **章末坐下攥床单必须写出**——坐在床上→攥住床单边沿→指节发白→张威的手按上肩膀→闭眼。这是本章的终点画面，也是连接第2章的钩子

**边界禁止（不涉及）：**
1. **本章没有任何肉戏**——不写插入、不写鸡巴尺寸、不写被操的细节、不写张威脱衣服。本章核心是场景铺垫和人物建立
2. **不要写张野在浴室里哭**——他的状态是「不知道该怎么办」的空，不是「想哭但又憋回去」的悲伤。后者的写法在第1卷已经用过了
3. **不要让张野逃跑**——系统惩罚倒计时必须让他留下。手按在玻璃门上可以缩回来一次，但门必须推开
4. **不要让张威变成调情高手**——他粗犷、直接、没有花哨。不是陆辞那种温柔型，也不是赵铮那种阴险型。他就是个豪放的健身房老板
5. **不要对比陆辞**——本章不提陆辞的名字（只在场景1中暗示「聊天记录停在一周前」），不用张威的动作去对照陆辞，不用「和陆辞不一样」这类句式
6. **不要出现分析性结论**（如「这标志着张野正式进入恶堕的第二阶段」「这意味着张野从心碎期进入了肉体沦陷期」）
7. **不要写张威一直盯着张野的裸体看**——他扫了一眼确认了一下就移开了。他是粗犷豪放不是猥琐色狼
8. **不要写张野在去健身房的路上心里揣测客人是谁**——他不知道，也没想那么多。他是空的，不是紧张的期待

**角色禁区：**
张野不会主动说话——本章除了一个「呃」和几声「嗯」以外没有主动对话
张野不会在健身房门口挣扎很久——三秒就是极限了，再多就不像他了（他逼急了就是闷头直接干）
张野不会在浴室里自慰或摸自己——那不是他现在的状态
张野不会在坐下后偷偷看张威——他全程没看张威的脸，只看着床单
张威不会问张野的个人情况——他不会问「你是哪个系的」「多大了」「第一次做吗」
张威不会对张野表现出过度的兴趣或好奇——他只是接了一个单，来的人碰巧是那个办卡不来的学生而已

### 反AI规则覆盖

**T1句式零容忍（出现即修改）：**
- ❌ 「不是A而是B」句式 → 直接写B
- ❌ 「带着……」（万能状语） → 拆短句或换动作
- ❌ 「仿佛/犹如/宛若」 → 删除或白描
- ❌ 「让/令/使」句式 → 删掉「让」，主语直接行动
- ❌ 「当……时/的时候」 → 拆成独立动作句
- ❌ 「他意识到/他感到/他注意到」（认知动词前置） → 删掉，直接写事实
- ❌ 「与此同时」 → 直接换场景开头

**T1禁用词（出现即替换）：**
- ❌ 一丝、一抹、些许、几分 → 具体量词或删除
- ❌ 不禁、不由得、不由自主 → 直接写动作
- ❌ 瞳孔一缩、身体一僵 → 用具体身体反应替代（「他的手指攥紧了床单」「肩膀绷起来了」）
- ❌ 倒吸一口凉气、呼吸一滞 → 用呼吸变化细节替代（「呼吸停了半拍」「呼出来的气不稳定」）
- ❌ 露出（表情） → 用具体笑法/表情动作替代
- ❌ 陷入（沉默） → 写沉默后的具体行为（「他站在水下没有动」「两个人谁都没说话」）
- ❌ 涌起、泛起（情绪） → 用身体反应替代
- ❌ 嘴角勾起一抹…… → 写具体动作（「张威笑了一下」——就够了，不用修饰）
- ❌ 眼中闪过一丝…… → 用动作替代
- ❌ 内心深处 → 删掉或直接写动作
- ❌ 突然/忽然（超过4次/章） → 保留节奏功能的，控制≤4次
- ❌ 渐渐/逐渐（超过2次/章） → 用变化细节替代
- ❌ 温暖的/柔和的（泛光感） → 用具体光影（「前台灯光」「走廊的日光灯」）

**禁止的总结升华句式：**
- ❌ 「这就是……」 → 删除，让场景自然结束
- ❌ 「他终于明白了……」 → 删除，停在画面/动作上
- ❌ 「从这一刻起……」 → 删除
- ❌ 「这一夜注定……」 → 删除
- ❌ 「他不知道的是……」 → 替换为张野视角内的动作

**禁止的心理描写替代：**
- ❌ 「张野感到……」 → 直接写动作/身体反应
- ❌ 「张野心想……」 → 用动作外化（他站了三秒，然后推门进去了）
- ❌ 「张野意识到……」 → 写事实，不总结

### 风格约束

**视角控制（严格第三人称有限——张野视角）：**
- 全章只写张野看到、听到、感觉到的一切
- 不写张威的内心活动——通过张威的对话、动作、体型让读者推测
- 不写上帝视角总结——不写「张野不知道更大的改变正在发生」「张野不知道这个男人将彻底改变他的身体」
- 不写其他健身房会员的内心活动——他们只是背景角色（有人跟张威打招呼叫威哥，够了）

**段落精度分层：**
- **高权重场景（场景4、7、8）**：细节饱满，至少3处动作/感知层级细节。场景4细化张威出场的视觉冲击（肌肉撑背心、花臂铺满、寸头胡茬）和张野的反应（张不开口、耳朵红到脖子根、手在兜里攥手机）；场景7细化浴室里的动作层次（撑洗手台看手指→开水→站水下不动→关水→擦干很慢）；场景8细化章末的画面（走过去坐下→攥床单指节发白→倒计时归零→肩膀被按住→闭眼→窗外全黑）
- **中权重场景（场景3、5、6）**：正常叙事节奏，空间和情绪过渡为主
- **低权重场景（场景1、2）**：快速开场，压缩至100-200字

**张野的说话方式不变：**
- 本章几乎没有主动说话——被张威认出来时只说了一个「呃」，被调侃年卡时「嗯」了几声
- 声音小、闷闷的——不是撒娇不是害羞做作，就是内向的体育生说话声小
- 紧张时话更少或只说单字

**张威的说话方式：**
- 粗犷直接、带脏字但不猥琐——「操，你不就是那个办了年卡的体育生？」
- 调侃但不恶意——「办了卡不来练？今天来了就别走了」
- 简短指令——「走吧，进去说」「先去冲一下」「过来坐」
- 不解释、不商量——他的台词都是陈述句或指令句

**身体的诚实反应替代心理描写：**

| 情绪 | 不写 | 写 |
|------|------|----|
| 空了/麻木 | 「张野觉得心里空空的」 | 他拿着手机但没在刷，看了一会儿按了「接单」，动作太平静 |
| 尴尬 | 「张野很尴尬」 | 耳朵从尖红到脖子根，张了张嘴只说了一个字「呃」，手在兜里攥手机 |
| 被迫 | 「张野不想进去但他没办法」 | 手按在玻璃门上又缩回来——倒计时亮了——站了三秒——推门进去 |
| 紧张 | 「张野很紧张」 | 手指攥床单边沿，指节发白，呼吸变浅，没看张威的脸 |
| 犹豫 | 「张野犹豫了一下」 | 站在淋浴间里不开水，手撑着洗手台低头看自己的手指，站了一会儿才开水 |

**禁止的AI疲劳词/句式：**
- ❌ 「张野感到/觉得/意识到」 → 用动作替代
- ❌ 「某种/一股/一阵」（搭配抽象情绪） → 删掉或具体化
- ❌ 「慢慢地/缓缓地」 → 删掉或换具体速度（「他擦得很慢」「脚步不快」）
- ❌ 「温暖的/柔和的」（泛光感） → 用具体光影（「健身房日光灯的冷白光照在他身上」「窗外天黑了」）
- ❌ 「坚定的/深邃的」 → 删掉或换具体状态

**对话标签规则：**
- 删副词修饰（不写「张威粗犷地说」「张野小声地回答」）
- 用动作替代：张威说话时用「他靠在柜台上」「他转着打火机」「他拍了拍床沿」替代「他说」
- 张野的「嗯」前面不写副词——直接写「嗯了几声」或「闷闷地嗯了一声」

**结构类禁止：**
- 连续4句相同主语 → 变换主语和句式
- 连续3段相同开头 → 每段首句不重复
- 连续5句相同句式 → 陈述/短句/对话交替
- 同一情绪词重复（尴尬/紧张/空）→ 单章≤3次
- 连续2段描写同一物 → 景物/人物不重复描写

**本章特有的氛围规则（第二卷·肉体征服的开端）：**
- 笔触从第一卷的暖色转为中性/冷感——张野的难受不是哭出来的，是攥着床单指节发白那个动作
- 张威的力量感通过体型、动作（拍床沿的随意感、虎口卡肩胛骨）和空间占用来体现——不写他有多「攻」，写他有多「大」
- 健身房的描写保持真实——不是健身广告那种光鲜，是汗味、铁锈味、消毒水混在一起的真实气味
- 系统在本章不仅是一个面板——它还通过惩罚倒计时展示了物理层面的强制力。倒计时数字的颜色变化（红→绿）可以写，但不要过度拟人化（别写「系统似乎在嘲笑他」）

### 第二卷·第1章 情绪锚点（关键定格画面）

1. **按下接单的平静**（场景1）——张野坐在暗了的房间里，手机屏幕的光照着他的脸。他看了系统面板几秒，然后按了「接单」。动作太安静了，不像是做了一个改变生活的决定。

2. **站在健身房门口的手缩回来**（场景3）——手按在玻璃门上又缩了回来。红色倒计时在视网膜上跳。他站了三秒，推开了门。玻璃门在身后关上的声音比他想象的重。

3. **张威认出他的瞬间**（场景4）——张威从柜台后面走出来，满身腱子肉把黑色背心撑得很紧，花臂从肩膀铺到手腕。看到张野就笑了：「操，你不就是那个办了年卡的体育生？」张野耳朵红到脖子根，张了张嘴说不出一句话。

4. **浴室里站着不动的水声**（场景7）——张野站在淋浴间里开了水但没有马上洗。水声很大，白气升起来，他站在水下没有动，让热水从头顶浇下来。淋浴间里全是水蒸气的时候他才关水。

5. **攥着床单的指节发白**（场景8）——张野在张威旁边坐下。床垫陷了一点。他的手指攥住床单的边沿——指节发白。没有看张威。窗外天全黑了。张威的手按在他肩膀上。他没有躲。也没有回头。闭了一下眼。

### 本章注意点

- **「安静」的运用**：本章张野几乎不说话，沉默是他状态的主旋律。但不同的沉默意义不同——场景1的沉默是空的、场景3站在门口的沉默是被迫的、场景7水声中的沉默是最后的犹豫、场景8坐在床上的沉默是放弃抵抗的。通过动作区分沉默的不同质地。
- **张威的粗犷不油腻**：他说话带脏字（「操」）是自然的街语气质，不是刻意猥琐。他看张野的身体只「扫了一眼」就移开了——这不是一个色狼的眼神，是一个老板确认「货到了」的随意。他在本章没有任何性暗示动作（除了最后按在肩膀上的手）。
- **张野的手**：手指是全章反复出现的身体部位——接任务时指腹按在屏幕上、手插兜里攥手机、按在玻璃门上又缩回来、撑在洗手台上看自己的手指、攥床单指节发白。手的状态是张野情绪的最直接外化。
- **空间递进感**：本章的空间动线是单向的——出租屋→路上→健身房门口→前台→器械区→休息室→淋浴间→床边。每一步都在往里走，没有回头路。空间的深度变化要写清楚（前台灯光越来越远、走进器械区、推开门看到床）。
- **系统的出场方式**：系统在场景1是「面板弹出一个新任务推送」（蓝色字），在场景3是「红色倒计时数字闪现」（00:05:00），在场景8是「数字从红色变成绿色：任务进行中」。颜色变化暗示状态变化（待确认→强制执行中→进行中）。别写系统拟人化（不写「系统在催他」、不写「系统像在看着他」）。
- **和第一卷的衔接**：第一章结束在「攥着床单指节发白」——画面感强、落到身体动作上、不总结不升华。和第一卷第8章结尾「按了接单」的动作形成呼应——从「按下去」到「坐下来」完成了一个完整的任务执行弧线。
- **「呃」字的使用**：张野被张威认出来时只说了一个「呃」——这个字要写到位。不是「呃……」拖长音，是一个被噎住的、喉咙里挤出来的单音。写好了比任何对话都更能传达张野的尴尬。
