# Prompt — 第四卷·第3章「车轮战」

## 角色

你是一个都市题材（urban）小说作家。现代都市背景，所有场景发生在真实城市空间——地下搏击馆休息室、沙发、昏暗灯光。超自然元素仅限「阿鼻系统」本身，其余世界观完全写实。不涉及修仙、穿越、异能战斗等非都市元素。

---

## 任务指示

创作第四卷第3章正文（在全卷中的角色：第二层·升级——肉体沦陷期→群体期过渡的核心肉戏章，张野第一次经历群体使用）。

### 字数与压缩策略

- **目标字数**：建议 5000 字（±10% 可接受，叙事完整性优先于字数精确）
- **压缩策略**：超字数时优先压缩低权重场景（场景1续rush过渡/场景6章末收束），推荐压缩至100字以内——保留1句环境切换 + 1句人物动作 + 1句情绪/氛围留白；不得因压缩删改红线内容或破坏情绪主线

### 本章驱动力

张野在rush辅助下第一次经历"被群体使用"——从"一对一被操"到"可以被共享使用"的彻底突破。药物让他不会拒绝，嘴巴和屁眼持续被填满、轮换，每根鸡巴的脸都记不清。

### 信息释放节奏

冲突紧迫型——群交车轮战的持续性和不间断填满感，读者被节奏推着走。从紧张（续rush）→被侵入（双洞填满）→适应（轮换开始）→飘然（续rush+持续填满）→麻木（章末瘫软），终点落在"飘飘然的麻木"。

### 叙事目标

1. 核心悬念：张野能在药物辅助下接受多少根鸡巴的轮换？
   → 本章结束时：**加深了**——张野瘫在沙发上浑身精液瞳孔涣散，读者想知道"他还能接受更多吗？"
2. 读者情绪：麻木、飘然——从紧张到被填满到飘然再到麻木，终点落在"飘飘然的麻木"
3. 爽点设计：群交/车轮战持续性填满的快感——嘴巴和屁眼从未同时空过，轮换节奏紧凑不间断
   - 铺垫于场景2（双洞首次同步填满的冲击）
   - 释放于场景4-5（持续性轮换填满，张野意识模糊不再分辨第几根）

### 写作方法论

1. **焦点锁定**：对每一场景，识别其唯一的"核心冲突"或"人物情绪"或"信息差"（三选一，不叠加）。将70%以上笔墨集中于此焦点对象，次要元素简笔带过。高权重场景细化到动作与感知层级，且必须同时应用"感官分层取舍"规则；低权重场景允许快速转场，但不得违反焦点锁定或感官分层取舍规则

2. **感官分层取舍**：以 2 种感官为主进行细化，氛围允许时增至 3 种（如听觉+触觉），其余一笔略写。禁止全维度均匀堆砌。声音例外：仅当声音线索被明确标记为"叙事线延伸"时方可作为叙事线延伸，不计入感官名额；其余声音描述仍需遵守2-3种感官上限。

3. **叙事节奏交替**：一段密集动作/对话 + 一段短留白 + 一句轻环境过渡循环。禁止连续长描写、连续对话、通篇等长段落。

4. **信息释放控制**：关键伏笔、人物信息拆分碎片化带出，不一次性完整交代。允许留白。

## 背景信息

### 前情上下文

上章结尾画面：张野瘫在沙发上，视线散焦。那三个男人的脸在他视野里叠在一起又分开，分不清谁是谁。王队的声音从远处传来，像隔了一堵墙。张野没有力气想接下来要发生什么——他甚至不想想了。脑子里最后一个清晰的念头是：好轻。然后就什么都没有了，只剩下沙发靠垫托着他的后脑勺，和一股淡淡的化学甜味残留在鼻腔里。防线没有了——不是被操垮的，是被化掉的。

读者情绪残留：飘浮（药物带来的失控轻飘感）

上章留下的缺口：张野在rush作用下身体完全放松、括约肌自主张开、恐惧消失——读者在等车轮战正式开始，看张野如何在药物辅助下被多人使用、嘴巴和屁眼被不同男人轮着操。

### 角色初始状态

**张野：**
  起点：瘫在沙发上，视线散焦，鼻腔里残留rush的化学甜味，括约肌在药物作用下自主张开。身体已准备好被使用，但意识处于"慢半拍"的飘浮状态，说话嘴唇不听使唤。
  → 经历：续rush（药效叠加，世界更模糊）→ 第一根入嘴 → 第二根入屁眼（双洞同步填满）→ 续rush → 嘴巴和屁眼轮换射精（根1射→根3接上，根2射→根4接上）→ 更粗插入（小赵的20cm）→ 持续轮换 → 最后一轮射脸
  → 终点：瘫在沙发上，浑身精液和汗，屁眼合不拢，精液从嘴角往下淌，瞳孔涣散，飘飘然的麻木。低头看自己身上——胸肌上全是精液和汗的混合物——不觉得恶心不觉得羞耻，就是觉得"就这样了"
  本章微习惯/小动作：嘴巴一直被塞着，只能发出含含糊糊的闷哼和呜咽；王队让翻个身就翻、让换姿势就换，不说话不反抗不做任何主动请求

**王队：**
  起点：坐在沙发上，刚确认张野rush药效上来了，朝对面几个男人扬了扬下巴"谁先来？"
  → 经历：指挥轮换（"含深点，别光用舌头舔""别急，慢慢来"）→ 续rush（"再吸一口，深吸，憋住""再续一口，别停"）→ 点评（"小李你今天持久不行啊，换小赵上""翻个身，换跪姿"）→ 最后一轮后走到张野面前看他的脸（"爽了吧？第一次就吃得这么好"）→ 递烟
  → 终点：递烟给张野，"歇会，一会儿还有事跟你说"
  本章必须完成的一件事：组织3-4人车轮战全程，但不参与操张野（留到第4章双龙压轴）

**3-4名派对参与男性（脸全部模糊）：**
  起点：站在沙发对面，等待王队发令
  → 经历：按轮换顺序分别操张野的嘴和屁眼——根1（第一根入嘴，约17cm）→ 根2（第一根入屁眼，约18cm）→ 根3（嘴里射后换入）→ 根4/小李（屁眼射后换入，较细）→ 小赵/根5（屁眼换入，约20cm极粗）→ 最后一轮射脸
  → 终点：穿好裤子在边上聊天抽烟，像什么都没发生过
  印象：所有脸必须模糊，区分只靠尺寸/身体特征（肤色偏黑、身材壮等），不给任何特写镜头或个人特征

## 案例

**【群交场景写作示例 · 感官分层取舍（触觉+听觉）】**

> 龟头顶到喉咙口的时候，张野的脖子往后仰了一下——没躲开，沙发靠垫顶着后脑勺，退不了。龟头的轮廓在舌面上刮过去，咸腥味在口腔里散开。身后那根也顶进来了，手指蘸唾沫草草扩了两下就直接往里推，撑开感从屁眼一路蔓延到小腹。喉咙里挤出一声闷哼。
>
> 王队的声音从边上飘过来："含深点，别光用舌头舔。"
>
> 张野想往下咽，喉咙被龟头顶着咽不下去。唾液顺着嘴角往下淌，滴在锁骨上。
>
> （演示：感官分层取舍·触觉+听觉——以口腔/肛门的触感为主，辅以声音层次【肉体撞击、脏话、闷哼】，省略视觉细节，符合张野在rush作用下视觉模糊的状态）

**【群交场景写作示例 · 焦点锁定（核心冲突）】**

> 不知道第几根了。嘴巴塞着一根，屁眼顶着一根，两根同时进出的节奏不一样——嘴里那根频率快一些，屁眼那根慢而深，每次顶到底都像要把小腹捅穿。张野已经不数了。脸全部模糊，尺寸也记不清了。只知道自己嘴巴一直有东西塞着、屁眼一直有东西顶着。
>
> 王队的声音："翻个身，换跪姿。"
>
> 张野翻过身，膝盖撑在沙发上。嘴里那根退出去又换了一根——温热的新龟头怼到嘴唇上。他张嘴含住。屁眼那根也退了出去，但不到五秒，更粗的一根顶了进来，括约肌被撑开到极限，张野身体往前一弓，喉咙里挤出含混的声音。
>
> （演示：焦点锁定·核心冲突——"持续性填满"是本场景的核心冲突，笔墨集中在"嘴和屁眼从未同时空过"这个持续性上，略写其他因素）

**【王队对话写作示例 · 对话标签替代】**

> 王队没动窝，坐在沙发另一头抽着烟。烟头的红光在昏暗的灯光里明灭。
> "小李你今天持久不行啊。"
> "换小赵上。"
>
> 张野听到他们在说话，但那些话像隔了一层水，跟自己好像没什么关系。嘴里塞着的东西在进出，带着淡淡的汗腥味。
>
> （演示：对话标签替代——删掉"王队说"，用抽烟动作+直接引语承接，用张野的听觉过滤来呈现对话的"隔了一层"感）

## 输入

### 场景原材料

**场景 1｜续rush·启动｜紧张基调｜权重：低**
  核心事件：王队拧开rush瓶盖，递到张野鼻子底下"再吸一口，深吸，憋住" → 张野照做 → 药效叠加，世界更模糊，肌肉彻底松弛，括约肌从"自主张开"变成"完全放空" → 王队拍了拍张野的脸，朝对面扬下巴"谁先来？" → 张野坐在沙发上没动，等着人过来
  情绪意义：读者跟着张野一起进入更深层的药物状态——紧张感在"等待被侵入"中累积
  信息差：张野知道"要被多人操"但不知道具体怎么轮换 ↦ 读者和张野一样不知道下一个被插哪里
  情绪拐点：紧张（等待）→ 更深层的飘浮（续rush后）
  → 下一场景的衔接：其中一人走过来，直接进入第一根入嘴
  本场景焦点：人物情绪（等待被侵入的紧张感）
  本场景权重：低（快速过渡，约100-200字）
  感官建议：触觉（第二次吸入时鼻腔的化学刺激）+ 听觉（王队的指令、拉链声）
  本场景出口情绪：紧张+飘然混合

**场景 2｜第一根入嘴·双洞填满｜被侵入基调｜权重：高**
  核心事件：一个男人走过来，裤子拉链拉开，鸡巴怼到张野嘴边 → 张野张嘴含住（没有犹豫，不是主动但也不躲）→ 脸是模糊的，只知道身材壮肤色偏黑，鸡巴约17cm中等粗度 → 龟头轮廓在舌面上划过，喉咙被顶入有压迫感，淡淡腥味在口腔散开 → 王队"含深点" → 喉咙被顶更深，闷哼 → 第二个人绕到身后，手指蘸唾沫草草扩了一下，顶进屁眼 → 张野被前后同时填满，身体一僵，嘴里那根含得更深，龟头顶到喉咙口差点干呕 → 第二根约18cm比第一根粗，撑开感明显 → 王队"别急，慢慢来，第一轮别操太猛" → 张野嘴里塞着东西说不出话，只能含含糊糊呜咽
  情绪意义：读者看到张野第一次被双洞同步填满——物理上最直接的"被群体使用"冲击
  信息差：张野从"知道会被多人操"变成"正在被两个人同时操"（信息缺口：不知道后面还有几根等着）↦ 读者知道后面还有人等着（比张野信息多）
  情绪拐点：紧张等待（场景1）→ 物理冲击（被前后同时填满的僵硬瞬间）
  → 下一场景的衔接：第一根在嘴里射了，进入嘴巴轮换阶段
  本场景焦点：核心冲突（双洞首次同步填满的物理冲击）
  本场景权重：高——细化到口腔触感（龟头轮廓、喉咙压迫感）和肛门触感（撑开感、深度），省略视觉（张野看不到也看不清）
  感官建议：触觉（口腔黏膜接触龟头的纹理/喉咙被顶入的压迫感/肛门括约肌被撑开）+ 听觉（王队的指令声、拉链声、肉体接近声）
  本场景出口情绪：被填满的冲击感

**场景 3｜嘴巴轮换·第一次射精｜适应基调｜权重：中**
  核心事件：第一根在嘴里射了——温热的一大股精液，张野被呛了一下，喉咙本能地吞咽 → 精液顺着喉管咽下去，咸腥味 → 龟头从嘴里退出去，但不到三秒又一根怼到嘴边 → 换了一个人，张野张嘴含住，嘴巴一直没空过 → 精液的味道在口腔里还没散尽已经含住新的龟头 → 王队抽了口烟"轮着来，嘴巴别空着" → 第二根节奏快了一些，进出间唾液往外淌
  情绪意义：读者看到"轮换"的节奏建立起来——射精不是结束，是轮换的信号，嘴巴不会被空出来
  信息差：张野开始模糊"第几根"计数 ↦ 读者和张野一起丧失精确计数能力
  情绪拐点：冲击（第一次被射在嘴里）→ 适应（第二根接上，节奏建立）
  → 下一场景的衔接：王队递rush续一口，进入下一阶段
  本场景焦点：核心冲突（嘴巴不空——轮换的持续性）
  本场景权重：中——重点在射精瞬间的感官（温热/咸腥/本能吞咽）和轮换的无缝衔接
  感官建议：味觉（精液的咸腥味）+ 触觉（龟头进出替换间的短暂空档感）
  本场景出口情绪：轮换节奏开始建立

**场景 4｜续rush·屁眼轮换·更粗插入｜飘然基调｜权重：高**
  核心事件：王队又递一次rush"再续一口，别停" → 张野趁嘴里那根退出去的间隙深吸了一口，脑袋更晕了——房间在转，灯光像隔了一层水 → 屁眼那根也射了，退出去 → 第三根（小赵的）接上 → 这根明显更粗——目测至少20cm，粗得像小孩手腕，顶入时括约肌被撑到极限 → 张野闷哼了一声，身体往前弓 → 嘴里又塞进来了，再次被同时填满 → 在rush作用下没有痛感，只有满满的撑开感和从肠道深处涌上来的酥麻 → 王队"小李你今天持久不行啊，换小赵上" → 张野听到他们的对话，但跟自己好像没什么关系
  情绪意义：本章强度峰值——在rush叠加下最粗一根插入屁眼，物理极限被逼近但没有痛感，只有酥麻的撑开感
  信息差：张野知道"换了更粗的人"但不知道对方叫什么/长什么样 ↦ 读者从王队的点评获得"小李/小赵"的区分，但和张野一样不知道他们的脸
  情绪拐点：适应的节奏 → 被更粗插入的冲击（但又因rush没有痛感，混着酥麻）
  → 下一场景的衔接：持续性轮换，前后面同时被不同的节奏操着
  本场景焦点：核心冲突（最粗一根的极限撑开感 + 药物作用下的酥麻替代痛感）
  本场景权重：高——细化肛门被撑开到极限的触感（碾压式胀满/肠道深处的酸胀酥麻）+ 张野闷哼和身体前弓的生理反应
  感官建议：触觉（括约肌极限撑开/肠道深处碾压式胀满）+ 听觉（王队的点评对话，但张野听着像隔层水）
  本场景出口情绪：撑开感+酥麻的混合飘然

**场景 5｜持续性填满·轮换循环｜麻木基调｜权重：高**
  核心事件：嘴巴和屁眼轮着换人——射了就换、换了再射、射完再换，但从来没有同时空过 → 张野已经不记得到底第几根了——嘴巴塞过几根、屁眼塞过几根，全混在一起了 → 脸全部模糊，尺寸也记不清了 → 持续性才是重点——没有被操和停下交替，是持续的、不间断的填满 → 王队坐在沙发上抽烟，偶尔说一句"翻个身，换跪姿"，张野就照做 → 前后面被不同的节奏操着，嘴巴含着一根，屁眼顶着一根，偶尔节奏对上了两根同时往深处顶
  情绪意义：读者感受到群交的"持续性"——不是一根接一根的计数，而是不间断的填满状态
  信息差：张野丧失了对"第几根""谁在操"的所有认知 ↦ 读者通过王队的出现/台词维持少量在场信息，但和张野一样不清楚轮换顺序了
  情绪拐点：从"还在分辨"到"不分辨了"——麻木状态的深化的瞬间
  → 下一场景的衔接：最后一根射在脸上，进入结尾
  本场景焦点：人物情绪（持续性填满带来的麻木/飘然）

### 场景写作指引

本章涉及3种场景类型（group-scene / dialogue / transition）：

**群交场景（group-scene）—— 本章主体：**
1. 感官分层取触觉+听觉优先——张野在rush作用下视觉模糊（灯光隔水、脸全部模糊），所以触觉（被插入的撑开感/口腔黏膜接触/精液喷出的温热）和听觉（王队的指令/肉体撞击声/自己的闷哼）是核心描写通道。视觉仅保留模糊光感和"脸叠在一起"的碎片印象
2. 节奏控制——轮换节奏要有变化：前半段（scene 2-3）每一根切换相对清晰可辨，中段（scene 4）最粗插入达到强度峰值，后段（scene 5）节奏模糊化、计数丧失，用"不知道第几根了"收束进入麻木状态

**对话场景（dialogue）：**
1. 王队的对话全部用动作替代说话标签（抽烟、弹烟灰、拧rush瓶盖、拍脸等），不写"王队说"
2. 张野的"对话"只有含含糊糊的闷哼和呜咽——不是完整句子，通过喉咙挤压出的声音表达

**过渡场景（transition）：**
1. 续rush场景（scene 1）压缩至100-150字：1句王队拧开瓶盖递过来 + 1句张野吸入 + 1句药效叠加的身体感觉变化 + 王队"谁先来"
2. 章末场景（scene 7）压缩至150-200字：1句张野瘫在沙发上的物理状态 + 1句低头看自己身上精液混合汗 + 1句"不觉得恶心不觉得羞耻" + 王队递烟 + "一会儿还有事跟你说"

**核心原则：这是GV式硬核肉文，不是文学小说。放弃精致感，追求刺激感和肉欲冲击。每一段都要让读者硬。**

#### 1. 受方台词（强制）

张野虽然说话小声闷闷的（公狗嗓），但在rush和操弄下必须有亵渎台词。嘴巴被塞着时说不清也努力含含糊糊地说：

- **最低要求**：在轮换间隙、被操到失神时，含含糊糊地漏出"操……操死我……""好胀……骚逼被撑满了……""想被填满……求填满……"
- **进阶**：在持续操弄下进入状态后，低声重复"操烂贱狗""骚逼让老公干烂""贱狗想要肉棒"
- **音色**：不是夹嗓子不是撒娇不是尖叫——是**公狗嗓低声求操**，闷闷的、哑哑的、含含糊糊但句子完整
- **分布**：第2~8场景之间均匀分布，至少4处不同的骚话

#### 2. 受方生理失控（强制）

每次被粗鸡巴操到极限时必写以下细节（至少选3种叠加使用）：

- **翻白眼**——眼球往上翻，只露眼白，持续数秒
- **舌头外伸**——嘴合不上，舌头耷拉出来，口水顺着舌头滴落
- **口水失禁**——唾液失控往外淌，下巴脖子上全是黏糊糊的口水
- **射精失控**——没被碰鸡巴但自己射了（操到前列腺高潮），一股一股地往外冒
- **失禁感**——膀胱被顶到产生尿意/渗尿，描写控制不住的感觉
- **全身抽搐**——被操到持续的小幅度痉挛，大腿/腰腹/脚趾都在抖

#### 3. 肉体变形（强制）

插入后必须写身体被改变的物理痕迹：

- **屁眼外翻**——被操开后穴口不能合拢，肠壁组织外翻成一圈粉红色的肉花，随着抽插翻进翻出
- **肚皮鼓包**——粗鸡巴顶入时，从小腹外面能隐约看到顶起的形状，手按着小腹能感觉到鸡巴的轮廓在掌心下顶过去
- **穴口合不拢流精**——鸡巴抽出去后精液直接从张开的屁眼里往外涌，像没关紧的水龙头

#### 4. 攻方台词（强化）

每个攻方的台词必须更粗暴、更羞辱性：

- 轮换时："张嘴""含深点""屁股撅好""别躲"
- 王队点评：要带烟嗓从容感——"小李你今天持久不行啊，换小赵上""第一次就吃得这么好"
- 其他人：可以有低沉的"操死你""骚逼夹这么紧"等
- **所有攻方台词必须带脏字**，语气要像在干一个物件而不是一个人

#### 5. 体液描写量化+粗俗（强化）

- 精液：写几股、射了多少、稠不稠、颜色偏黄还是白
- 拉丝：唾液拉丝、精液拉丝、混合液从嘴角/屁眼往下滴
- 灌满感：精液灌进肠道时写温度（热乎乎）、量（像小半杯温水灌进去）、溢出（顺着大腿流）

#### 6. 粗俗比喻清单（必须使用，每处至少用1个）

在描写鸡巴时，从以下选最合适的一个：驴屌、矿水瓶、小孩手腕、擀面杖、棒球棍、钢管、手臂、火腿肠、狗鸡巴

#### 7. 阴茎崇拜语态（贯穿）

张野对粗大肉棒的敬畏/贪恋/跪舔式描写必须贯穿全文——大鸡巴插入时不是"疼"而是"被填满的满足感"，粗鸡巴抽出去时有"空虚/不满足感"，换更粗的进来时有"终于满了"的臣服感
