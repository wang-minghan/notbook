# 题材设定指南

> 题材是 Phase 2 第一步——定了题材再聊世界观和角色。选完后写入 `settings/genre-setting.md`。
> 共 5 个字段：选定类型 / 满足类型 / 节奏规则 / 避免套路 / 类型禁忌。逐个聊，逐个写。

---

## 先选题材

这步不是写进文件的字段，是聊其他字段的前提——题材定了，才知道该聊哪些方面。

### 怎么聊：让作者说故事，你来判断

不要给作者甩分类列表。作者到这一步肯定有想写的东西，只是不懂术语。直接问：

> "你心里想写的大概是个什么故事？不用管类型分类，说说主角是谁、在什么世界、做什么事就好。"

**不管作者说了什么，你听完后做两件事：**

1. 从作者描述中提取**3 个关键信号**——时代背景、力量体系、核心冲突类型
2. 用下表判断最匹配的类型

| 关键信号 | 匹配类型 |
|---------|---------|
| 现代都市+日常情感+现实生活 | 都市日常 |
| 现代都市+金手指系统+脑洞奇想 | 都市脑洞 |
| 现代都市+种田建设+商战职场 | 都市种田 |
| 现代都市+全民修炼+灵气复苏 | 都市高武 |
| 现代都市+修仙/修真体系 | 都市修真 |
| 现代都市+战神归来+赘婿翻身 | 战神赘婿 |
| 古代背景+种田/朝堂/争霸 | 历史古代 |
| 古代背景+金手指+脑洞 | 历史脑洞 |
| 抗战时期+军事/谍战 | 抗战谍战 |
| 异世大陆+高武+废柴逆袭 | 传统玄幻 |
| 异世大陆+脑洞设定 | 玄幻脑洞 |
| 现代+探案/推理/刑侦 | 悬疑刑侦 |
| 现代+灵异/风水/恐怖 | 悬疑灵异 |
| 诡异复苏+规则怪谈+惊悚游戏 | 悬疑脑洞 |
| 末世/丧尸/星际/机甲/未来科技 | 科幻末世 |
| 西方世界+魔法/斗气/奇幻 | 西方奇幻 |
| 东方世界+修仙/修道/修真 | 东方仙侠 |
| 影视剧同人改编 | 男频衍生 |
| 游戏/动漫/二次元同人 | 动漫衍生 |
| 网游/竞技/体育/世界游戏化 | 游戏体育 |

**判断后展示给作者确认：**

> "听起来你写的是 **{类型}**——{一句理由}。对吗？"

✅ 正确：
> 作者："我想写一个普通上班族某天突然能看到别人头顶的死亡倒计时，他试着去救这些人，结果发现倒计时越改越短。"
> → 信号提取：现代都市（上班族）+ 金手指系统（死亡倒计时）+ 救人反噬
> → 判断：**都市脑洞（urban-brained）**
> → 确认："听起来你写的是都市脑洞——现代都市背景，主角获得特殊能力，但这个能力有代价。对吗？"

❌ 错误：
> 作者："我想写个故事。" → 继续问："主角是什么样的人？故事发生在什么地方？古代还是现代？"

**条件分支：**

| 作者说 | 怎么接 |
|--------|--------|
| "对，就是这个" | 进入字段 1，写入选定类型 |
| "不太对，更像 XX" | 按作者修正重新判断，或者直接手动选 |
| "还是不知道算什么类型" | "没关系，说说你故事里有没有超自然力量？古代还是现代？——这些定了就好分类了。" |
| "不想选类型" | "题材定了世界观才好聊——仙侠和末世的世界观完全不同。先定题材，世界观才聊得准。""暂不选择"也是有效答案，后续随时补选。 |

**判断标准：** 作者明确说了一个类型（或"暂不选择"）？模棱两可 → 继续聊。

**不需要读外部文件。** 下面每个字段都内联了多题材示例，直接参考。

## 字段 1：选定类型

**写什么：** 类型 id + 中文名，一行。

```
xianxia — 东方仙侠
```

id 用小写英文连字符，中文名用作者认可的表述。如果选融合类型，标注主类型 + 叠加要素：

```
urban-brained — 都市脑洞（叠加悬疑刑侦要素）
```

**判断标准：** 读完能知道这是什么题材、融合了什么要素。

---

## 字段 2：满足类型

**写什么：** 3-5 种读者想从这个题材获得的"满足感"，每种一句话说清核心吸引力。

### 底层模型：压抑→获得→释放→再压抑

所有题材的满足感共享同一个底层循环。写清楚不同题材只是给这个循环换了层皮：

```
压抑（被看不起/资源匮乏/身处底层）
  → 获得（奇遇/突破/关键线索/盟友）
    → 释放（打脸/装逼/反转/破案）
      → 再压抑（更强对手/新谜团/更高阶层对手）
        → 循环……
```

缺了"压抑"直接给"获得"，读者不觉得爽——没有对比就没有满足。缺了"再压抑"，读者觉得故事要结束了——没有继续追的动力。

**不同题材只是给这个循环换了层皮：**

| 题材 | 压抑阶段（缺什么） | 获得阶段（得到什么） | 释放阶段（怎么爽） | 再压抑（新压力） |
|------|------------------|-------------------|------------------|----------------|
| 东方仙侠 | 灵根差/资源少/被看不起 | 天材地宝/功法/突破 | 境界碾压/打脸 | 更高境界/宗门博弈 |
| 都市脑洞 | 普通社畜/没钱没权 | 金手指/能力觉醒 | 装逼/逆袭 | 更强能力者/组织 |
| 悬疑刑侦 | 线索中断/被骗/被误导 | 关键证据/突破口 | 推理破案/反转 | 更大阴谋/新谜团 |
| 历史古代 | 被政敌压制/财政吃紧 | 人才/资源/圣眷 | 朝堂翻盘/改革见效 | 新派系/外部威胁 |
| 都市日常 | 职场不顺/情感矛盾 | 转机/贵人/觉悟 | 化解矛盾/关系升温 | 新问题/新冲突 |

**所以聊的时候不问"有什么爽点"，问：**

> "你这个题材，读者最想看到主角摆脱什么、得到什么、做到什么？"
>
> ——摆脱什么 = 压抑阶段，得到什么 = 获得阶段，做到什么 = 释放阶段。

**判断标准：** 每个满足类型能说清对应的"压抑→获得→释放"三阶段？不能 → 缺了压抑读者不觉得爽。

## 字段 3：节奏规则

**写什么：** 三条规则——节奏倾向、张力周期、铺垫上限。

### 节奏倾向

这章是推进剧情还是铺垫气氛？一卷看下来是全程紧绷还是有张有弛？这决定不了具体每一章怎么写，但决定了一卷的整体节奏感。用一句话描述就行，不用精确计量。

| 倾向 | 含义 | 适合 |
|------|------|------|
| 行动偏多 | 冲突/战斗/对抗为主，节奏快 | 爽文、热血、快节奏升级 |
| 铺垫偏多 | 信息铺设/氛围/日常为主，节奏慢 | 悬疑、日常、慢热 |
| 张弛交替 | 紧张几章后给一章喘息 | 大部分题材通用 |
| 全程紧绷 | 几乎没有日常和缓冲 | 无限流、末世、强悬念 |

**怎么聊：**
> "你的故事节奏偏快还是偏慢？——几章得来一次小高潮？能接受连续几章铺垫不推进？"

### 张力周期

多少章一个小高潮、多少章一个大高潮。

**规律：**
- 爽文：3-5 章小高潮，10 章大高潮
- 悬疑：4-6 章小高潮，15 章大高潮
- 日常：5-8 章小高潮，20 章大高潮

### 铺垫上限

"最多连续几章没有进展"。超过这个上限读者会弃书。

**范围参考：** 快节奏 2 章，中等 3 章，慢热 4 章。

✅ 正确（仙侠快节奏）：
> - 节奏倾向：行动偏多
> - 张力周期：3-5 章一小高潮，10 章一大高潮
> - 铺垫上限：最多连续 2 章铺垫，第 3 章必须有进展

✅ 正确（悬疑慢热）：
> - 节奏倾向：铺垫偏多
> - 张力周期：4-6 章一小高潮，15 章一大高潮
> - 铺垫上限：最多连续 3 章铺垫，第 4 章必须有事件推进

❌ 错误：
> 节奏适中，该快则快该慢则慢——说了等于没说。

**判断标准：** agent 读了会不会"知道了但不知道怎么执行"？会 → 不够具体。

## 字段 4：避免套路

**写什么：** 这个题材读者看腻了的桥段。≥3 条。

套路不是不能写，而是不能无意识地写——写之前要知道"这是个套路"，然后决定是颠覆它、绕过它、还是承认它但写得更好。

**怎么聊：**
> "这个题材有什么桥段你不想写的？——就是那种一看开头就知道结尾的老套路。"

作者可能列出来的东西需要加工：

| 作者说了 | 写成 |
|---------|------|
| "不想写老套的拍卖会打脸" | 拍卖会必装逼打脸 |
| "不要跳崖奇遇" | 主角跳崖必得奇遇 |
| "不想主角上来就无敌" | 主角觉醒就是 S 级 |

✅ 正确：
> 主角跳崖必得奇遇——真跳崖了未必有奇遇，反而可能是埋伏。

❌ 错误：
> 不要写套路——太宽泛，什么算套路？

**各题材示例（直接参考）：**
- 仙侠：跳崖得奇遇、拍卖会装逼打脸、入门测试被看不起、境界高隐藏实力震惊全场
- 都市脑洞：主角觉醒就是 S 级、特管处全是废物、异能者穷得吃不起饭、女性角色只负责受伤被救
- 悬疑刑侦：第一目击者就是凶手、最不可能的凶手、天才侦探一眼看穿、最终boss总是熟人

**判断标准：** 每一条能不能写进一个 5 分钟视频的"网络小说套路吐槽"里？能 → 合格。

## 字段 5：类型禁忌

**写什么：** 这个题材绝对不能出现的东西——写了读者弃书、类型不成立。

禁忌不是可选的"尽量避开"，是写进去整本书变味的东西。

**常见禁忌分三类：**

| 类别 | 含义 | 例子 |
|------|------|------|
| 体系崩坏 | 自洽性被打破 | 战力忽高忽低、修炼一步登天、法宝随心所欲 |
| 题材错位 | 出现题材不该有的东西 | 仙侠世界讲人人平等、古代出现现代价值观 |
| 承诺失信 | 核心体验被破坏 | 推理小说靠超自然破案、末世文没有生存压力 |

**怎么聊：**
> "你觉得这个题材什么东西绝对不能写？——写了读者会骂'这写的什么玩意'那种。"

如果作者说不上来，顺着已选的题材举例询问：

> "仙侠文里战力崩坏是常见雷区——你介意吗？还有现代价值观乱入修真界呢？"

✅ 正确（仙侠）：
> - 禁止力量体系崩坏——战力不能为剧情方便忽高忽低
> - 禁止现代价值观乱入修真界

❌ 错误：
> 不要写得太离谱——什么算离谱？谁定义的？

**各题材示例（直接参考）：**

| 题材 | 常见禁忌 |
|------|---------|
| 东方仙侠 | 战力崩坏、修炼一步登天、现代价值观乱入、法宝随意设定、天劫走过场 |
| 都市脑洞 | 异能无代价、政府完全不知情、主角全知全能、体系逻辑不自洽 |
| 悬疑刑侦 | 超自然破案（非灵异题材）、凶手动机牵强、线索凭空出现、反派降智 |
| 历史古代 | 现代思想穿越、科技跳跃过快、历史人物纸片化、金手指过强无制约 |

**判断标准：** 作者能不能想象"违反这条禁忌写出来是什么样"？不能 → 太抽象。

## 写入文件

写入 `settings/genre-setting.md`，Markdown 格式。完整格式参考：

```markdown
# 题材设定

## 选定类型
xianxia — 东方仙侠

## 满足类型
- 境界突破（硬实力跃升）
- 秘境夺宝（资源争夺战）
- 打脸反杀（实力碾压后反转评价）
- 大道争锋（理念冲突 vs 实力对决）

## 节奏规则
- 节奏倾向：行动偏多
- 张力周期：3-5 章一个小高潮，10 章一个大高潮
- 铺垫上限：最多连续 2 章铺垫，第 3 章必须有进展

## 避免套路
- 主角跳崖必得奇遇
- 拍卖会必装逼打脸
- 入门测试必被看不起

## 类型禁忌
- 禁止力量体系崩坏
- 禁止修炼一步登天
- 禁止现代价值观乱入修真界
```

## 各字段怎么被下游使用

写进 `settings/genre-setting.md` 不是终点，每个字段都有对应的消费环节。Agent 写方向提案/章纲/提示词时必须读对应字段，否则写了白写：

| 字段 | 谁用 | 怎么用 |
|------|------|--------|
| 选定类型 | `prompt-crafter agent` 组装提示词 | 读 genre_profile，去 `knowledge/genre-example/` 取 `prompt_segment` 注入提示词 |
| 满足类型 | `chapter-planner agent` 方向提案 | 本章提案必须兑现至少 1 种满足类型，一段不兑现=读者白等一章 |
| 节奏倾向 | `chapter-planner agent` 方向提案 | 控制连续同类型章节数（比如"行动偏多"也不能连续 3 章全是高潮，读者绷不住） |
| 张力周期 | `updater-archive skill` 滑动窗口审视 | 超过张力周期上限还没小高潮 → 提醒作者"节奏可能有风险" |
| 铺垫上限 | `chapter-planner agent` 方向提案 | 连续铺垫章数接近上限时，提案必须推一个有进展的方向 |
| 避免套路 | `writer agent` 质量门禁 | 检查正文是否命中了 avoid_cliches 列表中的模式 |
| 类型禁忌 | `prompt-crafter agent` 组装提示词 | 禁忌写入提示词的"写作原则与禁忌"段，subagent 不得违反 |

**规则：** 上面任何一个字段为空或只有占位值，对应消费环节的 Agent 会在缺失约束的情况下自行判断。某章写出来味道不对，先检查"消费环节读了 genre-setting.md 没有"。

## 验收：从下游使用反推

验收不是看"有没有写"，是看"下游能不能用"。每个字段模拟一次消费场景来验证：

### 选定类型

| 验收方式 | 怎么验 |
|---------|--------|
| 有没有 | 有类型 id 或"暂无选择"，非空 |
| 该不该 | 是作者确认过的，不是 Agent 猜的 |
| 对不对 | 后续 `prompt-crafter agent` 要根据这个 id 去 `knowledge/genre-example/` 找对应文件。id 写错了就找不到。→ 验证：去 knowledge/genre-example/ 看一眼对应的文件存在吗？ |

### 满足类型

| 验收方式 | 怎么验 |
|---------|--------|
| 有没有 | ≥ 3 条 |
| 该不该 | 每条是具体的吸引力描述，不是"爽""热血"抽象标签 |
| 对不对 | **方向提案阶段**要用这 3-5 条来判断"本章兑现了哪条满足类型"。如果某条太宽泛（"热血"），Agent 做方向判断时会说不上来"这章到底满不满足这条"。→ 验证：挑一条，问"本章写什么内容算兑现了这条？"能答上来 → 合格 |

### 节奏规则

| 验收方式 | 怎么验 |
|---------|--------|
| 有没有 | 节奏倾向 + 张力周期 + 铺垫上限均有值 |
| 该不该 | 三条都是具体可执行的描述，不是"该快则快" |
| 对不对 | **滑动窗口审视**要用节奏倾向来判断实际节奏是否偏离预期。**方向提案**要用铺垫上限来避免连续铺垫超限。如果张力周期写了"3-5 章一小高潮"但实际 8 章没高潮，要能发现偏差。→ 验证：读上周期的 5 章章节标题和情绪状态，对照问"偏离了节奏倾向吗？" |

### 避免套路

| 验收方式 | 怎么验 |
|--------|--------|
| 有没有 | ≥ 3 条 |
| 该不该 | 每条是一个具体的桥段（"跳崖得奇遇"），不是宽泛概念（"不要套路"） |
| 对不对 | **正文质量门禁**要用这个列表扫描正文。如果一条写成"不要写太俗"，扫描脚本翻遍全文也判断不了"什么算俗"。→ 验证：把列表给一个人看，ta 能说出"这段正文就是命中这个套路了"吗？ |

### 类型禁忌

| 验收方式 | 怎么验 |
|--------|--------|
| 有没有 | ≥ 2 条 |
| 该不该 | 每条可想象违反它是啥样，不是"不要写太离谱" |
| 对不对 | **提示词组装**要把禁忌写入"写作原则与禁忌"段。如果一条写成"禁止体系崩坏"，subagent 读完后不知道"写什么算崩坏"。→ 验证：改写成 subagent 能判断的形式——"禁止战力忽高忽低：前文写金丹打不过筑基，本章不能写筑基秒金丹" |

### 参考案例
