# 写作风格

## role（叙事身份）

第三人称有限视角——主要跟随沈骁（受方），局部切换到林越或赵叔。通过"人物的感知"来描写世界，不许越到角色知识之外做全知叙述。

## core_principles（不可违背的写作信条）

1. **写实系BL**：不是甜宠，是带着痛感的沉沦。每一个心理转折都要有内在逻辑。
2. **"毒"的核心公式**：高价值身份 × 系统性摧毁 × 不可逆证据 = 每章必须推进这三个维度至少一个。
3. **身体先于理智投降**：堕落的核心是不由自主的生理反应——硬了、湿了、射了，是身体替脑子做决定。
4. **蚕食策略**：赵叔不急于一次吃干抹净，每次只推进一步底线。每一章都有新羞辱层级。
5. **六层感官沉浸**（GV六层体系）：每场肉戏必须包含——声音层/气味层/触感温度层/生理动态层/权力对话层/心理递进层。

## depiction_techniques（描写层次和手法）

### 句子节奏模板（从参考文提取）
```
日常叙事段 → 中等长度，客观描写
情绪升温段 → 句子变长、逗号连接，黏腻沉溺感
动作高潮段 → 短句/断裂句/单字句（"操！""啪！""啵"）
事后平复段 → 中长句，带喘息式的省略号
```

### 身体描写关键词链
- 肌肉总貌：精悍/结实/紧致/饱满/喷张/隆起/充血/油亮
- 腹肌：钢板/巧克力/整齐排列/抽搐/痉挛
- 胸肌：厚实/坚挺/两块鼓起的/被掐出红痕/挂着汗水
- 臀肌：铁蛋/挺翘/砖块/结实/被撞得发颤
- 大腿：修长/肌肉暴突/结实有力/布满汗珠
- 皮肤色差：麦色vs黝黑/浅肉色vs黑肌/黑白分明
- 鸡巴：粗长/上翘/淡褐/紫黑/青筋暴起/龟头鹅蛋大
- 穴口：粉嫩/紧致/皱褶/肉环/外翻/水润/一张一合

### 感官层次规则（每场至少含三种）
视觉（肌肉/色差/进出/体液）||触觉（紧/热/湿/烫/硬/滑）||味/嗅觉（汗/机油/烟草/精液/酒气）||听觉（啪啪/水声/喘息/闷哼）||体感（酸麻/胀/触电/痉挛/失禁感）

### 脏话系统（分层使用）
- 初期（嘴硬）："滚蛋""死变态""我操你妈"
- 中期（被动服从）："行了""别墨迹""快点"
- 中后期（身体背叛）："操……轻点……嗯……"
- 后期（彻底沦陷）："爸爸/主人操我""骚狗要"

### 叙事技巧
1. **对比**是核心修辞：肤色差/尺寸差/前后反差/公开vs私下/对男友vs对赵叔
2. **堕落阶梯**：意外失守→被迫接受→剃毛仪式→主动赴约→突破新底线→主动骑乘→多P→求操
3. **第三视角见证**：通过林越/队友的视角看沈骁的堕落，产生"原来如此"的震撼
