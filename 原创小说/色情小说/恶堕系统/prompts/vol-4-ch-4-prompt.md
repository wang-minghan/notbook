# Prompt — 第四卷·第4章「双龙」

## 角色

你是一个都市题材（urban）小说作家。现代都市背景，所有场景发生在真实城市空间——地下搏击馆休息室、昏暗灯光、旧沙发、茶几上有rush瓶和烟灰缸。超自然元素仅限「阿鼻系统」本身，其余世界观完全写实。不涉及修仙、穿越、异能战斗等非都市元素。

---

## 任务指示

创作第四卷第4章正文（在全卷中的角色：第3层·突破——物理极限突破阶段，全卷肉体强度最高的一章。张野第一次经历双龙——两根鸡巴同时塞进同一个屁眼，在rush辅助下撑开到不可能的程度。同时完成王队「第一次亲自操张野」的仪式感，以最高难度的双龙作为王队的主场秀。章末rush空瓶递接为第5章「依赖」做直接准备。）

### 字数与压缩策略

- **目标字数**：建议 5000 字（±10% 可接受，叙事完整性优先于字数精确）
- **压缩策略**：超字数时优先压缩低权重场景（场景1过渡/场景9章末收束），推荐压缩至100字以内——保留1句环境切换 + 1句人物动作 + 1句情绪/氛围留白；不得因压缩删改红线内容或破坏情绪主线

### 本章驱动力

张野在rush辅助下第一次经历双龙——从「被多人轮换使用」到「两根在同一屁眼里交替进出」的物理极限突破。王队的22cm棒球棍和兄弟老周约18cm的肉棒同时撑开同一个洞口，在rush作用下没有痛感只有极限撑开的胀满和空白。王队第一次亲自操张野，仪式感拉满——第一操就是这种顶级玩法。

### 信息释放节奏

冲击感型——双龙场景的物理极限感和视觉冲击力，读者被「两根进去」的画面推着走。从休息后的平静（过渡）→紧张（宣告双龙）→预备（rush+扩张放松）→突破（两根同时插入的极限撑开）→适应（交替操干中的模糊快感）→巅峰（双龙同时射精）→余韵（瘫软、合不拢、空瓶递接）。终点落在「被拆过的感觉」和空瓶的安静信号。

### 叙事目标

1. 核心悬念：张野的身体能在rush辅助下承受两根同时塞进屁眼的极限吗？
   → 本章结束时：**加深了**——他承受了，但读者想知道「他还能承受什么？依赖开始了吗？」
2. 读者情绪：从紧张到极限突破的冲击感再到高潮后的瘫软——终点落在「被拆过的瘫软」+「空瓶的安静信号」
3. 爽点设计：双龙的物理极限撑开视觉冲击 + 王队第一次亲自操张野的仪式感 + 双龙交替/同步进出的节奏快感
   - 铺垫于场景3-4（宣告+预备的紧张累积）
   - 释放于场景5（两根同时塞入的极限撑开瞬间——本章标志性画面）
   - 持续释放于场景6-8（交替操干+同步抽插+双龙同时射精）

### 写作方法论

1. **焦点锁定**：对每一场景，识别其唯一的「核心冲突」或「人物情绪」或「信息差」（三选一，不叠加）。将70%以上笔墨集中于此焦点对象，次要元素简笔带过。高权重场景细化到动作与感知层级，且必须同时应用「感官分层取舍」规则；低权重场景允许快速转场，但不得违反焦点锁定或感官分层取舍规则

2. **感官分层取舍**：以 2 种感官为主进行细化，氛围允许时增至 3 种（如听觉+触觉），其余一笔略写。禁止全维度均匀堆砌。
   - 本章感官优先顺序：**触觉 > 视觉 > 听觉**
   - 触觉：被两根同时撑开的极限拉伸感、棒球棍在肠道内的碾压感、交替进出时肠壁的摩擦感、精液灌入的温热感
   - 视觉：22cm棒球棍弹出画面、屁眼被撑到极限的夸张圆圈、张野翻白眼流口水的特写、小腹上龟头顶出的轮廓
   - 听觉：张野的闷哼和含含糊糊的「操死我」、精液和润滑剂混在一起咕叽咕叽的水声、王队的从容说话声
   - 声音例外：仅当声音线索被明确标记为「叙事线延伸」时方可作为叙事线延伸，不计入感官名额；其余声音描述仍需遵守2-3种感官上限。

3. **叙事节奏交替**：一段密集动作/对话 + 一段短留白 + 一句轻环境过渡循环。禁止连续长描写、连续对话、通篇等长段落。

4. **信息释放控制**：关键伏笔、人物信息拆分碎片化带出，不一次性完整交代。允许留白。章末空瓶的象征意义不直接解释——写在手里、写动作、不写心理。

## 背景信息

### 前情上下文

上章结尾画面：张野瘫在沙发上，浑身精液和汗，屁眼合不拢，王队递了根烟说「歇会，一会儿还有事跟你说」。烟夹在手指间没怎么抽，身体还泡在rush的余韵里，飘飘的。

读者情绪残留：飘飘然的麻木 + 对「一会儿还有事」的期待和紧张。

上章留下的缺口：王队说「一会儿还有事跟你说」——读者知道王队还没亲自操张野（上章他只是旁观指挥），知道「一会儿」的事一定跟王队亲自上有关。卷纲已预告双龙，读者在等王队的22cm棒球棍。

### 角色初始状态

**张野：**
  起点：瘫在沙发上，浑身精液和汗，屁眼合不拢，烟夹在手指间没怎么抽，身体泡在rush余韵里。休息了十来分钟，身体缓过来一点但精神还是飘的。
  → 经历：被告知要玩双龙 → 续rush+扩张准备 → 王队的22cm棒球棍先入屁眼（开道）→ 老周的18cm贴着棒球棍塞入（两根同在）→ 交替进出 → 同步抽插 → 双龙同时射精
  → 终点：整个人像被拆过的玩具，屁眼合不拢，精液混着润滑剂往外淌。翻着的白眼慢慢翻回来，瞳孔还有点散。王队递来rush空瓶，他揣进了裤兜。
  本章微习惯/小动作：王队问「敢不敢」他闷闷地「嗯」了一声——不是勇敢，是没有说不的习惯；被操到翻白眼流口水时含含糊糊喊「操死我」——公狗嗓低声喊的，不尖不叫不夹不妖；章末攥着空瓶，指腹在瓶口螺纹上摩挲了两下，揣进裤兜

**王队：**
  起点：坐在沙发上，刚说「歇会，一会儿还有事跟你说」
  → 经历：站起来活动肩膀，低头看张野「行了，休息够了，咱们来玩个大的」→ 拍张野后脑勺宣告双龙 → 拧开rush瓶续药 → 不急不缓地拉下拉链，22cm棒球棍弹出 → 先入开道（第一操就是双龙）→ 指挥老周「来，贴着进」→ 交替进出时边操边点评：「第一次玩双龙就吃得这么好，不愧是我带的人」→ 双龙射精后擦了擦棒球棍，在张野大腿上抹了抹：「第一次，不错」→ 递rush空瓶「下次来了自己带」
  → 终点：穿上裤子点了根烟，问「还能自己回去不？」
  本章必须完成的一件事：王队第一次亲自操张野——而且是双龙。仪式感拉满：不急不缓、像拆礼物、游刃有余。宣告主权：第一操就是最高难度。

**老周（功能性第二根）：**
  起点：站在边上，膀大腰圆，一脸横肉
  → 经历：蹲到张野身后做扩张（三根手指并拢捅进屁眼搅和）→ 扶着自己的鸡巴贴着王队的棒球棍挤入 → 和王队交替进出 → 配合王队同步抽插 → 和王队同时射精
  → 终点：拉上裤链，跟没事人一样
  印象：不给面部特写，不给个性特征。知道是王队的兄弟、膀大腰圆、脖子跟脑袋一样粗、鸡巴约18cm黑肤色中等偏粗。功能定位就是「第二根」。

## 案例

**【双龙插入写作示例 · 感官分层取舍（触觉+视觉）】**

> 王队跪到张野身后，一手扶着那根22cm的棒球棍——圆钝的龟头顶住张野的屁眼口。入口已经被rush和手指撑松了，但棒球棍的粗度还是让绷出一个夸张的圈。王队缓缓往里推——视觉上像一根棒球棍大小的物体在慢慢塞进一个不应该塞进这么粗东西的洞口。
>
> 龟头突破括约肌的瞬间，张野的腰猛地弓了一下，嘴里发出一声闷闷的「呃——」。但rush让括约肌像橡皮筋一样延展开来，没有撕裂感，只有被撑到极限的胀。22cm一寸一寸地没入肠道，龟头刮过肠壁时的粗粝触感从尾椎骨一路窜上来。

（演示：感官分层取舍·触觉+视觉——以肛门被撑开的视觉冲击和触觉感受为主，省略听觉细节，符合场景焦点在「插入的物理冲击」）

**【双龙操干写作示例 · 焦点锁定（核心冲突）】**

> 王队和老周开始配合——一根推进、一根退出，交替着操同一个屁眼。王队的棒球棍每次顶入都像一截实心木棍捅进肠道深处，龟头碾压前列腺时张野的身体就会抽一下；老周的鸡巴贴着棒球棍的侧面进出，摩擦着已经被撑到极致的肠壁内衬。
>
> 两根鸡巴在同一个肠道里交替运动——一根进去把空间撑满，另一根退出去留出缝隙让润滑剂和肠道黏液混在一起发出咕叽咕叽的水声。张野已经躺平了——背靠着沙发靠垫，两条腿被掰开到最大角度架在沙发扶手上，中间那个洞被两根男人的肉棒交替填满。

（演示：焦点锁定·核心冲突——「两根在同一屁眼里交替进出」是本场景的核心冲突，笔墨集中在两根的运动配合和物理感受上，缩写其他因素）

**【王队对话写作示例 · 对话标签替代】**

> 王队边操边说——语气不紧不慢，像在评点一笔划算的买卖：「第一次玩双龙就吃得这么好，不愧是我带的人。」
>
> 张野已经说不出话了。翻着白眼，口水流了半张脸。

（演示：对话标签替代——删掉「王队说」，用操干动作+直接引语承接，用张野的状态作为对话的衬托）

### 场景原材料

**场景 1｜过渡·接上一章｜平静基调｜权重：低**

  情绪意义：从第3章的麻木状态中缓过来——读者知道「大的」要来了，紧张感重新累积
  信息差：张野不知道「玩大的」具体是什么（但已经学会不问）↦ 读者从章标题「双龙」知道大概是什么玩法
  情绪拐点：平静（休息中）→ 紧张（被告知要玩大的）
  → 下一场景的衔接：王队宣告双龙
  本场景焦点：人物情绪（等待的紧张——知道要发生但不知道具体）

  情绪意义：读者亲耳听到王队宣告双龙——从「猜测」到「确认」的紧张落地
  信息差：张野知道要被双龙了，但不知道具体什么感受 ↦ 读者和张野一样知道要被双龙但不知道体感
  情绪拐点：紧张的期待 → 确认的紧张落地（「嗯」的那一刻）

  感官建议：听觉（王队的笑声从胸腔里震出来、声音粗犷从容）+ 视觉（王队拍后脑勺的随意动作、扬下巴指人的大哥做派）

**场景 3｜推进剧情·rush续+润滑准备｜恍惚基调｜权重：中**

  情绪意义：读者看到「准备阶段」的全部细节——rush续药、扩张、王队22cm棒球棍露出的画面，紧张感在准备动作中持续累积
  信息差：张野看到王队的22cm棒球棍（比之前任何一根都粗）↦ 读者和张野一起看到，一起紧张
  情绪拐点：紧张 → 预备的恍惚（rush吸入后身体开始变轻）混着对棒球棍尺寸的本能恐惧/敬畏
  → 下一场景的衔接：王队跪到张野身后，龟头顶住屁眼口
  本场景焦点：人物情绪（面对即将到来的双龙的紧张累积——药物带来的恍惚 vs 视觉冲击带来的本能反应）
  本场景权重：中——重点在三个动作的并行：rush续药（恍惚感）、老周扩张（手指撑开的预备感）、王队拉下拉链（22cm棒球棍弹出的视觉冲击——本章最标志性的画面之一）

  信息差：张野知道了「王队的棒球棍塞进去是什么感觉」↦ 读者和张野一样在体验——但读者知道这只是第一根，还有第二根要来
  情绪拐点：预备的紧张 → 第一根进入的突破（括约肌被撑到极限但没有撕裂感——rush起作用了）
  → 下一场景的衔接：王队保持插到底的姿势没动，对老周说来
  本场景焦点：核心冲突（22cm棒球棍塞入屁眼的物理极限冲击——一根不应该塞进去的粗度被一寸一寸地塞进去了）

**场景 5｜推进剧情·双龙插入·第二步：两根同在｜极限基调｜权重：高**
  核心事件：王队保持插到底的姿势没动，对老周说：「来，贴着进。」老周扶着自己的鸡巴对准张野的屁眼口——那里已经被王队的棒球棍撑到最大了，但两根一起塞还是超出了身体能理解的尺寸。老周的鸡巴约18cm、中等偏粗、肤色偏黑，贴着王队的棒球棍柱身往里挤——两根肉棒并排撑开同一个洞口。张野的括约肌被撑到了一个不可能的角度和宽度，肠壁被横向拉伸到极限，每一寸褶皱都被展开、填平。张野的身体猛地一僵——头往后仰，翻起了白眼，嘴巴不由自主地张开，口水顺着嘴角淌下来，拉成一条亮晶晶的线。但rush的药效让他没有发出惨叫——只有断断续续的粗喘和一声含含糊糊的闷哼：「呜……操……」不是疼，是一种被撑到极限后大脑主动切断分辨能力的空白。

  情绪拐点：第一根的胀满 → 第二根挤入时的极限撑开到大脑空白（翻白眼、流口水、失去分辨能力）

  本场景焦点：核心冲突（两根同时在一个屁眼里的极限物理撑开——张野翻白眼流口水的瞬间是本章最标志性的图像）

**场景 6｜推进剧情·双龙操干·交替进出｜节奏基调｜权重：高**

  情绪拐点：极限撑开的空白感 → 操干节奏中的模糊快感（翻着白眼含含糊糊喊操死我——进入了被操的状态）

**场景 7｜推进剧情·王队的仪式感·主权宣示｜宣告基调｜权重：中**

  情绪拐点：模糊快感的适应 → 王队宣告主权（加速操干+同步节奏，让张野的认知更深地沉入被操状态）

  本场景焦点：人物情绪（王队的仪式感——从容展示、不紧不慢的点评、宣告主权）

  信息差：张野知道双龙结束了，身体有被灌满和掏空的双重感受 ↦ 读者看到张野的物理状态（合不拢的屁眼、往外涌的精液）和精神状态（瞳孔散、嘴巴张着说不出话）

  → 下一场景的衔接：安静下来，王队递水递空瓶
  本场景焦点：核心冲突（双龙射精的释放感——两根同时在肠道内射精的热量+量+混合溢出感+退出去后屁眼合不拢的物理结果）

**场景 9｜过渡·章末/为第5章铺垫｜余韵基调｜权重：低**
  核心事件：休息室里安静了几秒，只有张野粗重的喘气声和老周拉裤链的声音。王队递了一瓶水给张野——「喝点水，别脱水了。」张野接过水，手还在微微发抖，拧了两下才拧开盖子。他低头看了一眼自己——胸肌上被溅了几滴精液和汗混在一起，腹肌上全是汗和水光，大腿根湿了一片，那个还在淌精液的屁眼敞开着。沉默中他脑子里闪过一个念头：如果没有rush，这一下他绝对撑不下来——那个念头很短，像水面上冒了个泡就没了。王队从茶几上拿起一个rush的空瓶，随手扔到张野怀里：「下次来了自己带，我这瓶贵。」张野攥着那个空瓶，没说话。指腹在瓶口的螺纹上摩挲了两下——塑料的触感，轻飘飘的，但他没松手。王队穿上裤子点了根烟：「还能自己回去不？要不要叫个车？」张野摇了摇头。他把那个rush空瓶揣进了裤兜里。

## 不可违反规则

### 优先级（从高到低）

1. **约束红线** — 情节红线、角色禁区、边界禁止
2. **字数** — 5000字目标（超限启用压缩策略）
3. **T1词 / 认知动词 / 感官** — 修饰阈值、认知动词节制、感官分层
4. **叙事规则**（高于同层·写作规范其他规则）
5. **写作规范同级其他规则**

### 红线约束

**冲突阶梯**：第3层·突破（物理极限突破+首次药物性行为）——本章是整卷物理强度最大的场景

**情节红线（不可删减）：**
1. **双龙必须写出极限撑开的物理感受**——肠壁被横向拉伸的压迫感、两根肉棒互相挤压的触感、括约肌被撑到极限的灼热绷紧、张野翻白眼流口水的标志性画面
2. **22cm棒球棍比喻必须写足**——「像一截棒球棍」不是修饰，是要写出的视觉印象：极粗、龟头圆钝、青筋凸起、整根硬挺挺地杵着。同时搭配其他粗俗具象比喻：小孩手腕、拳头、擀面杖
3. **张野翻白眼/流口水/含含糊糊喊「操死我」**——本章标志性画面，必须写足写出画面感
4. **rush覆盖全程**——从预备阶段的「多吸点」到双龙过程中的认知模糊到章末余韵，rush始终在场
5. **章末王队递rush空瓶「下次来了自己带」——张野把空瓶揣进裤兜**——第5章「依赖」的直接钩子，不可省略

**边界禁止（不涉及）：**
1. 张野的说话方式必须保持公狗嗓——翻着白眼流着口水，喊出来的「操死我」也必须是从闷闷的公狗嗓里碾出来的，不尖不叫不夹不娘
2. 不要出现女性化描写形容张野——即使是翻白眼流口水的失控状态，也是男性身体的失控：肌肉痉挛、粗喘、闷哼
3. 不要出现上帝视角解构——不写「这标志着依赖的开始」「这证明了」「张野心想」
4. 老周的鸡巴不要详细描写——知道尺寸（约18cm）和颜色（肤色偏黑）就够了，焦点始终在王队的22cm棒球棍和双龙的整体感受上

**角色禁区：**
张野不会主动说「再来一次」或「还要」（他还停留在「被操」的接受阶段，主动留到后续章节）
张野不会主动脱衣服或换姿势（王队指挥）
张野章末不会主动说「下次再来」（他只是把空瓶揣进裤兜——下意识的动作）

### 网文风格基线（任何章节适用）

1. 认知动词节制使用（≤2次/章）
2. 因果关系自然呈现
3. 叙事自然有温度——禁止极端白描
4. 全程线性叙事

### 风格建议（可偏离）

- **视角**：第三人称有限，紧贴张野。不写他人内心。禁止全知解释。
- **描写**：用动作感知推动叙事，感官服务于氛围。本章以**触觉（被撑开/拉伸/填满的体感）和视觉（棒球棍/翻白眼/撑圆/精液倒灌的画面）** 为主要感官。
- **节奏**：段落自然起伏，句式长短交替。以上若服务情绪或剧情可适度偏离。
- **张野的说话方式**：本章只有几句含含糊糊的「操死我」——公狗嗓低声、闷闷的、不尖不叫不夹不娘。其余时间只有闷哼和粗喘。
- **量化写实**：每次鸡巴出场需带尺寸（22cm/18cm），搭配粗俗比喻（棒球棍/小孩手腕/拳头/擀面杖）。
- **中式粗口贯穿始终**：王队的台词必须带脏字（「操」「他妈的」等），但语气从容不迫、游刃有余。
- **受方生理失控细节必写**：翻白眼、流口水、身体痉挛、大腿内侧抖动、腹肌抽搐、屁眼合不拢、精液往外淌。
- **王队的仪式感**：第一次操张野——不急不缓地拉裤链、像拆礼物、边操边点评的从容语气、宣告主权的游刃有余

### 场景写作指引

本章涉及3种场景类型（group-scene / dialogue / transition）：

**双龙场景（group-scene）—— 本章主体：**
1. 感官分层取**触觉+视觉优先**——张野在rush作用下认知模糊，但视觉（棒球棍的轮廓/屁眼被撑圆的画面/翻白眼的表情/精液倒灌）和触觉（被极限撑开的拉伸感/两根交替进出的摩擦感/精液灌入的温热感）是核心描写通道。听觉辅助（张野的闷哼和含含糊糊的「操死我」/精液和水声）
2. 节奏控制——双龙操作要有递进：王队先入（场景4）→两根同在（场景5，强度峰值）→交替进出（场景6）→同步加速（场景7）→同时射精（场景8，释放）。从慢到快，从静止到运动，从交替到同步

**对话场景（dialogue）：**
1. 王队的对话全部用动作替代说话标签（活动肩膀、拍后脑勺、拉下拉链、边操边说、穿裤子等），不写「王队说」
2. 张野的「对话」只有含含糊糊的「操……操死我……」——不是完整清晰的说话，是翻着白眼流着口水从喉咙里碾出来的闷喊

**过渡场景（transition）：**
1. 场景1（接上一章）压缩至100-150字：1句张野瘫在沙发上的状态 + 1句王队宣布 + 1句化学味飘过来
2. 场景9（章末收束）压缩至150-200字：1句安静 + 1句王队递水 + 1句张野接水看自己身上 + 1句空瓶递接 + 1句揣进裤兜

### 质感要求

**无用细节（不服务主线的细碎生活化细节）：**
1. 茶几上的rush瓶堆——几个空瓶滚在烟灰缸旁边，其中一个瓶口凝着化学味的水汽
2. 沙发皮面上被身体压出的声响——张野被操到往前滑的时候皮肤蹭过皮面的摩擦声
3. 昏暗灯光下汗珠在皮肤上的反光——张野胸肌上的汗珠在昏暗灯光下亮晶晶的
4. 王队擦棒球棍的动作——在张野大腿上随手一擦了事的那份随意

**对话节奏：允许半截话、语气词、无意义停顿**
- 王队说话简短粗暴但不急促——从容的大哥式语气：名词+动词直接出，偶尔带笑
- 张野：含含糊糊的闷哼和「操死我」的重复——单个句子被打碎，每一个字都是碾出来的

**段落精度分层（按权重分配笔墨）：**
- **高权重场景（场景4、5、6、8）**：细节饱满，至少3处动作/感知层级细节用于核心冲突/情绪展开。场景4细化22cm棒球棍入屁眼的全程触感和视觉；场景5细化两根同在的极限撑开和张野翻白眼流口水的标志性画面；场景6细化交替进出的配合和张野含含糊糊喊操死我的声音；场景8细化双龙同时射精的释放和张野被拆过的状态
- **中权重场景（场景2、3、7）**：正常叙事节奏，宣告/预备/仪式感的推进感为主
- **低权重场景（场景1、9）**：快速转场

**额外约束（本章特有）：**
1. **双龙物理极限必须写到**——两根同时在一个屁眼里不能只有「进去了」的结论，要写出每一根怎么进、两根并排时的撑开感、交替进出时肠壁的摩擦感、同步进出时空间的极致利用
2. **王队第一次操张野的仪式感**——不是急色的，是游刃有余的「展示」。不急不缓地拉裤链、像拆礼物、边操边点评、完事后擦棒球棍的随意和「第一次，不错」的从容
3. **张野的说话方式不变**——翻着白眼流着口水，喊出来的「操死我」也必须是公狗嗓低声闷闷的，不尖不叫不夹不娘
4. **rush覆盖全程**——从场景3的续rush到章末，药物始终在场，张野的认知模糊是全程底色，但偶尔清醒被操醒的瞬间可以写
5. **章末空瓶不要过度解读**——不写「这意味着他开始依赖了」，只写动作：攥着、摩挲螺纹、揣进裤兜

### ★ GV风格追加指令（强制覆盖，优先级高于上方所有同级规则）

**核心原则：这是GV式硬核肉文，不是文学小说。放弃精致感，追求刺激感和肉欲冲击。每一段都要让读者硬。本章是双龙——全卷物理强度最高的一章，GV感必须拉到最满。**

---

#### 1. 受方骚话（强制）

张野虽然说话小声闷闷的（公狗嗓），但在双龙操弄到失神时必须说亵渎台词：

- **最低要求**：在场景5-8中，含含糊糊地喊出「操……操死我……操死我……」（至少3次，分布在不同场景）。不是夹嗓子不是撒娇不是尖叫——是公狗嗓低声求操，闷闷的、哑哑的、含含糊糊但每个字都是从喉咙里碾出来的
- **进阶**：在操干节奏中掺入「好胀……」「两根……塞满了……」等短句
- **音色**：翻着白眼流着口水，说话声音是被操出来的——闷闷的低声，不是清晰的句子，是被顶碎了的字
- **分布**：场景5（两根刚进去时含含糊糊的闷哼「呜……操……」）、场景6（交替操干中喊「操死我」）、场景8（高潮射精时再喊一次「操死我」）

#### 2. 受方生理失控（强制）

每次被撑到极限时必写以下细节（至少选3种叠加使用）：

- **翻白眼**——眼球往上翻，只露眼白，持续数秒（场景5标志性画面，场景6-8持续或间歇出现）
- **舌头外伸**——嘴合不上，舌头耷拉出来，口水顺着舌头滴落（场景5-8）
- **口水失禁**——唾液失控往外淌，下巴脖子上全是黏糊糊的口水，场景5第一次淌下来「拉成一条亮晶晶的线」，场景6「口水流了半张脸」，场景8「口水拉成一条线垂到胸口」
- **全身痉挛**——腹肌一抽一抽的，大腿内侧肌肉不受控制地抖动（场景8射精后必写）
- **认知空白**——大脑主动切断分辨能力，那一刻没有想法没有感受只有空白（场景5的极限撑开瞬间）

#### 3. 肉体变形（强制）

插入后必须写身体被改变的物理痕迹：

- **屁眼极限撑开**——被王队22cm棒球棍塞入时「入口绷出一个夸张的圈」；被两根同时撑开时「括约肌被撑到了一个不可能的角度和宽度」「肠壁被横向拉伸到极限，每一寸褶皱都被展开、填平」；退出去后「合不拢了——一个撑到极限的圆形洞口，精液混着润滑剂和肠道黏液往外涌」
- **肚皮鼓包**——王队22cm操到底时「小腹能隐约看到龟头顶出的轮廓」（场景4和场景7必写）
- **穴口合不拢流精**——两根抽出去后精液直接往外涌（场景8必写）

#### 4. 攻方脏话（强化）

王队的台词必须更粗暴、更羞辱性——但注意语气是**从容游刃有余**的，不是急促粗暴的：

- 宣告时：「双龙听说过吧？」「两根一起塞你屁眼里，敢不敢？」——声音从胸腔里震出来的笑，带着大哥式的随意
- 指挥时：「来，贴着进」——简短的命令
- 操干中点评：「第一次玩双龙就吃得这么好，不愧是我带的人」——不紧不慢，像在评点一笔划算的买卖
- 射精后：「第一次，不错」——随手在张野大腿上擦了擦棒球棍，那份随意就是仪式感
- 章末：「下次来了自己带，我这瓶贵」——扔空瓶的动作带点大哥式的抠门，但其实是默认了张野还会再来
- **所有攻方台词必须带脏字**，语气要像在操一个物件而不是一个人，但王队的脏字是嵌在从容语气里的，不是骂街式的

#### 5. 精液量化+粗俗（强化）

- 精液：写几股（一大股/第二股）、量（量很大/很烫/像小半杯温水灌进去）、质地（浓稠）、颜色（偏黄白色）
- 灌满感：精液灌进肠道时写温度（热乎乎的）、量（在肠壁内扩散填满褶皱）、溢出（混在一起从屁眼口倒灌/顺着大腿根往下淌/滴在沙发皮面上）
- 混合液：精液混着润滑剂和肠道黏液往外涌的画面，两根鸡巴上挂着亮晶晶的混合物

#### 6. 粗俗比喻清单（必须使用，每处至少用1个）

在描写鸡巴时，从以下选最合适的一个（本章必须写到的比喻）：

- 王队的22cm鸡巴 = **棒球棍**（核心比喻，必须使用）——极粗、龟头圆钝、整根像一截棒球棍，青筋凸起在昏暗灯光下泛油光
- 其他可用：小孩手腕、拳头、擀面杖、驴屌
- 至少3处不同的鸡巴比喻（王队的棒球棍是贯穿比喻，老周的鸡巴可以写「和他手腕差不多粗」「黑了不止一个色号」等）

#### 7. 阴茎崇拜语态（贯穿）

张野对粗大鸡巴的敬畏/本能反应必须贯穿：

- 场景3：王队22cm弹出来时「张野的视线扫过那根东西，喉咙动了一下——吞咽的动作，不受控制的」
- 场景4：王队插入时「视觉上像一根棒球棍大小的物体在慢慢塞进一个不应该塞进这么粗东西的洞口」——敬畏感
- 场景5：两根同时塞入时大脑的极限空白感——超出了身体能理解的尺寸
- 大鸡巴塞进去时不是「疼」而是「被撑满的胀」，退出去时空了但不是「解放了」而是「空了」
