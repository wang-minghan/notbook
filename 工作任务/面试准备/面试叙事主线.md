# 面试叙事主线

这页是你的**面试叙事 overview**：先把整条故事线讲顺，再把细节问答交给 [[面试问答整理]]。

你的面试表达不应该是“我做过很多东西”，而应该是一个统一叙事：

> 用领域知识嵌入模型推理。

## 主线结构

### 第一段：CAE 仿真降阶
- 问题：物理仿真太慢
- 方法：POD + 代理模型
- 结果：把有限元级别的问题加速到可服务化

### 第二段：大模型推理增强
- 问题：LLM 幻觉和业务规则不一致
- 方法：RAG + 本体约束 + Agent 编排 + LoRA
- 结果：让推理可信、可控、可落地

## 一句话总结

你不是纯做模型的人，你是做**领域规则进入推理链路**的人。
