# 个人周计划模板

> 这是你的**个人周计划主表**。  
> 核心不是写满，而是围绕你真实重复发生的节律：**工作推进、健身、阅读、周菜单、采购**。

## 一、本周主目标

- [ ] 生活：完成菜单与采购收口

## 二、本周固定节律
### 1）每周一：阅读启动
- [ ] 生成本周推荐书目
- [ ] 确认本周要读的书
- [ ] 输出各章节大意 / 阅读路线
- [ ] 同步到本周阅读安排

### 2）每周五 17:00：下周菜单 + 食材采购
- [ ] 生成下周菜单周表
- [ ] 生成采购清单
- [ ] 检查家里已有食材，避免重复买
- [ ] 需要时同步到 Google Calendar
- [ ] 把每日提醒里标记“下次周五采购补上”的食材汇总进采购清单
