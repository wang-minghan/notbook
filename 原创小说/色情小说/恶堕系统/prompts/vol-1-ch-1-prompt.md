# Prompt — 第一卷·第1章「遇见」

## 角色

你是一个都市题材（urban）小说作家。现代都市背景，所有场景发生在真实城市空间——大学篮球场、傍晚的光线、宿舍床铺。本章不涉及任何超自然元素（阿鼻系统尚未出现），是完全写实的都市青春叙事。不涉及修仙、穿越、异能战斗等非都市元素。

---

## 任务指示

创作第一卷·第1章正文（在全卷中的角色：第1层·入门——初恋阶段的起点。本章是张野与陆辞的初遇，通过篮球场上的物理碰撞到赛后的暧昧互动，建立两人关系的开端。张野对陆辞的心动完全不自知——他不知道自己是 Gay，只觉得「心跳有点快」「脸有点热」「说不清为什么」。无肉戏。）

### 字数与压缩策略

- **目标字数**：2800-3500 字（±10% 可接受，情感完整性优先于字数精确）
- **压缩策略**：超字数时优先压缩场景1（环境描写）和场景7-8（宿舍收束），推荐压缩至80-120字/段——保留1句环境/状态 + 1句动作 + 1句情绪留白；不得因压缩删改暧昧互动核心场景（3-6）

### 本章驱动力

篮球场上张野一记盖帽拍掉了陆辞的上篮，两人从对手到相识。陆辞主动递水、拍肩说「打得好」、加微信——每一动作都让张野耳朵发红、心跳加速。张野回到宿舍躺在床上翻来覆去地看陆辞微信头像，耳朵还是红的，他自己也说不清为什么。这是初恋的起点——他不知道这就是心动。

### 信息释放节奏

细腻情感型——从球场日常（平静）→ 盖帽碰撞（意外）→ 比赛配合（默契建立）→ 赛后互动（暧昧升温）→ 宿舍余韵（情绪沉淀）。终点落在张野独自看头像的安静时刻——耳朵还是红的，什么都没想明白。

### 叙事目标

1. **核心悬念**：张野为什么心跳加速、耳朵发红？
   → 本章结束时：**加深了**——他经历了全部互动，仍然不明白这种感觉是什么。读者比他更早意识到「他心动了」。
2. **读者情绪**：从专注看球的平静 → 盖帽碰撞的意外 → 赛后互动的甜和心跳 → 宿舍余韵的暧昧。终点落在「甜而不自知」的感受上。
3. **爽点设计**：陆辞的每一个主动动作（递水、拍肩、加微信）都是甜蜜的推进，张野的害羞反应（耳朵红、语速慢、不敢对视）是读者嗑糖的来源。
   - 铺垫于场景1-2（打球日常，建立两人球场上的物理碰撞）
   - 释放于场景3-5（赛后互动——递水时耳朵红、拍肩时身体僵了一下、加微信时低头不敢看对方）
   - 持续余韵于场景6-8（宿舍里翻来覆去地看头像，耳朵还是红的）

### 写作方法论

1. **焦点锁定**：对每一场景，识别其唯一的「核心情绪」或「人物反应」或「信息差」（三选一，不叠加）。将70%以上笔墨集中于此焦点对象，次要元素简笔带过。高权重场景细化到动作与感官层级；低权重场景允许快速转场。

2. **感官分层取舍**：以 2 种感官为主进行细化，氛围允许时增至 3 种，其余一笔略写。禁止全维度均匀堆砌。
   - 本章感官优先顺序：**视觉 > 听觉 > 触觉**
   - 视觉：傍晚篮球场的光线、张野盖帽时的爆发力、陆辞的笑容、张野耳朵红起来的样子、手机屏幕上陆辞的头像
   - 听觉：篮球拍地的声响、球鞋摩擦地面的声音、陆辞的笑声和说话声、宿舍走廊里的脚步声和关门声
   - 触觉：手掌拍掉球时的震动感、被拍肩膀时身体的僵硬感、握住手机时手指的微颤、躺在床上时心跳的咚咚声
   - 声音例外：仅当声音线索被明确标记为「叙事线延伸」时方可作为叙事线延伸，不计入感官名额；其余声音描述仍需遵守2-3种感官上限。

3. **叙事节奏交替**：一段密集动作/对话 + 一段短留白 + 一句轻环境过渡循环。禁止连续长描写、连续对话、通篇等长段落。

4. **信息释放控制**：张野对陆辞的心动不直接解释——只写生理反应（耳朵红、心跳快、手心出汗、翻来覆去），不写心理分析（「他意识到自己喜欢陆辞」）。留白让读者自己去感受。

## 背景信息

### 前情上下文

无前情。这是小说第一章，张野和陆辞互不相识。

读者不知道张野是隐藏 Gay ——读者和张野一样，只知道他是个人高马大的体育生，性格内向腼腆。

### 角色初始状态

**张野：**
  起点：站在篮球场上准备比赛，注意力全在球上。运动中的放松状态，不说话，专注。
  → 经历：比赛中和陆辞对位 → 盖帽把陆辞的球拍掉 → 陆辞笑着搭话 → 比赛配合 → 赛后陆辞递水拍肩 → 陆辞加微信 → 回到宿舍
  → 终点：躺在床上，翻来覆去地看手机里陆辞的微信头像，耳朵还是红的。什么都没想明白，就是那个状态。
  本章微习惯/小动作：被拍肩时耳朵红了（生理反应，他自己控制不了）；掏手机加微信时手有点僵，不太自然，低头不看对方眼睛，语速比平时还慢；回到宿舍后，躺在床上刷了几下手机又锁屏，锁了又打开

**陆辞：**
  起点：控卫，运球突破，注意力在比赛上。
  → 经历：上篮被张野盖帽 → 愣了一秒，然后笑出声来 → 比赛中观察到张野打球的风格（干净利落、不废话） → 赛后主动递水、拍肩膀夸打得好 → 主动提出加微信
  → 终点：加了张野微信，聊了几句，回宿舍了。
  印象：阳光、主动、说话随意、笑的时候眉眼舒展。对张野有兴趣——但本章只写到他主动的程度（递水+拍肩+加微信+多聊了几句），不写他内心的想法（不切换视角）

**场景 1｜过渡·环境｜篮球场傍晚｜权重：低**
  核心事件：傍晚光线偏暗，球场上人群嘈杂。张野站在场上准备比赛，注意力全在球上。氛围：日常、运动中的放松状态。不做过多环境堆砌——用一两句光线和声音带出球场氛围。
  情绪意义：正文的起点——从日常平静开始，建立场景真实感
  信息差：无——读者和张野看到的是同样的球场
  情绪拐点：平静 → 专注（比赛开始）
  → 下一场景的衔接：比赛开始，张野队和陆辞队对打
  本场景焦点：人物情绪（运动前的专注和平静——日常状态）
  本场景权重：低（≈80-100字，快速带入场景）
  感官建议：视觉（傍晚偏暗的光线、球场地板的反光）+ 听觉（拍球声、人群嘈杂声、球鞋摩擦声）
  本场景出口情绪：平静的日常感

**场景 2｜推进剧情·比赛·盖帽｜意外基调｜权重：高**
  核心事件：张野队与陆辞队对打。陆辞是控卫，运球突破上篮，张野是前锋在篮下防守。陆辞起跳上篮的瞬间——张野一记盖帽把球拍掉了。陆辞愣了一秒，随即笑出声：「哥们手劲挺大。」语气随意、阳光、不介意被盖帽。张野站在那儿——没有因为盖帽得意，也没有辩解，就是不太习惯被搭话，低头「嗯」了一声。
  情绪意义：两人的第一次物理碰撞和言语接触——盖帽的爆发力 + 陆辞阳光的反应 + 张野不擅社交的沉默
  信息差：陆辞知道张野手劲大、打球干净利落 ↦ 张野不知道陆辞对他有兴趣
  情绪拐点：专注比赛 → 意外（盖帽后愣了一下）→ 轻微的尴尬/不习惯被搭话（张野的反应）
  → 下一场景的衔接：比赛继续，两人因为打球有几次接触
  本场景焦点：核心冲突（盖帽的物理碰撞——张野的本能反应 vs 陆辞阳光随意的化解）
  本场景权重：高（≈350-400字）——盖帽瞬间要写出爆发力和视觉冲击力，陆辞的反应要写出阳光随意的反差
  感官建议：听觉（拍球的响声、盖帽的闷响、陆辞的笑声）+ 视觉（盖帽瞬间的手掌球体接触、陆辞愣了一秒后笑开的表情变化）
  本场景出口情绪：意外的碰撞——平静被打破，两人第一次对上话

**场景 3｜过渡·比赛继续·默契｜推进基调｜权重：中**
  核心事件：比赛继续。陆辞控卫、张野前锋——两人不在同队但几次对位/配合中打出了默契。陆辞运球突破时张野的防守卡位，陆辞分球时张野的协防补位——有几次眼神交流和下意识的配合。张野不擅长说话，但球场上有眼神交流和默契配合——那是他唯一「不需要说话就能交流」的方式。
  情绪意义：两人通过比赛建立「不需要语言的默契」——身体的熟悉感先于言语的熟悉感
  信息差：陆辞通过比赛确认张野球技好、防守干净 ↦ 张野通过比赛感觉到「和这个人打球挺顺的」（但不会说出口）
  情绪拐点：意外的碰撞 → 比赛中的默契建立
  → 下一场景的衔接：比赛结束
  本场景焦点：人物情绪（通过球技建立的默契——张野不擅长说话，但球场上的配合让他觉得「和这个人打球挺顺」）
  本场景权重：中（≈200-250字）——重点是默契感，不是比赛细节。一两句配合描写就够了，不写完整比赛过程
  感官建议：视觉（眼神交流的瞬间、配合时的位置感）+ 触觉（防守卡位时的身体对抗）
  本场景出口情绪：渐生的默契——两人虽不说话但球场上有交流

**场景 4｜推进剧情·赛后·递水拍肩｜暧昧基调｜权重：高**
  核心事件：比赛结束。陆辞主动走过来——手里拿着两瓶水，递了一瓶给张野：「给。」张野愣了一下接过来——没来得及说谢谢，陆辞已经抬手拍了拍他的肩膀：「打得好。」张野耳朵刷地红了——他每次被拍肩膀都会红，这次也不例外。他自己也不知道为什么，就是控制不了。陆辞看到张野耳朵红了，笑了一下（不是嘲笑，是觉得有意思的那种笑），但没说什么。
  情绪意义：本章情绪强度峰值之一——陆辞的主动温柔（递水+拍肩+夸赞）和张野的不自知生理反应（耳朵红透）。读者在这一刻能闻到暧昧的味道。
  信息差：陆辞看到张野耳朵红了但没追问 ↦ 张野自己也不知道为什么耳朵红——只觉得很热
  情绪拐点：比赛结束的松弛 → 陆辞递水的意外 → 拍肩后耳朵红的生理反应（不自知的暧昧信号）
  → 下一场景的衔接：陆辞继续说话——提出加微信
  本场景焦点：核心冲突（陆辞的主动攻势 vs 张野的不自知生理反应——耳朵红是本章最关键的信号）
  本场景权重：高（≈350-400字）——细化张野耳朵红的过程：从「没反应过来」到「刷地红了」到「他自己也知道耳朵红了对方面前更不好意思」。陆辞拍肩的动作要写出力度和温度——不多不少，就是一个随意的、兄弟式的拍肩
  感官建议：视觉（张野耳朵红起来的画面——从耳根开始蔓延到耳廓）+ 触觉（陆辞拍肩时手掌的温度和力度、张野肩膀被拍时僵了一下的感觉）
  本场景出口情绪：暧昧的心跳——张野的生理反应比他的意识更诚实

**场景 5｜推进剧情·加微信｜心跳基调｜权重：高**
  核心事件：陆辞喝了一口水，掏出手机，很自然地说：「加个微信？以后打球方便约。」语气随意，像在说一件再普通不过的事。张野掏手机的动作慢了半拍——手伸进口袋摸了半天才拿出来，解锁的时候手指有点僵。两人加微信的时候张野低着头，不敢看陆辞的眼睛。陆辞扫了他的码，发了好友申请，随口聊了几句——陆辞话多、主动，问「你哪个系的？」「大几了？」「打球打了多久？」张野回答简短——「体育系」「大四」「从小打」——声音闷闷的、小声的，语速比平时还慢。陆辞没觉得奇怪，笑了笑说「好，下次约」。走之前又说了句「你盖帽挺厉害的」——这句话让张野的耳朵又红了一点。
  情绪意义：关系的正式建立——从「球场上认识的」到「加了微信的人」。读者看到张野在社交场合的不自在和陆辞的主动包容
  信息差：张野知道陆辞是体育系不同专业的、23岁、控卫 ↦ 陆辞知道张野是体育系大四的、前锋、话少但球打好。两人知道的信息对等，但陆辞的主动程度远超张野
  情绪拐点：暧昧的心跳 → 加微信时的心跳加速（张野低头不敢看对方眼睛、手僵、语速慢）
  → 下一场景的衔接：两人各自回宿舍
  本场景焦点：人物情绪（张野在社交上的不自然——掏手机的慢、解锁的僵、回答的简短和低声、不敢对视）
  本场景权重：高（≈400-450字）——细化张野加微信时的每个不自然动作，和陆辞的随和形成对比
  感官建议：视觉（张野低头操作手机时的侧脸、不自觉地咬着下嘴唇、耳朵还是红的）+ 触觉（手指在手机屏幕上滑动时微微发僵的感觉、手机握在手里的汗）
  本场景出口情绪：心跳还没平复——加了微信但不明白为什么心跳还是很快

**场景 6｜过渡·各自回宿舍｜余韵准备基调｜权重：低**
  核心事件：两人各自回宿舍。场景从球场切换到校园路上——傍晚的光线已经暗下来了，路灯亮了。张野走在回宿舍的路上，手里握着手机，屏幕上还是和陆辞的聊天界面——他看了一眼又锁屏，锁了又拿出来看了一眼。没什么特别的理由，就是看了一眼。
  情绪意义：从公共空间（球场/校园）到私人空间（宿舍）的过渡——张野的情绪在独处时开始沉淀
  信息差：张野看了聊天界面一次（读者看到他的动作）↦ 张野自己没多想
  情绪拐点：心跳未平复 → 独处时的安静（看了又锁、锁了又看的动作）
  → 下一场景的衔接：回到宿舍，躺床上
  本场景焦点：人物情绪（独处时的下意识动作——看了锁、锁了看，没什么理由）
  本场景权重：低（≈80-100字，过渡场景）
  感官建议：视觉（傍晚路灯亮起的光线、手机屏幕的光映在张野脸上）
  本场景出口情绪：安静的余韵准备

**场景 7｜推进剧情·宿舍·看头像｜余韵基调｜权重：中**
  核心事件：张野回到宿舍。室友还没回来，屋里就他一个人。他洗了把脸，换了件干净T恤，躺到床上。手机亮着——是陆辞的微信头像。不是刻意打开聊天界面看的——是刷朋友圈的时候刷到陆辞发了条动态，点进去看了他头像，然后退出来，然后又不自觉地打开看了一眼。没有特别的理由——就是看了一眼，然后又看了一眼。他翻了个身，把手机放在枕头旁边，过了几秒又拿起来看了看。
  情绪意义：私人空间里的情绪余韵——读者看到张野独处时的状态：翻来覆去地看头像，没有任何心理独白，就是那个「停不下来」的状态
  信息差：张野看了很多次陆辞的头像但他自己没想过为什么 ↦ 读者比他更早意识到「他这是心动了」
  情绪拐点：独处安静 → 翻来覆去看头像（不自知的反复动作）
  → 下一场景的衔接：最后安静下来
  本场景焦点：人物情绪（独处时的下意识反复——什么心理活动都不写只写动作：看一眼、锁屏、又打开、翻身、又拿起来）
  本场景权重：中（≈200-250字）——重点是动作的反复和不自知，不做心理解读
  感官建议：视觉（手机屏幕的光映在躺着的张野脸上、头像的特写）+ 触觉（手指在手机边缘来回摩擦的小动作、翻身时床板的轻微响声）
  本场景出口情绪：说不清的悸动——什么都没想明白，但就是停不下来

**场景 8｜过渡·章末收束·安静｜余韵·留白基调｜权重：低**
  核心事件：宿舍里安静下来。张野侧躺着，手机屏幕的光暗了（自动锁屏了）。他没再打开——就那样侧躺着，耳朵还是红的（过了这么久还没完全退下去）。窗外的路灯透过窗帘缝隙在墙上投下一道细细的光线。张野闭上眼睛——但没睡着，就是不想动了。
  情绪意义：章末收束——所有动作停止了，但情绪还在。读者看到张野安静下来的状态——耳朵还是红的、什么都没想明白。
  信息差：无——就是收束画面
  情绪拐点：翻来覆去 → 安静下来（但耳朵还是红的——暗示情绪还在）
  → 下一章的衔接：陆辞会发消息过来（第2章）
  本场景焦点：人物情绪（安静的余韵——动作停止，但耳朵还是红的）
  本场景权重：低（≈80-100字，章末留白）
  感官建议：视觉（自动锁屏后屏幕变暗、窗帘缝隙透进来的光线、墙上细细的一道亮光）
  本场景出口情绪：安静的甜——什么都没想明白，但身体记得（耳朵还是红的）

## 不可违反规则

### 优先级（从高到低）

1. **约束红线** — 情节红线、角色禁区、边界禁止（本章的特殊性：「无肉戏初恋章」）
2. **字数** — 2800-3500字目标（超限启用压缩策略）
3. **T1词 / 认知动词 / 感官** — 修饰阈值、认知动词节制、感官分层
4. **叙事规则**（高于同层·写作规范其他规则）
5. **写作规范同级其他规则**

### 红线约束

**冲突阶梯**：第1层·入门——本章以关系建立和情感铺垫为主，不涉及对抗/肉戏。

**情节红线（不可删减）：**
1. **盖帽瞬间必须写**——张野拍掉陆辞上篮的物理碰撞是初遇的标志性画面，不可删
2. **陆辞递水+拍肩必须写**——这是两人关系从「球场对手」到「开始认识」的转折点
3. **张野耳朵红了必须写**——拍肩后耳朵红是全章最关键的生理信号，不可省略
4. **加微信过程必须写**——从「互不相识」到「加了微信的人」是本章的关系改变
5. **宿舍看头像必须写**——章末情绪收束的核心画面，张野翻来覆去地看、耳朵还是红的

**边界禁止（不涉及）：**
1. 本章不写肉戏——不写插入式性爱、不写裸露、不写性器描写
2. 张野的说话方式必须保持——小声、闷闷的、不娘不妖、不爱对视、语速慢
3. 张野对陆辞的心动是「不自知的」——不写「他觉得自己喜欢上陆辞了」，只写生理反应（耳朵红、心跳快、翻来覆去）
4. 不要出现上帝视角解构——不写「这标志着两人关系的开始」「他不知道更大的暧昧正在酝酿」
5. 陆辞的视角不打开——本章全程紧贴张野的第一人称有限/第三人称有限视角，不写陆辞的内心想法

**角色禁区：**
张野不会主动约陆辞（被动性格）
张野不会在章末想明白「自己喜欢陆辞」
张野的耳朵红是生理反应——他自己控制不了，也不用美化/浪漫化这个反应
陆辞的主动不超过「递水+拍肩+加微信+多聊了几句」——本章他不表白、不越界、不写他心动

### 网文风格基线（任何章节适用）

1. 认知动词节制使用（≤2次/章）
2. 因果关系自然呈现
3. 叙事自然有温度——禁止极端白描
4. 全程线性叙事
5. 拒绝完美叙事——允许留白和思绪跳跃
6. 禁止刻意堆砌技法
7. 禁止诗词腔

### 风格建议（可偏离）

- **视角**：第三人称有限，紧贴张野。不写他人内心。禁止全知解释。
- **描写**：用动作和感知推动叙事。本章以**视觉（光线/表情/手机屏幕）和听觉（球场声音/对话/安静）** 为主要感官通道。
- **节奏**：段落自然起伏，句式长短交替。比赛场景可以短句快节奏，宿舍场景可以长句慢节奏。
- **张野的说话方式**：小声、闷闷的、语速慢、回答简短。不娘不妖、不爱对视。
- **陆辞的说话方式**：随意、阳光、话多、主动。笑的时候眉眼舒展，说话带自然的热络感。
- **本章情感基调**：暧昧、甜蜜的悸动、不自知的心动——像夏天的第一口汽水，气泡在喉咙里噼啪作响但你不知道那是什么。

### 场景写作指引

本章涉及3种场景类型（sports-scene / dialogue / transition）：

**运动场景（sports-scene）—— 场景1-3：**
1. 感官优先取**视觉+听觉**——球场的傍晚光线、球员的跑动、拍球声、球鞋摩擦声
2. 盖帽瞬间写爆发力和速度——张野跳起、手掌和球接触的瞬间、球被拍飞出去的方向
3. 不写完整比赛过程——只写和两人互动相关的几个瞬间，其余略写

**对话场景（dialogue）—— 场景4-5：**
1. 陆辞的对话用动作替代说话标签（递水、拍肩、掏手机、笑），少写「陆辞说」
2. 张野的对话简短低声，用低头/别开视线/耳朵红替代对话标签
3. 对话要有节奏——陆辞话多但自然，张野回答短但不算冷淡（只是不善言辞）

**过渡场景（transition）—— 场景1、6、8：**
1. 场景1（球场傍晚）压缩至≈80-100字：1句光线 + 1句嘈杂声 + 1句主角状态
2. 场景6（回宿舍路上）压缩至≈80-100字：1句路灯亮了 + 1句张野看了眼手机 + 1句锁屏
3. 场景8（章末收束）压缩至≈80-100字：1句手机暗了 + 1句耳朵还是红的 + 1句安静

### 质感要求

**无用细节（不服务主线的细碎生活化细节）：**
1. 篮球拍在地上时地面的灰尘扬起来一点
2. 张野球衣后背被汗浸湿了一小片
3. 矿泉水瓶瓶身上凝结的水珠，张野接过来的时候手指上沾了水
4. 宿舍走廊里有人说话的声音从远处传来
5. 手机自动锁屏时的轻微震动

**对话节奏：允许半截话、语气词、无意义停顿**
- 张野说话：「嗯」「好」「体育系」「大四」「从小打」——短句、小声、闷闷的
- 陆辞说话：「哥们手劲挺大」「给」「打得好」「加个微信？以后打球方便约」「你哪个系的？」「你盖帽挺厉害的」——自然随意、话多但不压迫

**段落精度分层（按权重分配笔墨）：**
- **高权重场景（场景2、4、5）**：细节饱满，至少3处动作/感知层级细节用于核心情绪展开。场景2细化盖帽瞬间的爆发力和陆辞的反应；场景4细化递水拍肩的每个动作和张野耳朵红的全程；场景5细化加微信时的每个不自然动作
- **中权重场景（场景3、7）**：正常叙事节奏，默契和余韵的推进感为主
- **低权重场景（场景1、6、8）**：快速转场，环境铺垫和收束

### 额外约束（本章特有）

1. **本章是纯爱/初恋章，不涉及任何肉戏**——不写插入式性爱、不写裸体、不写性器、不写自慰。甜味来自暧昧和心跳，不是来自肉体描写。
2. **张野的不自知是核心设定**——他不知道自己为什么心跳加速、为什么耳朵红、为什么翻来覆去地看头像。不写「意识到」「感到」「觉得」引导的心理分析。只写动作和生理反应。
3. **张野的耳朵红不要过度描写**——写了就行（「耳朵刷地红了」「耳朵还是红的」），不要每次出现都大段描写。
4. **陆辞的主动要自然**——不是刻意的撩拨，就是一个阳光开朗的人自然的社交方式。递水、拍肩、加微信——对他来说就是正常操作。
5. **章末不要升华**——不写「他不知道这就是初恋的味道」「这一夜注定改变了他」。停在画面和动作上：耳朵还是红的、手机暗了、安静了。
