# 薪资谈判完整指南

---

## 一、谈判前的认知校准

### 理解薪资决策的底层逻辑

薪资不是HR一个人决定的，而是由四个因素共同决定：
1. **公司薪酬体系：** 每个岗位有对应的薪资带（band），HR只能在带内调整
2. **岗位预算：** 这个headcount批了多少预算，HR不能超
3. **内部公平性：** 不能让新人比老人高太多，否则团队不稳定
4. **候选人的市场价格：** 你现在的薪资 + 市场行情 + 手里的offer

**关键洞察：** HR的KPI是"在预算内招到合适的人"，不是"把薪资压到最低"。如果你足够优秀且价格合理，HR有动力帮你争取更好的package——因为你接了offer，他的招聘任务就完成了。

### 中国市场薪资结构解析

```
月度现金 = 底薪 + 绩效工资
  └→ 底薪和绩效的比例很重要：底薪80%+绩效20% vs 底薪60%+绩效40%，稳定性差很多

年度总包 = 月度现金 × 月数（12/13/14/15/16薪不等）
          + 年终奖（保底型 vs 浮动型）
          + 签字费（Sign-on Bonus，一次性）
          + 股票/期权（分年归属，关注归属schedule）
          + 补贴（餐补、交通补、房补等）

隐性收入 = 五险一金（重点关注公积金比例：5%-12%差距巨大）
          + 补充商业保险
          + 培训预算
          + 年假天数
```

**一个真实的例子：**
```
Offer A：月薪20K × 14薪 = 28万/年
Offer B：月薪18K × 16薪 + 年终奖2个月 = 36万/年

表面看A的月薪更高，但年度总包B多了8万。
所以永远要算年度总包，不要只看月薪。
```

## 二、谈判前的信息收集

### 了解市场行情的渠道

```
1. 脉脉：搜索"[公司名]+[岗位]+薪资"，看在职员工讨论
2. 看准网/Glassdoor：查公司薪资范围
3. 牛客网（校招）：校招offer汇总帖，薪资数据比较全
4. 猎头朋友：如果认识猎头，直接问是最准的
5. 同行交流：同行业同岗位朋友的薪资是最好的参考
6. 招聘网站JD：部分JD会写薪资范围，注意是税前还是税后
```

### 评估自己的议价能力

**议价能力强的信号：**
- 手里有其他offer（最强筹码）
- 目标公司急招（从JD发布时间和HR催促程度判断）
- 你的技能稀缺（市场上不好找）
- 面试评价很高（HR会透露一些信号）

**议价能力弱的信号：**
- 目前在职但被裁/即将被裁
- 已裸辞超过3个月
- 校招且没有其他offer
- 目标公司候选人充足
- 岗位替代性强

### 确定三个关键数字

```
理想薪资（Stretch Target）：
  拿到就非常满意，不会再考虑其他机会
  通常设在市场行情的75-90分位

目标薪资（Target）：
  综合评估后的合理预期
  通常设在市场行情的50-75分位
  对外报的数字

底线薪资（Walk-away Point）：
  低于这个数你就不去——但绝对不要说出来
  这是你自己的决策边界，不是谈判筹码
```

## 三、不同场景的谈判策略

### 场景一：面试过程中被问"期望薪资"

**最佳策略：尽量让对方先出牌**

```
话术选择：

A（推荐）："我现在还在了解阶段，薪资方面比较灵活。
  能先了解一下这个岗位的薪资范围吗？
  这样我可以给您一个更准确的预期。"

B（如果A不行，对方坚持要数字）：
  "基于我[X]年的经验和目前的市场行情，
  我的期望年度总包在[目标薪资]左右，
  当然具体还要看薪酬结构和整体package。"

C（如果被问现在的薪资）：
  "我目前的年度总包在[X]左右（可以适当美化10-15%，但不要造假），
  考虑到这次岗位的变化和职责提升，我期望有[20-30]%的涨幅。"
```

**注意事项：**
- "美化"和"造假"的区别：把年终奖、补贴、期权预期值算进去是美化；编造不存在的收入是造假
- 背调可能会查你的银行流水或社保记录，造假风险很高

### 场景二：收到offer，数字低于预期

**关键原则：永远不要当场接受，也不要当场拒绝**

```
Step 1：表达感谢和兴趣
  "非常感谢，我对这个机会很感兴趣。"

Step 2：争取时间
  "我需要综合考虑一下，可以给我[2-3]天时间吗？"
  
Step 3：在电话/邮件中提出调整
  "我仔细考虑了这个offer，对岗位本身非常认可。
   坦白说，薪资方面和我的预期有一些差距。
   考虑到[具体理由]，
   我希望能在[底薪/年终/签字费]上做一些调整。
   不知道是否有空间？"
```

**可以争取的维度（如果底薪谈不动）：**
```
签字费（一次性，公司更容易批）
年终奖保底条款（"第一年年终奖不低于[X]个月"）
股票/期权数量
公积金缴纳比例
入职职级（影响后续涨薪和晋升基线）
试用期时长缩短
年假天数
远程办公灵活度
```

### 场景三：有竞品offer作为筹码

**使用原则：真实、专业、不威胁**

```
话术：
  "坦白说，我手里有[另一家公司]的offer，
   整体薪酬package在[X]左右。
   但我更倾向于加入贵公司，因为[真实原因——业务方向/团队/成长空间]。
   想看看在薪酬上是否有机会拉近差距？"

注意：
  - 可以说大概范围，不必说精确数字
  - 不要编造不存在的offer——HR圈子不大，容易被戳穿
  - 语气是"我想来但有顾虑"，不是"不加钱我就走"
```

### 场景四：校招/管培，薪资体系固定

```
可以尝试的：
  - 问是否有SP/SSP档位（特别优秀候选人的加薪档）
  - 争取签字费
  - 争取更好的部门/团队选择权
  - 了解试用期考核标准和转正后的调薪机制

不建议硬谈的：
  - 基础底薪（大厂校招同档位基本固定）
  - 入职时间（校招通常有统一入职日期）
```

## 四、谈判红线

**绝对不要做的事：**

| 红线 | 为什么 |
|------|--------|
| 说出底线薪资 | HR一定会压到这个数 |
| 撒谎说有别的offer | 背调或后续可能暴露 |
| 拿到口头offer就拒掉其他 | 口头offer不是书面合同 |
| 发最后通牒"不加钱就不来" | 除非你真的不在乎 |
| 在社交平台公开讨价还价过程 | 职业形象受损 |
| 和HR争论"这个薪资不公平" | 情绪化谈判没有赢家 |
| 同时和多个HR透露对方的薪资细节 | 违反保密原则，失去信任 |

---

## 五、收到书面offer后的确认清单

```
□ 薪资结构是否和口头谈的一致？（底薪+绩效+月数+年终奖）
□ 五险一金的缴纳基数和比例？
□ 试用期多长？试用期薪资是否打折？
□ 竞业协议的范围和期限？有没有补偿？
□ 期权/股票的归属schedule（vesting）是怎样的？
□ 入职时间和报到流程？
□ 有没有签字费？提前离职是否需要返还？
□ offer有效期是多久？

如果以上任何一项和口头沟通不一致，立即和HR确认。
书面offer才是最终的法律依据，口头承诺没有约束力。
```
