# 第一章·野狗

- **memo:** 建立沈骁的身份高度——田径队王牌、林越的男友、自信到有点自负。同时埋下赵叔的注视和林越的不安。
- **stage:** archived
- **字数:** 约3400字（完整正文已归档）

## 场景

### 场景1｜训练场｜黄昏｜权重：高

- **入口情绪**：自信、阳光
- **核心事件**：沈骁完成间歇训练 → 队友叫去吃饭 → 沈骁去器材室
- **信息差**：沈骁觉得赵叔看他的眼神不太对但没在意 ↦ 读者知道赵叔别有用心
- **情绪拐点**：被赵叔的目光打量时的轻微不适
- **出口情绪**：轻微不安但被自负掩盖

### 场景2｜食堂｜夜晚｜权重：中

- **核心事件**：林越等沈骁吃饭 → 日常恩爱互动 → 林越约沈骁晚上去他宿舍
- **情绪意义**：展示两人关系的温暖面，为后面的坍塌做铺垫

### 场景3｜林越宿舍｜熄灯后｜权重：中

- **核心事件**：林越提到赵叔的眼神奇怪 → 沈骁敷衍过去 → 但他心里其实也在意
- **信息差**：林越直觉准确但被沈骁安抚 ↦ 读者知道林越的直觉是对的

### 场景4｜烧烤摊→回宿舍路上｜夜晚｜权重：中（转场）

- **核心事件**：省赛前庆祝 → 沈骁喝了不少 → 在宿舍楼下被赵叔拦截 → 被诱骗到器材室
- **情绪拐点**：从轻松愉快 → 酒醒后警觉
- **出口情绪**：紧张、悬念——门锁落下了

## 人物状态

- **沈骁起点→终点**：自信满满的队长 → 陷入陷阱前的最后一刻清醒
- **赵叔出场**：器材室初亮相，话不多但目光有侵略性
- **林越出场**：温柔体贴的男友，直觉敏锐
