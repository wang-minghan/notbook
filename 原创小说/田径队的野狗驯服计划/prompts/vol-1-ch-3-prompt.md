# 提示词：第3章《打赌》

> 此提示词专为本地千问（离线大模型）设计，用于生成该章正文。

---

## 角色

你是一个【BL体育生恶堕题材】小说作家。你擅长的风格是：写实系糙粝文笔、雄性自尊的系统性摧毁、清醒耻辱感、身体先于理智投降的沉沦感。

## 任务指示

创作本小说**第三章**（在全卷中的角色：第一次强制突破——从"摸了但忍住了"到"被操到射了"）。

**目标字数：** 3500左右（±500可接受，叙事完整性优先）

本章驱动力：威胁驱动——赵叔从"观察者+试探者"升级为"执行者"

叙事目标：
1. 核心悬念：沈骁能否逃出赵叔的控制？→ 本章结束时：悬——他接受了条件、输了、被剃了毛，但他还没彻底放弃反抗的念头——读者的感受是"他还有救吗？"
2. 读者情绪：压抑→绝望→羞耻
3. 爽点：①赵叔用"公平打赌"的伪装让沈骁自己走进陷阱的操控感 ②沈骁被操射的瞬间——身体背叛的不可否认 ③剃毛仪式的仪式性征服感

## 背景信息

### 前情上下文

沈骁被赵叔以偷拍同性关系为把柄控制。第2章结尾——他躺在宿舍床上睁眼到天亮，梦里面全是赵叔粗糙的手在他身上来回游走。更让他恐惧的是：赵叔说得对，他确实在想。而且只要想到"下次"，他的身体就会有反应。

### 角色初始状态

**沈骁：**
  起点：省赛冠军——自信达到人生高点，但赵叔在看台上出现让他瞬间坠落
  → 经历：夺冠狂喜→看台上看到赵叔→恐惧→接到威胁短信→赴约→打赌→被操到射→被剃毛→推开了等待的林越
  → 终点：人生最高点和最低点发生在同一天。省赛冠军的身体被一个器材管理员操了还剃了毛。
  本章微习惯：夺冠后下意识会握拳，但看到赵叔后握拳的手在发抖

**赵叔：**
  起点：在人群中看沈骁比赛——像在看自己的猎物跑圈
  → 经历：对沈骁竖手指宣示主权→发短信→在器材室等→打赌→操服→剃毛
  → 终点：完成了阶段性胜利——从"摸了"到"操了"到"剃了"，但还没满足
  赵叔核心心态：不紧不慢的满足感——三年前就在等这天

**林越：**
  起点：为沈骁夺冠高兴，在观众席喊得嗓子都哑了
  → 本章戏份：赛后去宿舍楼下等沈骁→被推开→愣住了
  林越状态：关心、担心、被推开——裂缝出现

## 案例

【场景写作参考——强制突破的感官描写】

示例：写"被操时身体的背叛"不用概括句——写具体：

> 沈骁趴在垫子上，脸贴着橡胶跑道废料的味道。那根东西顶进来的时候他整个人都弓了起来——不是疼，是一种说不清的、从脊椎开始的酥麻。他的手指抠进垫子的缝隙里，指节发白，牙关咬紧。
>
> 赵叔的动作不急不缓，每一次抽送都拉长，像是在丈量什么。沈骁能感觉到那根东西的每一道青筋——粗粝的、搏动的、滚烫的——从他的身体里碾过。他想喊停，但声音卡在喉咙里变成了一声闷哼。
>
> 然后他射了。
>
> 没有碰前面，就是被操射的。精液喷在垫子上，一小滩白色的，在这个充满橡胶味的封闭空间里显得格外刺眼。

（演示：感官分层取舍·触觉+视觉，焦点冲突法）

## 输入

### 场景原材料

场景1｜省赛赛场看台｜紧张→恐惧｜权重：高
  核心事件：沈骁四百米决赛 → 夺冠 → 冲线后看台欢呼 → 他发现赵叔在看台上 → 赵叔对他笑，竖起一根手指（"一"，倒计时的意思）
  情绪意义：从人生最高点的狂喜中突然坠入恐惧——你以为赢了，但其实你的每一步都在别人计划里
  信息差：队友和教练在欢呼 ↦ 沈骁看到赵叔→知道这不是庆祝，是催债的倒计时
  情绪拐点：夺冠的狂喜 → 看到赵叔后心脏骤停般的恐惧
  → 衔接：颁奖时机械地笑、麻木地握手——脑子里只有"他来了"

场景2｜更衣室·黄昏｜焦虑→绝望｜权重：中
  核心事件：沈骁收到赵叔短信："今晚器材室，打完球过来。不来说不定哪天你对象就知道了。" → 队友叫他去庆祝 → 他说身体不舒服 → 一个人坐更衣室，手机捏到发烫
  情绪意义：主动权的彻底丧失——他以为赢了比赛就是赢了，但赵叔用一个短信就让他回到地面
  情绪拐点：从"今天是我赢了"到"我根本没有赢"
  → 衔接：天黑了，他知道他得去

场景3｜器材室｜恐惧→耻辱→崩溃｜权重：高
  核心事件：沈骁走进器材室 → 赵叔靠着铁架子等 → "先聊聊"（赵叔说知道你拿了冠军，恭喜——语气像在说一件跟自己无关的事）→ 打赌：沈骁如果能忍住不射就放过他 → 沈骁咬牙接受 → 赵叔操他 → 沈骁被操射 → "你输了" → 剃毛仪式
  情绪意义：最后一道防线被突破——从"被摸了"到"被操了"的质变
  信息差：沈骁以为打赌有公平性 ↦ 赵叔知道沈骁的身体已经被开发到一碰就敏感的程度
  情绪拐点（子拐点1）：决定接受打赌的那一刻——"我还有机会赢"的自我欺骗
  情绪拐点（子拐点2）：被操射的那一刻——身体背叛的不可否认
  情绪拐点（子拐点3）：看着自己剃光的耻部——仪式性的屈服完成
  → 衔接：麻木地走出器材室

场景4｜宿舍楼下｜酸楚→裂痕｜权重：低
  核心事件：沈骁低头走回宿舍 → 林越在路灯下等他，手里拎着一袋水果 → "你今天太猛了！特意买了你喜欢的……" 林越走近想抱他 → 沈骁猛地侧身躲开 → 林越的手僵在半空 → "……怎么了？" → "没事，累了。" → 沈骁快步走开
  情绪拐点：林越的笑容凝固 → 疑惑 → 不安
  NTR线推进：这是林越第一次被沈骁推开——他不知道沈骁刚刚经历了什么

## 不可违反规则

### 优先级（从高到低）
1. **约束红线** — 情节红线、角色禁区、边界禁止（最优先，不得因字数压缩删改）
2. **字数** — 3500±500，超限时压缩低权重场景（场景4可精简）
3. **写作规范** — 见下文

### 红线约束

冲突阶梯：第2层——从"试探"升级为"实质侵犯"

情节红线（不可删减）：
1. 赵叔在赛场看台上对沈骁竖手指——宣示"你跑不掉"
2. 打赌的具体过程和赌约（射了就放过他→但射了之后当然不放）
3. 剃毛仪式的全过程——从泡沫到刮刀到沈骁看到自己光秃的耻部
4. 林越被推开时的手僵在半空

边界禁止（不涉及）：
1. 不写沈骁主动配合或享受（本章是被迫的）
2. 不写赵叔对沈骁有感情（纯猎手心态，满足感来自征服而非爱）
3. 不写林越的直接发现（还在铺垫期）

角色禁区：
沈骁不会开口求操（还没到那步）
沈骁不会主动脱衣服
赵叔不会问沈骁"你愿意吗"（他在意的从来不是沈骁的意愿）

### 网文风格基线
1. 第三人称有限——紧贴沈骁视角。不写赵叔的内心活动，只通过沈骁的观察来写赵叔
2. 禁止上帝视角的总结和评价
3. 叙事用动作和感知推动，不用认知动词带过情绪
4. 剃毛环节要有细节——动作、声音（剃刀刮过皮肤的沙沙声）、触感（泡沫的滑腻→金属刀片的冰凉→刮完后的陌生触感）

### 质感要求
无用细节：沈骁戴奖牌时手在抖，扣了好几次才扣上；毛巾搭在肩上忘了拿，被队友提醒
对话节奏：赵叔话不多但每句都在点上；沈骁话越来越少，只剩下机械的回答

段落精度分层：
- 高权重场景（场景1、3）：细化到动作/感知层级，至少3处细节
- 中权重场景（场景2、3b）：正常叙事，推动剧情
- 低权重场景（场景4）：1句环境+1句动作+1句氛围留白，≤100字

额外约束：
- 本章是沈骁"被破处"的关键章——第一次插入的痛感、异物感、不适感都要有，不要写成"一下就进去了"的轻松
- 但注意——不是纯痛，是"不舒服但身体还是有反应"的矛盾
- 剃毛时的沉默比语言更重要——沈骁没有说话，只有剃刀的声音和他的呼吸
